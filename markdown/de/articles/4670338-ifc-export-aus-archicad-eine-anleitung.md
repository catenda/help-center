# IFC Export aus Archicad - eine Anleitung

Wenn Sie eine IFC-Datei exportieren, sollten Sie darauf achten, was für Ihren IFC-Export relevant ist. Die IFC-Datei kann groß und schwer zu bearbeiten sein, wenn sie viele Informationen enthält. Daher ist es wichtig, keine unnötigen Informationen zu exportieren. In diesem Bericht erhalten Sie verschiedene Tipps zum Filtern Ihres IFC-Exports in Archicad.

Um das gesamte Projekt zu exportieren. Sie müssen sich in der 3D-Ansicht befinden. Stellen Sie sicher, dass Sie den richtigen Übersetzer verwenden.

![](https://catenda-as.intercom-attachments-1.com/i/o/271056907/3ed1a2d39b9276705eca2953/image-0.png?expires=1781092800&signature=727a3898f989f68df24662a23ff14957bf59781f3bc27d84f57a5035967a4361&req=dicmFsx4lIFYFb4V1XW4gSsqFAIns%2BLVOCRTS1jsXxQqqssEUzUY2qHudMas%0AwiXzkKrZCxh05B46OxMksUBk8A%3D%3D%0A)

Klicken Sie auf die Schaltfläche Optionen, um eine Zusammenfassung Ihrer IFC-Exporteinstellungen zu sehen. Unter Modellfilter können Sie auch wählen, ob Sie mehr herausfiltern möchten.

![](https://catenda-as.intercom-attachments-1.com/i/o/271056913/07e3a491c52e1af168324efe/image-1.png?expires=1781092800&signature=5003112051c8e8137b5bd8a0b7928b9d95227b2fe1f992494c057307472e3fe0&req=dicmFsx4lIBcFb4V1XW4gbS3lr63ymIv%2Bs4YYj8FOwEC8e03LRJ%2F1CBZuT%2F3%0A3q1TIGFHJXIvB8QkpSEHWMf08w%3D%3D%0A)

**IFC-Export-Einstellungen**

Im Feld IFC-Übersetzer können Sie die Übersetzereinstellungen anzeigen oder ändern oder neue Übersetzer erstellen.

Wenn Sie Ihren IFC-Export ändern möchten, empfiehlt es sich, einen der vorgefertigten Übersetzer zu duplizieren, damit Sie nicht einen der Standardübersetzer durcheinander bringen.

Klicken Sie auf Neu > Duplizieren von > wählen Sie den Übersetzer aus, den Sie duplizieren möchten.

Wenn Sie die IFC zusammenführen möchten, können Sie dies unter demselben Banner tun.

Verschiedene Einstellungen beim Export

1. Mit dem Modellfilter können Sie die zu exportierenden Daten anhand verschiedener Voreinstellungen herausfiltern.

2. Mit der Typzuordnung können Sie wählen, als welcher IFC-Typ jedes Element exportiert wird.

3. Mit der Geometriekonvertierung können Sie auswählen, welche Art von Geometrie Sie exportieren möchten.

4. Mit der Eigenschaftszuordnung können Sie Kriterien auf der Grundlage von Typen einrichten.

5. Bei der Datenkonvertierung können Sie auswählen, welche Art von Daten Sie aus dem Modell exportieren möchten.

6. Mit der Einheitenumwandlung können Sie wählen, welche Maßeinheiten Sie in Ihr IFC exportieren möchten.

## **2. Type Mapping**

Wenn eine IFC exportiert wird, wird allen Elementen des Modells ein IFC-Typ zugewiesen.

Wenn Sie den IFC-Übersetzer ausgewählt haben, den Sie verwenden möchten, können Sie zur Typzuordnung gehen und auf IFC-Typen für den Import zuordnen klicken, um die Art der Typzuordnung zu verwalten, die Sie für Ihre exportierte IFC-Datei wünschen.

![](https://catenda-as.intercom-attachments-1.com/i/o/271056940/57204561a8ccd3176e04d381/image-4.png?expires=1781092800&signature=8b2550d1949b9bcf2166050a67b0157d580cbf7308ffb90554f3424dd1d003dd&req=dicmFsx4lIVfFb4V1XW4gRT3Fi9%2FTyVfVfmhSH8DRamqc%2FFvbJuyRfAk7nhc%0AqQSucfuCAs4ih7jWnNIOcahFzg%3D%3D%0A)

Bei der Typzuordnung gibt es hauptsächlich zwei verschiedene Optionen, nach denen Sie Ihre IFC-Typen sortieren können.

### **Element type**

Jedem Element wird automatisch ein grundlegender IFC-Typ zugewiesen. Sie können den jedem Element zugewiesenen IFC-Typ im Projektmanager und in den Elementeinstellungen auswählen.

### Klassifizierung

Diese Methode ermöglicht eine flexiblere und detailliertere IFC-Typenzuordnung, entsprechend den spezifizierten Klassifizierungsstandards. Zone und Öffnungselemente werden auf einen festen IFC-Typ festgelegt. IFCSpace und IFCOpeningElements.

## 3. Konvertierung der Geometrie

Mit Geometriekonvertierung für IFC-Export können Sie Ihre Geometrie auf verschiedene Arten konvertieren.

![](https://downloads.intercomcdn.com/i/o/852256533/6acaa196d2acf7499d9aaafa/image.png?expires=1781092800&signature=477ab462c03abaca08a3a1bb95231c979a16c909f98d569a6e25e90855732436&req=fCUlFMx4mIJcFb4V1XW4gbFA7WWR17Jl6uaRn16tGKO80nJZcAqI0Q8myZSQ%0AUi30%2BZpM7YkT%2Bi5%2FWjhulkJmiA%3D%3D%0A)

### Konvertierung von Archicad-Elementen

Exportieren Sie die Geometrie aller Modellelemente als:

Parametrisch mit Ausnahmen - Standard

BREP - Präzise Geometrie - Wenn dies ausgewählt ist, sind die nächsten beiden Optionen ebenfalls BREP

Elemente in Solid Element Operations:

Stranggepresst/gedreht - Standard

BREP - Präzise Geometrie

Elemente mit Kreuzungen

Einige Elemente können sich mit anderen Elementen kreuzen, wodurch Teile der Extrusion abgeschnitten werden können.

Extrudiert/gedreht ohne Kreuzungen - Standard - Dies beschleunigt den Export

Stranggepresst/gedreht -

BREP - Präzise Geometrie

Parametrische Elemente

Archicad-Elemente können als parametrische Geometrie exportiert werden, oft in Form von Führungslinien, die auf eine bestimmte Entfernung extrudiert werden. Parametrische Geometrie erleichtert die Bearbeitung des Objekts, nachdem es erstellt worden ist. Während diese Option die Bearbeitbarkeit beibehält, kann die Geometrie oft etwas ungenau sein, da die Definition eine Annäherung an die Geometrie ist.

Diese Option wird empfohlen, wenn Sie beabsichtigen, die IFC zurück in Archicad oder ein anderes Bearbeitungsprogramm zu importieren, um weitere Bearbeitungen vorzunehmen.

BREP

Die Elemente können auch nach BREP exportiert werden. Anstatt die Objekte durch ihre Parameter zu beschreiben, wird jede Oberfläche als separate Fläche mit einer Position in 3D beschrieben.

Bei dieser Option gehen zwar die Parameter verloren, wodurch die Objekte nach dem Export schwieriger zu bearbeiten sind, aber Sie gewinnen an Präzision, da jede Oberfläche eine genaue Darstellung der Oberfläche im Objekt ist.

Mit dieser Option kann jedes Objekt in der IFC-Datei auch eine Farbe haben.

Wenn Sie diese Option wählen, wird Ihre IFC-Datei in Catenda Hub die gleiche Farbe haben wie in Archicad.

Definieren Sie die Position des IFC-Modells nach:

Diese Option bestimmt den Projektstandort.

Es ist wichtig, sich bereits zu Beginn des Projekts auf einen gemeinsamen Koordinatenpunkt zu einigen, um das Projekt zu referenzieren. Dies könnte bedeuten, dass mehrere Berater ihre Objekte verschieben müssen, was gegen Ende eines Projekts oft nicht mehr möglich ist.

Vermessungspunkt und Projektursprung - Standard

Mit dieser Option entspricht der Abstand von (0,0,0) zu Ihrem Modell in Archicad dem Abstand von (0,0,0 + Koordinaten Ihres Vermessungspunktes) zu Ihrem Modell in Catenda Hub + die

Wenn Sie einen Projektstandort festgelegt haben, wird Ihr Modell an diesen Koordinaten in Catenda Hub enden.

Nur Projektursprung

Mit dieser Option wird der Abstand von (0,0,0) zu Ihrem Modell in Archicad der Abstand von (0,0,0) zu Ihrem Modell in Catenda Hub sein.

Nur Vermessungspunkt

Mit dieser Option wird der Abstand vom Vermessungspunkt in Archicad zu Ihrem Modell der Abstand von (0,0,0) zu Ihrem Modell in Catenda Hub sein.

### IFC-Schema-bezogene Optionen

Materialerhaltungsmodus (nur IFC2x3)

- Niemals Elemente explodieren, Erhaltung ist nicht garantiert

Mit dieser Option wird das gesamte Objekt als ein Objekt exportiert

- Nur auflösen, wenn nötig, um Materialien zu erhalten - Standard

Mit dieser Option exportieren Sie nur dann separate Objekte für jedes Verbundmaterial, wenn dies erforderlich ist.

- Alle Elemente in Teile auflösen, Materialien bewahren

Mit dieser Option werden alle Objekte als separate Objekte für jedes Material exportiert

Wenn Sie ein zusammengesetztes Wandmaterial haben, bedeutet dies, dass für jedes Material ein eigenes Objekt exportiert wird.

Sie haben oft eine Dampfsperre von 1-5 mm, was zu einem sehr dünnen Objekt führt.

Wenn Objekte so dünn sind, kann die Geometrie ungenau sein.

Problematische Wandmaterialien können etwa so aussehen:

![](https://downloads.intercomcdn.com/i/o/1081851011/f299a5efe75792604e232f1e/image.png?expires=1781092800&signature=4e5bd2fc5b5a425217c5a10336a579578dbeb66300d566bb3a1904e44fbe7fd0&req=dSAvF8F7nIFeWPMW3nq%2BgQHJk%2FcTXuZhurrDgjjRlo8WQINdlX%2Fot4EpHXfb%0AHq%2B946vaAfTcPCXm3FcsVCYgn0E%3D%0A)

Dies kann es der Software erschweren, Löcher in die Oberfläche zu schneiden, da es aufgrund der Ungenauigkeit schwierig ist, genau zu bestimmen, wo das Loch sein soll.

Wenn Sie also feststellen, dass Ihre Öffnungen nicht durchgeschnitten werden, kann es helfen, diese Option zu aktivieren.

Wenn Sie diese Option aktivieren, wird die Wand als ein einziges Objekt exportiert und nicht als viele dünne zusammengesetzte Objekte.

### Zusammengesetzte Strukturen und komplexe Profile

Komplexe Bauelemente in Teile aufteilen

Hier können Sie auswählen, für welche Art von Elementen das zusammengesetzte Element aufgeteilt werden soll und für welche nicht.

Wenn Sie diese Option wählen, können Sie keine Auswahl für den Materialerhaltungsmodus treffen.

## **4.** Eigenschaftszuordnung

In der Eigenschaftszuordnung (Datei > IFC > Interoperabilität > Eigenschaftszuordnung) können Sie wählen, welche Art von IFC-Version Sie exportieren möchten. Es gibt die Standardversion IFC2x3 und die Standardversion IFC4. Sie können auch P-Sets zum Export mit Ihrer IFC hinzufügen. Wenn Sie dies tun, sollten Sie ein Duplikat des von Ihnen gewählten IFC-Schemas erstellen.

![](https://catenda-as.intercom-attachments-1.com/i/o/271056948/23de75ed67ebb8120f9ef63d/image-6.png?expires=1781092800&signature=2314c41343dba2f942862c4fb165cc070dcfa98658050e223df6a2fe8cecd606&req=dicmFsx4lIVXFb4V1XW4gX0eUSHQutujQy5PjUI1pttrN6jFy0gn5Wmw5l2r%0AvnVe595gd%2F5OcH05ksfM1MFdjA%3D%3D%0A)

Nachdem Sie ein Duplikat Ihrer Standard-IFC erstellt haben, können Sie die gewünschten Eigenschaften zu dieser neuen Voreinstellung hinzufügen, indem Sie das IFC-Schema auswählen und auf IFC-Eigenschaften für den Export zuordnen klicken.

![](https://catenda-as.intercom-attachments-1.com/i/o/271056955/8fd00cf142ae8d45a0fb7df1/image-7.png?expires=1781092800&signature=e66c9d5f53bbfae153da829dffb45bc582bc112cc3198628772d8798ecf6aabf&req=dicmFsx4lIRaFb4V1XW4gct4hmNJw3Cxmyiu4iXSGkjI8COu4X1sCPk6Gl1D%0AoF5KclTEvV0MOwQ%2BOYYH9OHrDw%3D%3D%0A)

## 5. Datenumwandlung

Unter Datenkonvertierung wählen Sie aus, welche Art von Daten Sie zusätzlich zur Geometrie aus Ihrem IFC-Export erhalten möchten. kreuzen Sie an, was Sie exportieren möchten.

Elementparameter liest die Archicad-Elementparameter und konvertiert sie in IFC-Mengen oder IFC-Eigenschaften. Abhängig von ihrem Typ. Wenn Sie diese Option wählen, erhöhen Sie die Dateigröße erheblich.

![](https://catenda-as.intercom-attachments-1.com/i/o/271056963/bf0e775372c6fc6418e2727a/image-8.png?expires=1781092800&signature=7ae773823614711d50cdd4f7780fab8d8cc35ab2e7acad5a5c0474d4a31c0b92&req=dicmFsx4lIdcFb4V1XW4geoCNQZckbiMZP52Jc8%2BroBx3DCAZ7qhaLrtGOkQ%0AFPsbhSQya9mPEkHsU%2FfnXUXAaA%3D%3D%0A)

IFC Base Quantities liest die Parameter Größe, Fläche und Volumen. Wenn Sie dieses Kontrollkästchen nicht aktivieren, kann es beim Import Ihrer IFC in Catenda Hub zu Problemen kommen.

## 6. Umrechnung von Einheiten

Legen Sie die Einheiten Länge, Winkel, Fläche, Volumen, Währung und Zeit für Ihren Export fest.
