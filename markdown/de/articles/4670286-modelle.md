## 1. Modelle im Viewer öffnen
Es ist möglich, alle ausgewählten Modelle zu laden, indem Sie auf die Schaltfläche 3D oben neben den ausgewählten Modellen klicken.

![](https://downloads.intercomcdn.com/i/o/780871114/88ee16510b82a2b99230ff05/image.png?expires=1780444800&signature=8e87b649363487754072d12fa67fabe4741ccda87a3bae7ac5858a46b536ef12&req=cygnHs5%2FnIBbFb4V1XW4gd13wHGyfZn2%2BK%2Bg6X7M8OD%2FgYDWw4hP8OQNUvVZ%0Ak7UvOLGXskES3cexleIPD8gelw%3D%3D%0A)

Es ist möglich, bestimmte Modelle in 2D und 3D zu öffnen, indem Sie auf die entsprechenden Schaltflächen klicken.

![](https://downloads.intercomcdn.com/i/o/753287059/8975e19f85e6e142200ab3cb/image.png?expires=1780444800&signature=8cb4a95bd9609403b80b22414f12db06597a71e9ce4b5e2274a48b9e12fa0ccd&req=cyUkFMF5nYRWFb4V1XW4gX2jt6Vgl9qSfjT24VXDb64APzpQdjj0R8Envv6i%0A8MIBd3MLpy%2BkuENZOt9VYcjFfA%3D%3D%0A)

Wenn Sie Zugang zum Herunterladen von Modellen haben, können Sie auf die Schaltfläche Download klicken, um die neueste Revision eines Modells als ifc-Datei herunterzuladen.

![](https://downloads.intercomcdn.com/i/o/753302330/c3f919e6fdb4c309d8dc153c/image.png?expires=1780444800&signature=68364e89ca512cf8944974867b426888b7b8745b7679bbb060cd78414fe90c82&req=cyUkFcl8noJfFb4V1XW4gYrvct9N55WI18qOS6Dcqn1TREQlwrTdKC%2Fupdi%2F%0AjDnBFssfEKPwxsDENkISQp9IBA%3D%3D%0A)

## 2. Modelle oberste Tasten
Wenn Sie Zugang zum Erstellen oder Herunterladen von Modellen haben, sehen Sie die entsprechenden Schaltflächen oben rechts.

![](https://downloads.intercomcdn.com/i/o/780871412/32f0254d9344b386eadb2548/image.png?expires=1780444800&signature=5b598a3ad96c6f16f534b05a152acbb9a2a62e8aaf6748b830add6716d50a1ca&req=cygnHs5%2FmYBdFb4V1XW4gYJruUiOhPEB91QuPs4N0k2%2BnkbG9Q5Xt%2FlivdCa%0ALgrKbt0ly5C%2BbNyqmnRb8513qg%3D%3D%0A)

- Neues Modell - Erstellen Sie ein neues Modell
- Exporte - Exportieren Sie eine Gruppe von Modellen

## 3. Modell Seite
Für jedes Modell ist die letzte Revision des Modells die Standardauswahl. Wenn Sie auf ein Modell klicken, werden Sie auf die Modellseite weitergeleitet, die zu diesem Modell gehört.
