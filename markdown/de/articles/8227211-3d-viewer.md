# 3D-Viewer

Den 3D-Viewer finden Sie als eines der vier Hauptfelder, die oben rechts auf Ihrem Bildschirm geöffnet werden können. Sie können entweder auf das 3D-Symbol klicken, um dieses Feld zu öffnen, oder Sie können [Umschalt+2](https://support.catenda.com/en/articles/4670267-summary-of-most-important-shortcuts-and-basic-controls#:~:text=3D%20viewer-,Shift%20%2B%202,-Trees%20panel) drücken.

![](https://raw.githubusercontent.com/catenda/help-center/main/images/gkxpq62p/01-intro.png)

Mit einem geladenen Modell kann der 3D-Viewer so aussehen:

![](https://raw.githubusercontent.com/catenda/help-center/main/images/gkxpq62p/02-intro.png)

## 1. **Modelle laden**

Modelle können auf mehrere Arten in den 3D-Viewer geladen werden. Auf der Seite Modelle kann 3D neben dem zu öffnenden Modell angeklickt werden, oder es können mehrere Modelle ausgewählt werden und oben in der Liste auf 3D geklickt werden. Im Bereich Dokumente können Sie Vorschauen von IFC-Dateien und Punktwolken-Dokumenten öffnen, indem Sie in der Dokumentenliste oder im rechten Informationsfeld auf 3D klicken. Sie können den 3D-Viewer auch ohne geladene Modelle öffnen, indem Sie oben rechts auf 3D klicken, um das 3D-Viewer-Feld zu öffnen.

### 1.1 **Revisions-Selector**

Wenn Sie noch keine Modelle geladen haben, können Sie Modelle laden und entladen, indem Sie die Dropdown-Liste des Revisions-Selectors öffnen. Klicken Sie [hier](https://support.catenda.com/en/articles/4670279-revision-selector), um mehr über den Revisions-Selector zu erfahren

![](https://raw.githubusercontent.com/catenda/help-center/main/images/gkxpq62p/03-revision-selector.png)

### 1.2 **Raster-Dropdown**

Wenn es Modelle in Ihren Projekten gibt, die Raster enthalten, können Sie diese mit dieser Dropdown-Liste in den Viewer laden. Klicken Sie [hier](https://support.catenda.com/en/articles/4670327-showing-grids), um mehr über Raster im 3D-Viewer zu erfahren.

![](https://raw.githubusercontent.com/catenda/help-center/main/images/gkxpq62p/04-grid-dropdown.png)

## 2. **Sichtbarkeit**

Das Dropdown-Feld Sichtbarkeit ermöglicht es Ihnen, Objekte anzuzeigen, auszublenden und durchsichtig zu machen.

![](https://raw.githubusercontent.com/catenda/help-center/main/images/gkxpq62p/05-visibility.png)

Wenn Sie das Dropdown-Feld Sichtbarkeit öffnen, kann es so aussehen:

![](https://raw.githubusercontent.com/catenda/help-center/main/images/gkxpq62p/06-visibility.png)

Hier können Sie die folgenden Aktionen durchführen:

### 2.1 **Objekte anzeigen**

**Alles anzeigen** Alle Objekt(e) anzeigen Dies kann nützlich sein, nachdem Sie einen Schnappschuss oder ein Lesezeichen mit versteckter Geometrie abgespielt haben, die Sie sehen möchten.

**Auswahl anzeigen** Zeigt die ausgewählten Objekt(e) an Wenn Sie ein ausgewähltes Objekt ausgeblendet haben, bleibt es ausgewählt, aber versteckt, bis Sie etwas anderes auswählen und es aus Ihrer Auswahl entfernen. Im Informationsfenster können Sie ausgeblendete Objekte aus dem Objektbaum und über Auswahlen auswählen. Im Inhaltsfeld können Sie Objekte aus Eigenschaftsbibliotheken und Objekte auswählen, die mit Problemen oder Dokumenten verknüpft sind.

### 2.2 **Objektdurchsichtigkeit**

Es ist nicht möglich, durchsichtige Objekte anzuwählen, indem man sie im 3D-Viewer anklickt.

**Durchsichtig machen** Macht die ausgewählten Objekt(e) durchsichtig Im Informationsfenster können Sie ausgeblendete Objekte aus dem Objektbaum und über Auswahlen auswählen. Im Inhaltsfeld können Sie Objekte aus Eigenschaftsbibliotheken und Objekte auswählen, die mit Problemen oder Dokumenten verknüpft sind. Ausgewählte durchsichtige Objekte können durch Anzeigen der Auswahl oder Anzeigen aller zur Standardsichtbarkeit zurückgesetzt werden.

**Andere durchsichtig machen** Macht alles außer den ausgewählten Objekt(en) durchsichtig Zeigen Sie alles an, um die Objekte auf die Standardsichtbarkeit zurückzusetzen.

### 2.3 **Objekte ausblenden**

Ausgeblendete Objekt(e) werden noch im Speicher Ihres Systems geladen.

**Ausblenden** Blendet ein Objekt aus dem Viewer aus

**Andere ausblenden** Alle Objekte außer dem ausgewählten ausblenden Dies ähnelt der Isolation von Objekten. Die ausgeblendeten Objekte werden noch im Speicher geladen.

## 3. **Kameraoptionen**

Das Dropdown-Feld Kameraoptionen ermöglicht es Ihnen, Ansichtspunkte, Kameratypen und Ansichten zu wählen.

![](https://raw.githubusercontent.com/catenda/help-center/main/images/gkxpq62p/07-br-camera-options.png)

### 3.1 **Ansichtspunkt**

So kann das Ansichtspunkt-Menü aussehen:

![](https://raw.githubusercontent.com/catenda/help-center/main/images/gkxpq62p/08-viewpoint.png)

_Startseite_ - Setzen Sie den Viewer auf die Standardposition zurück Dies kann helfen, einen Überblick über alle im Viewer geladenen Modelle zu erhalten.

Die folgenden Ansichten ermöglichen es Ihnen, das Modell von verschiedenen Seiten zu sehen:

**Zurück** Das Modell von hinten sehen

**Links** Das Modell von links sehen

**Vorne** Das Modell von vorne sehen

**Rechts** Das Modell von rechts sehen Diese Ansichten können ideal zum Erstellen von Schnitten durch vertikale Schnittebenen sein.

**Oben** Das Modell von oben sehen Diese Ansicht kann ideal für die Erstellung von Grundrissen durch eine horizontale Schnittebene sein.

### 3.2 **Kameratyp**

So kann das Kameratyp-Menü aussehen:

![](https://raw.githubusercontent.com/catenda/help-center/main/images/gkxpq62p/09-camera-type.png)

_Perspektive_ - Standard Ermöglicht Ihnen, das Modell so zu sehen, wie ein Mensch es sieht. Dieser Kameratyp ist ideal, um zu verstehen, wie es sein könnte, sich im Modell zu befinden. Da sich die Kamera im Auge des Menschen befindet, können Sie durch Wände hindurchwandern. Das Sichtfeld für den Perspektivkameratyp kann in den Viewer-Einstellungen angepasst werden.

**Orthografisch** Ermöglicht Ihnen, das Modell so zu sehen, als wären Sie unendlich weit weg. Dieser Kameratyp ist ideal für die Planung, da Abstände leichter zu beurteilen sind. Da sich die Kamera unendlich weit entfernt befindet, kann sie nicht durch Objekte hindurchgehen, und Sie müssen Objekte ausblenden oder durchsichtig machen, um zu sehen, was dahinter liegt.

### 3.3 **Ansicht**

So kann das Ansicht-Menü aussehen:

![](https://raw.githubusercontent.com/catenda/help-center/main/images/gkxpq62p/10-view.png)

> **Hinweis:** Stellen Sie sicher, dass die Hardware und Software, auf der Catenda geöffnet ist, [wie empfohlen](https://support.catenda.com/en/articles/6921941-hardware-recommendation) konfiguriert sind, da dies Auswirkungen auf die Anzeige des Viewers haben kann.

**Standard** Die Standardansicht ist standardmäßig ausgewählt. Dünne Linien mit genauer Objektanzeige. Ideal zum Messen.

Wenn für ein Objekt in der IFC keine Farbe definiert ist, wird dem Objekt eine Farbe basierend auf der für das Objekt konfigurierten Entität zugewiesen. Die Entität kann im Inspektionsmenü des Informationsfensters gefunden werden.

Da es im Catenda-Viewer keine Sonne oder künstliches Licht gibt, werfen Objekte keine Schatten aufeinander. Oberflächen werden durch ihre flache Farbe dargestellt und verdunkelt, je nachdem, ob die Kamera direkt auf sie schaut oder ob die Oberfläche von der Kamera abgewinkelt ist.

**Kontur** Cell Shading verwandelt Linien an Ecken mit Winkeln, die von Ihnen abgewandt sind, in dickere Linien, während Linien an Ecken, die zu Ihnen zeigen, dünnere Linien werden. Dies schafft eine zusätzliche Tiefenschicht, aber die dicken Linien können präzise Messungen überlagern. Kontur ist eine großartige Ansicht für den Export von Bildern.

> **Hinweis:** Stellen Sie sicher, dass die Hardware und Software, auf der Catenda geöffnet ist, [wie empfohlen](https://support.catenda.com/en/articles/6921941-hardware-recommendation) konfiguriert sind, da dies Auswirkungen auf die Schärfe der Linien haben kann.

**Skizze** Ideal für Meetings mit externen Parteien, die eine einfache Anzeige des Modells benötigen. Objekte ohne in der IFC angegebene Farbe werden mit Grautönen und einer Textur gefärbt, sodass die Oberflächen nicht so perfekt flach aussehen. Fenster behalten ihre gewohnte durchsichtige blaue Farbe.

> **Hinweis:** Stellen Sie sicher, dass die Hardware und Software, auf der Catenda geöffnet ist, [wie empfohlen](https://support.catenda.com/en/articles/6921941-hardware-recommendation) konfiguriert sind, da dies Auswirkungen auf die Schärfe der Linien haben kann.

## 5. **Joystick**

Wenn Sie auf diese Schaltfläche klicken, wird der Joystick auf dem Bildschirm angezeigt. Mit dem Joystick können Sie im Perspektivmodus sanft durch das Modell fliegen und im orthografischen Modus um einen zentralen Punkt drehen.

![](https://raw.githubusercontent.com/catenda/help-center/main/images/gkxpq62p/13-joystick.png)

Wenn Sie den Joystick einschalten, werden Sie sehen, dass sich die Schaltfläche ändert und ein Dropdown erhält.

![](https://raw.githubusercontent.com/catenda/help-center/main/images/gkxpq62p/14-joystick.png)

Mit dem Dropdown neben dem Joystick können Sie die Joystick-Position konfigurieren:

- Mittelleft
- Unten links
- Unten Mitte
- Unten rechts
- Mittelrecht

## 8. **Objektinformations-Fußzeile**

Standardmäßig wird eine Fußzeile unten im Viewer angezeigt, wenn Sie ein Objekt auswählen.

![](https://raw.githubusercontent.com/catenda/help-center/main/images/gkxpq62p/17-object-information-footer.png)

Diese Fußzeile kann Informationen über Themen und Bibliothekselemente enthalten, die mit dem Objekt verknüpft sind.

### 8.1 **Identifikation**

Objektname - Objektentität - Objekttyp Wenn mehrere Objekte ausgewählt sind, wird dies "Undefiniert" sagen.

### 8.2 **Themainformationen**

**Offen** Anzahl offener Probleme (Rot) Klicken Sie auf "Offen", um alle offenen Probleme zu sehen, mit denen dieses Objekt im rechten Fenster verknüpft ist.

**Geschlossen** Anzahl geschlossener Probleme (Grün) Klicken Sie auf "Geschlossen", um alle geschlossenen Probleme zu sehen, mit denen dieses Objekt im rechten Fenster verknüpft ist.

### 8.3 **Bibliotheken**

Wenn Sie Ihr ausgewähltes Objekt mit Bibliotheken auf Catenda Hub verknüpft haben, können Sie die Bibliothek mit der Anzahl der Links in der Informationsfußzeile sehen.

**SharePoint** Wenn Sie Ihr ausgewähltes Objekt mit SharePoint-Dokumenten verknüpft haben, sehen Sie hier den Namen der SharePoint-Bibliothek mit der Anzahl der verknüpften Dokumente

**Dokumente** Wenn Sie Ihr ausgewähltes Objekt mit Dokumenten auf der Dokumentenseite verknüpft haben, können Sie diese unter Dokumente und mit der Anzahl der verknüpften Dokumente hier sehen.

**Klassifikation** Wenn Sie Ihr ausgewähltes Objekt mit Klassifikationen verknüpft haben, können Sie hier den Namen der Klassifikation und die Anzahl der verknüpften Klassifikationen sehen.

**Links** Wenn Sie Ihr ausgewähltes Objekt mit Links aus der Links-Bibliothek verknüpft haben, können Sie "Links" und hier die Anzahl der verknüpften Links sehen.

**Dropbox** Wenn Sie Ihr ausgewähltes Objekt mit Dropbox-Dokumenten verknüpft haben, sehen Sie hier den Namen der Dropbox-Bibliothek mit der Anzahl der verknüpften Dokumente.

### 8.4 **Objektinformations-Fußzeile ausblenden**

Klicken Sie auf das X, um die Objektinformations-Fußzeile aus dem 3D-Viewer auszublenden.

### 8.5 **Objektinformations-Fußzeile anzeigen**

Wenn Sie die Informationsfußzeile schließen, können Sie sie mit der i-Schaltfläche unter dem 2D-Viewer oder der 2D-Schaltfläche wieder öffnen.

![](https://raw.githubusercontent.com/catenda/help-center/main/images/gkxpq62p/18-showing-the-object-information-footer.png)

## 9. **Optionsmenü**

Wenn Sie im 3D-Viewer Rechtsklick machen, wird das [Optionsmenü](https://support.catenda.com/en/articles/4670267-summary-of-most-important-shortcuts-and-basic-controls#:~:text=Double%20click-,Options%20menu,-Show%20all) angezeigt.
