## **Modelle laden**

Modelle können auf mehrere Arten in den 3D-Viewer geladen werden.

Auf der Modellseite kann neben dem zu öffnenden Modell auf 3D geklickt werden oder es können mehrere Modelle ausgewählt werden und oben in der Liste auf 3D geklickt werden.

Im Abschnitt Dokumente können Sie Vorschaubilder von IFC-Dateien und Punktwolkendokumenten anzeigen, indem Sie in der Dokumentenliste oder im rechten Informationsfenster auf 3D klicken.

Sie können den 3D-Viewer auch ohne geladene Modelle öffnen, indem Sie oben rechts auf 3D klicken, um das 3D-Viewer-Fenster zu öffnen.

### **Revisions-Auswahl**

Wenn Sie noch keine Modelle geladen haben, können Sie Modelle laden und entladen, indem Sie die Revisions-Auswahl-Dropdown-Liste öffnen.

Klicken Sie [hier](https://support.catenda.com/en/articles/4670279-revision-selector), um mehr über die Revisions-Auswahl zu erfahren

## **Sichtbarkeit**

Das Sichtbarkeitsdropdown ermöglicht es Ihnen, Objekte anzuzeigen, auszublenden und halbtransparent zu machen.

![](https://downloads.intercomcdn.com/i/o/845934711/3df6af991ef5772735f8781c/image.png?expires=1779991200&signature=dca171ed06454fcd8d7e7ae056dfe9f6ff328c59c6edd069312f61b0bd7b607f&req=fCQiH8p6moBeFb4V1XW4gS5O5h2LhmYU%2BwpNf5%2BZRiY1HXXO%2BXzhIhz4bZKf%0AhtYo4yEqD5k2v%2BWTqTEa%2FGl8LA%3D%3D%0A)

Wenn Sie das Sichtbarkeitsdropdown öffnen, kann es wie folgt aussehen:

![](https://downloads.intercomcdn.com/i/o/845941119/e9b39bbe5d117a597aca78df/image.png?expires=1779991200&signature=d8e169d8dfae1ba577f9755f336083babd3c76936c4099e3cd6401d90004a8bd&req=fCQiH81%2FnIBWFb4V1XW4ga%2F7qvKmHR8Z%2FIHcK78bQ1jxtNshfpGDFHFdVN%2BK%0AM6FAYnFYNp95MdveC%2FB69GoGAA%3D%3D%0A)

Hier können Sie die folgenden Aktionen ausführen:

### **Objekte anzeigen**

_Alle anzeigen_

Alle Objekt(e) anzeigen

Dies kann nützlich sein, nachdem Sie einen Snapshot oder ein Lesezeichen abgespielt haben, das Geometrie enthält, die Sie sehen möchten.

_Ausgewählte anzeigen_

Die ausgewählten Objekt(e) anzeigen

Wenn Sie ein ausgewähltes Objekt versteckt haben, ist es immer noch ausgewählt, aber verborgen, bis Sie etwas anderes auswählen und es aus Ihrer Auswahl entfernen.

Im Informationsfenster können Sie versteckte Objekte aus dem Objektbaum und über Auswahlen auswählen.

Im Inhalts-Panel können Sie Objekte aus Eigenschaftsbibliotheken und Objekte auswählen, die mit Problemen oder Dokumenten verknüpft sind.

### **Objekttransparenz**

Es ist nicht möglich, halbtransparente Objekte durch Anklicken im 3D-Viewer auszuwählen.

_Halbtransparent machen_

Die ausgewählten Objekt(e) halbtransparent machen

Im Informationsfenster können Sie versteckte Objekte aus dem Objektbaum und über Auswahlen auswählen.

Im Inhalts-Panel können Sie Objekte aus Eigenschaftsbibliotheken und Objekte auswählen, die mit Problemen oder Dokumenten verknüpft sind.

Ausgewählte halbtransparente Objekte können zur Standardsichtbarkeit zurückgebracht werden, indem Sie Ausgewählte anzeigen oder Alle anzeigen.

_Andere halbtransparent machen_

Alles außer den ausgewählten Objekten halbtransparent machen

Wählen Sie Alle anzeigen, um die Objekte zur Standardsichtbarkeit zurückzubringen.

### **Objekte ausblenden**

Ausgeblendete Objekt(e) werden immer noch in Ihrem Systemspeicher geladen.

_Ausblenden_

Blendet ein Objekt aus dem Viewer aus

_Andere ausblenden_

Blenden Sie alle Objekte außer dem ausgewählten aus

Dies ist ähnlich wie das Isolieren von Objekten.

Die ausgeblendeten Objekte werden immer noch im Speicher geladen.

### **Kameratyp**

So kann das Kameratypmenü aussehen:

![](https://downloads.intercomcdn.com/i/o/areracg3/1771082517/baca855e6cf24a2ece1f5cdb6355/image.png?expires=1779991200&signature=b50dd2ea23844eed61faa7667cb2e61e232fc5cc638b07724736cabf3cd5d5c9&req=dScgF8l2n4ReXvMW3nq%2Bgej5Tr4cvKVsBE5RMsJphVTXCMHJRhT9OHUadSBz%0AhrOa5Z0k77CsA6X%2BDGAR6CPdyCI%3D%0A)

_Perspektive_ - Standard

Ermöglicht es Ihnen, das Modell so zu sehen, wie es ein Mensch tut.

Dieser Kameratyp eignet sich hervorragend zum Verständnis, wie es sein kann, sich im Modell zu befinden.

Da sich die Kamera im menschlichen Auge befindet, können Sie durch Wände dringen.

Das Sichtfeld für den Perspektiv-Kameratyp kann in den Viewer-Einstellungen angepasst werden.

_Orthographisch_

Ermöglicht es Ihnen, das Modell so zu sehen, als wären Sie unendlich weit entfernt.

Dieser Kameratyp eignet sich hervorragend für die Planung, da Abstände leichter zu beurteilen sind.

Da sich die Kamera unendlich weit entfernt befindet, kann sie nicht durch Objekte dringen und Sie müssen Objekte ausblenden oder halbtransparent machen, um zu sehen, was dahinter liegt.

## **Objektinformationsfußzeile**

Standardmäßig wird unten im Viewer eine Fußzeile angezeigt, wenn Sie ein Objekt auswählen.

![](https://downloads.intercomcdn.com/i/o/1019401439/4d388129fa2d03729dc31e76/image.png?expires=1779991200&signature=1e2d5d45ea86b0f1def0eb6a8971448b722b63f290a425a481abb2c141ee540d&req=dSAmH81%2BnIVcUPMW3nq%2BgexpaN7X8l172Nva71O74Sd38%2BeLDn9nwcCn9rt4%0AVVFDa%2BxynIdig3UVk3DaJMm28ts%3D%0A)

Diese Fußzeile kann Informationen zu Themen und Bibliothekselementen enthalten, die mit dem Objekt verknüpft sind.

### **Identifikation**

Objektname - Objektentität - Objekttyp

Wenn mehrere Objekte ausgewählt sind, wird dies als undefiniert angezeigt.

### **Themeninformationen**

_Offen_

Anzahl der offenen Probleme (Rot)

Klicken Sie auf "Offen", um alle offenen Probleme anzuzeigen, bei denen dieses Objekt im rechten Fenster verknüpft ist.

_Geschlossen_

Anzahl der geschlossenen Probleme (Grün)

Klicken Sie auf "Geschlossen", um alle geschlossenen Probleme anzuzeigen, bei denen dieses Objekt im rechten Fenster verknüpft ist.

### **Bibliotheken**

Wenn Sie Ihr ausgewähltes Objekt mit Bibliotheken auf Catenda Hub verknüpft haben, können Sie die Bibliothek mit der Anzahl der Verknüpfungen in der Informationsfußzeile sehen.

_SharePoint_

Wenn Sie Ihr ausgewähltes Objekt mit SharePoint-Dokumenten verknüpft haben, sehen Sie hier den Namen der SharePoint-Bibliothek mit der Anzahl der verknüpften Dokumente

_Dokumente_

Wenn Sie Ihr ausgewähltes Objekt mit Dokumenten auf der Dokumentenseite verknüpft haben, können Sie diese unter Dokumente anzeigen und mit der Anzahl der verknüpften Dokumente.

_Klassifizierung_

Wenn Sie Ihr ausgewähltes Objekt mit Klassifizierungen verknüpft haben, können Sie hier den Namen der Klassifizierung und die Anzahl der verknüpften Klassifizierungen sehen.

_Links_

Wenn Sie Ihr ausgewähltes Objekt mit Links aus der Link-Bibliothek verknüpft haben, können Sie hier "Links" und die Anzahl der verknüpften Links sehen.

_Dropbox_

Wenn Sie Ihr ausgewähltes Objekt mit Dropbox-Dokumenten verknüpft haben, sehen Sie hier den Namen der Dropbox-Bibliothek mit der Anzahl der verknüpften Dokumente.

### **Ausblenden der Objektinformationsfußzeile**

Klicken Sie auf das x, um die Objektinformationsfußzeile aus dem 3D-Viewer auszublenden.
