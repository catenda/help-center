# Schritt 1 - Kontoeinrichtung

> Erstellen Sie Ihr Konto und werden Sie zu einem Projekt eingeladen

Eine Einladung wird Ihnen von Catenda per E-Mail mit einem Link zu den nächsten Schritten der Registrierung gesendet. Mit diesem Link können Sie ein Konto mit Ihren persönlichen Anmeldedaten erstellen.

![](https://raw.githubusercontent.com/catenda/help-center/main/images/m6qttvze/01-intro.png)

Nach Abschluss Ihrer Registrierung können Sie von jedem Webbrowser aus auf Ihre Projekte in Catenda Hub zugreifen. [https://hub.catenda.com/signin](https://hub.catenda.com/signin)

Weitere Informationen: [Anmeldeseite](https://support.catenda.com/en/articles/7891486-sign-in-page), [Kontoerstellung](https://support.catenda.com/en/articles/8412895-creating-an-account), [Startseite](https://support.catenda.com/en/articles/4670260-landing-page)

> **Hinweis:** Finden Sie [Schritt 2](https://support.catenda.com/en/articles/10576075-step-2-interface) unter [Erste Schritte](https://support.catenda.com/en/collections/2657208-getting-started) im Hilfecenter oder im Hilfemenü.
