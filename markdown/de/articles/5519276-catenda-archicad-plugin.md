# Catenda Archicad Plugin

> Installation des Catenda Archicad Plugins

> **Hinweis:** Die Installationsdatei für das Plugin finden Sie in [diesem Hilfeartikel](https://intercom.help/bimsync-arena/en/articles/8396532-catenda-plugins-and-integrations).

## 1. Installation

Wenn Sie das Catenda Archicad Plugin unter Windows installieren, erscheinen die Installationsdateien im Ordner "C:\\Programme\\Catenda\\Catenda Archicad Connection\\\<Archicad Version>\\Add-On" und werden beim nächsten Öffnen von Archicad im Add-On Manager aktiviert.

Beachten Sie, dass dies ein anderer Ordner ist als der Standard-Add-On-Ordner, der sich unter "C:\\Programme\\Graphisoft\\\<Archicad Version>\\Add-Ons" befindet.

## 2. **Catenda Palette**

Nach der Installation des Plugins wird in der oberen Leiste ein Catenda-Menüreiter angezeigt.

In diesem Menü finden Sie das Fenster "Catenda Hub Issue Manager", das die "Catenda Hub Issue Manager Palette" enthält. Um diese Palette nutzen zu können, muss entweder ein Grundriss oder eine 3D-Ansicht geöffnet werden.

- In der Palette können Sie:

- Zu Ihrem Catenda Hub Projekt navigieren

- Probleme sehen und erstellen

## 3. Einloggen

Wenn Sie die Catenda-Palette öffnen, sehen Sie als erstes die Anmeldeseite.

Die Anmeldeseite kann etwa so aussehen:

![](https://downloads.intercomcdn.com/i/o/areracg3/1587423600/d3953867b4abeb67aef280cab0b9/image.png?expires=1781092800&signature=5b5ee810d57bf9a02bc395a1e3817d67e481d60949c40203223503eca6729ea8&req=dSUvEc18nodfWfMW3nq%2Bgf9CS06s51RiyxFYyEBwLgv%2BBOWqPw7AyZ2KxtZB%0A53gvtErffDzEoa1OiZNtrVnwbvs%3D%0A)

Wenn Sie noch kein Catenda-Konto haben, können Sie sich oben rechts kostenlos anmelden.

Wenn Sie bereits ein Catenda-Konto haben, können Sie Ihre E-Mail und Ihr Passwort eingeben und auf Anmelden klicken.

Nach der Anmeldung werden Sie aufgefordert, den Zugriff auf Ihr Catenda-Konto zu gewähren.

Nachdem Sie diesen Zugang gewährt haben, wird das Archicad Plugin als Anwendung auf der Anwendungsseite Ihres Catenda-Kontos angezeigt.

Hier können Sie den Zugriff jederzeit widerrufen, wenn Sie ihn nicht mehr gewähren möchten.

### 3.1 Passwort zurücksetzen

Wenn Sie Ihr Passwort vergessen haben, können Sie auf Ich habe mein Passwort vergessen klicken, um es zurückzusetzen.

Die Seite zum Zurücksetzen des Passworts kann etwa so aussehen:

![](https://downloads.intercomcdn.com/i/o/areracg3/1587423602/90e22859394f297dcff4fef81f08/image.png?expires=1781092800&signature=d7ce03c20a2f88762215cee2c44f3ed1bd6a0478620c186d7a37bc35149543d4&req=dSUvEc18nodfW%2FMW3nq%2BgQ%2FuxHH4TBePOQgT7vdc4MYSlayCS5ezbtWl8aoi%0Ar4zs3d34rzMQD1%2BYmaMz2ZWKSTM%3D%0A)

Wenn Sie Ihre E-Mail-Adresse eingeben und auf "Bestätigungs-E-Mail senden" klicken, erhalten Sie eine E-Mail, die Sie durch das Zurücksetzen Ihres Passworts führt.

Bitte stellen Sie sicher, dass Sie diese E-Mail innerhalb von 5 Minuten erhalten haben.

Wenn Sie sie nicht in Ihrem Posteingang sehen, versuchen Sie es in Ihrem Spam- oder Junk-Postfach.

Wenn es länger als 5 Minuten dauert, wenden Sie sich bitte an den Support unter support@catenda.com

Um sich wieder anzumelden, klicken Sie oben rechts auf "Anmelden".

Modelle und Revisionen

So kann die Seite Modelle und Revisionen aussehen.

Hier können Sie alle Modelle sehen, auf die Sie in Ihrem Catenda-Projekt Zugriff haben.

![](https://downloads.intercomcdn.com/i/o/areracg3/1587423609/9221f7ebe6fdefa33b7c5753f224/image.png?expires=1781092800&signature=6fc2f6904deba96539ec30f43ff2f094693fdec721061d16ac7b48ccb6c55aac&req=dSUvEc18nodfUPMW3nq%2BgXjhpudqZPawaQmbuXxWXO4JPKfopvLUw7ShQDYr%0Ax0hyvUfoe6YKcchKz7R7qBPK3GM%3D%0A)

Der Zeitstempel des Modells wird relativ sein. Bewegen Sie den Mauszeiger über den Zeitstempel, um genaue Informationen darüber zu erhalten, wann die Revision veröffentlicht wurde.

Klicken Sie auf die Pfeilschaltfläche neben einer der Modellrevisionen, um die IFC-Datei, die als Revision hochgeladen wurde, zu importieren.

Wenn Sie keinen Zugriff auf Modelle in Ihrem Catenda-Projekt haben, sehen Sie folgendes:

![](https://downloads.intercomcdn.com/i/o/areracg3/1587423616/5a65ab1db179a074392f248af1a8/image.png?expires=1781092800&signature=eddb334d0538cfcf901d695a261cafd6c1516b5e72f129fd585cbd6925f201e9&req=dSUvEc18nodeX%2FMW3nq%2BgTBuKJUrnlNk1%2FFyoIaJYEEEoqMqXuuu91dro5Sx%0AS%2FhzmxBrIAn4loxQVU%2BElLS2PU8%3D%0A)

IFC hochladen

Laden Sie eine IFC aus Ihrem aktuellen Archicad-Projekt hoch

So kann die Seite IFC hochladen aussehen:

![](https://downloads.intercomcdn.com/i/o/areracg3/1587423611/13f541b9faec8b3baac4b9125c2f/image.png?expires=1781092800&signature=063f22698b4ff84a4a76e68c37141042299281c5bc64a0ac5da5b12061bcd0a7&req=dSUvEc18nodeWPMW3nq%2BgUmYPo4As27WbxQhv44v%2BzUMM4egbkW63LbxqVXf%0AcHQosFa5Caf7ujd4oit42Ju3SWE%3D%0A)

Standardmäßig ist der Dateiname der Name des Archicad-Projekts, der auch der Name der Revision in Catenda ist.

Wählen Sie aus, in welches Modell Sie Ihre Datei hochladen möchten.

Klicken Sie auf Einstellungen, um den Archicad IFC-Exportdialog zu öffnen.

Hier können Sie die Einstellungen konfigurieren, die Sie für den IFC-Export nach Catenda verwenden möchten.

Wenn Sie bereit sind, klicken Sie auf die Schaltfläche Hochladen, um ein Modell hochzuladen.

**Version**

Die Versionsnummer des Plugins.

**Logout**

Melden Sie sich von Ihrem Catenda-Konto ab
