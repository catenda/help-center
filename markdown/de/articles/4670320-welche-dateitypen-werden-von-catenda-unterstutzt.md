# Welche Dateitypen werden von Catenda unterstützt?

Für unseren Modellbetrachter unterstützen wir nur IFCs. Die folgenden Dateitypen können in den Dokumentenbereich hochgeladen und dort in der Vorschau angezeigt werden:

> **Anmerkung:** Die IFC- und Punktwolkenvorschau (e57 und las) kann in den 3D-Viewer geladen werden.

### Offene Dateiformate

Bei Catenda arbeiten wir gerne mit offenen Standards, damit alle unabhängig von der Software, die sie besitzen, zusammenarbeiten können.

Normalerweise kann sich jeder an ihrer Entwicklung beteiligen, da sie von Natur aus offen sind ([Wikipedia](https://en.wikipedia.org/wiki/Open_standard#:~:text=Typically%2C%20anybody%20can%20participate%20in%20their%20development%20due%20to%20their%20inherently%20open%20nature.)).

Das bedeutet, dass nicht nur eine Organisation kontrolliert, was entwickelt wird.

### **IFC versions**

Je nach Status der IFC-Version unterstützen wir die Version auf unterschiedliche Weise:

Offiziell, Im Ruhestand - Volle Unterstützung. Alle offiziellen und zurückgezogenen IFC-Versionen werden unterstützt.

Withdrawn - Teilweise Unterstützung. Die Modelle werden gemäß der letzten offiziellen Version verarbeitet.

In Entwicklung - Keine Unterstützung. Diese können sich noch ändern, bevor sie veröffentlicht werden.

Welche IFC-Versionen offiziell sind, finden Sie [hier](https://technical.buildingsmart.org/standards/ifc/ifc-schema-specifications/)

Offizielle BuildingSMART-Beispiel-IFC-Dateien finden Sie [hier](https://github.com/buildingSMART/Sample-Test-Files)

Punkte in Punktwolken können in der Vorschau entweder mit Farbe oder Intensität dargestellt werden.

## 1. Proprietäre Dateiformate

Es gibt eine Vielzahl von Dateiformatstandards, die von Organisationen entwickelt wurden und von vielen genutzt werden.

Für uns ist es wichtig, dass Catenda einfach zu benutzen ist. Deshalb tun wir unser Bestes, um möglichst viele der gebräuchlichsten Dateitypen zu unterstützen, die verwendet werden.
