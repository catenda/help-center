# Seite Berichtvorlagen

> **Hinweis 1:** Die Berichtsseite ist eine Opt-in-Funktion, die für laufende Projekte aktiviert werden kann. **Hinweis 2:** Neue Projekte, die auf Basis eines Vorlagenprojekts erstellt werden, bei dem diese Funktion aktiviert ist, haben diese Funktion nicht aktiviert.

Die Seite Berichtvorlagen finden Sie als Unterseite der [Berichtsseite](https://support.catenda.com/en/articles/12303098-reports-page) im linken Navigationsmenü. Auf der Seite Berichtvorlagen können Dokumente und Topic-Vorlagen verwaltet werden, die für das Projekt verfügbar sind. _Erforderlicher Zugriff:_ Administrator

Die Seite Berichtvorlagen kann ungefähr so aussehen:

![](https://raw.githubusercontent.com/catenda/help-center/main/images/5aw6t2k2/01-intro.png)

## 1. **Aktionen auf der Seite Berichtvorlagen**

Die Aktionen auf der Seite Berichtvorlagen können durch Klicken auf das Aktionsmenü oben rechts auf der Seite Berichtvorlagen aufgerufen werden. Die Aktionen auf der Seite Berichtvorlagen können so aussehen:

![](https://raw.githubusercontent.com/catenda/help-center/main/images/5aw6t2k2/02-actions-on-the-report-templates-page.png)

### 1.1 **Neue Topic-Vorlage**

Verwenden Sie die Aktion Neue Topic-Vorlage, um eine neue Topic-Vorlage zu erstellen

### 1.2 **Neue Dokumentvorlage**

Verwenden Sie die Aktion Neue Dokumentvorlage, um eine neue Dokumentvorlage zu erstellen

## 2. **Nach Name filtern**

Geben Sie in der Suchleiste ein, um die Berichtvorlagentabelle nach den Namen der Berichte zu filtern.

![](https://raw.githubusercontent.com/catenda/help-center/main/images/5aw6t2k2/03-filter-by-name.png)

Textsuche - `search=test`

## 3. **Berichtvorlagentabelle**

Sobald eine Berichtvorlage erstellt wird, wird sie als Zeile in der Berichtvorlagentabelle angezeigt. Die Berichtvorlagentabelle kann so aussehen:

![](https://raw.githubusercontent.com/catenda/help-center/main/images/5aw6t2k2/04-report-templates-table.png)

Klicken Sie [hier](https://support.catenda.com/en/articles/11748020-tables-on-catenda), um mehr darüber zu erfahren, wie Sie mit Tabellen in Catenda arbeiten.

### 3.1 **Zeileninhalt**

**Berichtszeile** Obwohl es möglich ist, auf eine Zeile zu klicken, um sie auszuwählen und grün hervorzuheben, können durch Auswahl einer Zeile keine Aktionen ausgeführt werden. Die Zeile wird ausgewählt, wenn das Zeilenaktionsmenü in der rechten Spalte ausgewählt wird.

### 3.2 **Spalten**

Basierend auf der konfigurierten Spaltenreihenfolge werden die ersten Spalten angezeigt, während die Tabelle möglicherweise nach links gescrollt werden muss, um andere aktivierte Spalten anzuzeigen. Die standardmäßige Reihenfolge und Sichtbarkeitseinstellung der Spalten auf der Seite Berichtvorlagen ist wie folgt:

Name - _Standard_ Der Name der Berichtvorlage

_Erstellt von_ - _Standard_ Der Ersteller der Berichtvorlage

Erstellt - _Standard_ Uhrzeit und Datum der Erstellung der Topic-Vorlage

Typ - Standard Der Typ der Berichtvorlage, entweder Dokumente oder Topics.

Aktionsmenü - Standard Klicken Sie auf das Aktionsmenü in der Aktionsmenü-Spalte, um das Zeilenaktionsmenü zu öffnen. Das Zeilenaktionsmenü für die Berichtvorlage kann so aussehen:

![](https://raw.githubusercontent.com/catenda/help-center/main/images/5aw6t2k2/05-columns.png)

Bearbeiten Verwenden Sie die Aktion Bearbeiten, um die ausgewählte Berichtvorlage zu bearbeiten.

Download Verwenden Sie die Aktion Herunterladen, um die ausgewählte Berichtvorlage mit dem Dateiformat `.mrt` herunterzuladen, das in einem beliebigen anderen Projekt erneut importiert werden kann.

Löschen Verwenden Sie die Aktion Löschen, um die ausgewählte Berichtvorlage zu löschen.
