# Aperçu des types de fichiers sur Catenda Hub

Pour notre visionneuse de modèles, nous ne supportons que les IFC. Tous les types de fichiers peuvent être téléchargés dans notre section documents.

> **Remarque :** Les aperçus IFC, Point cloud (e57, las) et GML peuvent être chargés dans la visionneuse 3D.

## 1. **Formats neutres du point de vue du fournisseur**

Chez Catenda, nous aimons travailler avec des normes ouvertes pour que chacun puisse collaborer quel que soit le logiciel dont il dispose.

_En général, tout le monde peut participer à leur développement en raison de leur nature intrinsèquement ouverte ([Wikipedia](https://en.wikipedia.org/wiki/Open_standard#:~:text=Typically%2C%20anybody%20can%20participate%20in%20their%20development%20due%20to%20their%20inherently%20open%20nature.))_

Cela signifie qu'une seule organisation ne contrôle pas ce qui est développé.

### 2.1 **Versions IFC**

Selon le statut de la version IFC, nous supportons la version de différentes manières : _Officiel, Retiré_ - Support complet. Toutes les versions IFC officielles et retirées sont supportées. _Retiré_ - Support partiel. Les modèles sont traités selon la dernière version officielle. _En cours de développement_ - Aucun support. Ceux-ci peuvent encore changer avant d'être publiés.

Vous pouvez trouver les versions IFC officielles [ici](https://technical.buildingsmart.org/standards/ifc/ifc-schema-specifications/) Les fichiers IFC d'exemple BuildingSMART officiels peuvent être trouvés [ici](https://github.com/buildingSMART/Sample-Test-Files)

### 2.2 **Nuages de points**

Les points dans les nuages de points peuvent être visualisés soit avec la couleur, soit avec l'intensité.
