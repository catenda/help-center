# Quels formats sont compatibles avec Catenda?

> formats lecturés par Catenda Hub; formats disponibles; formats acceptés;

3D : Notre visionneuse est compatible uniquement avec l'IFC.

Concernant le stockage, tout format est accepté. Ceci-dit en lecture, on traite :

- Images
- DWG
- DXF
- Office format
- Nuage de point (seulement **.e57** et **.las**)
- IFC
