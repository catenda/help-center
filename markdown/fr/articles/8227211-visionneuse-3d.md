# Visionneuse 3D

Vous trouverez la visionneuse 3D parmi les quatre panneaux principaux qui peuvent être ouverts en haut à droite de votre écran. Vous pouvez soit cliquer sur l'icône 3D pour ouvrir ce panneau, soit appuyer sur [Maj+2](https://support.catenda.com/en/articles/4670267-summary-of-most-important-shortcuts-and-basic-controls#:~:text=3D%20viewer-,Shift%20%2B%202,-Trees%20panel).

![](https://raw.githubusercontent.com/catenda/help-center/main/images/gkxpq62p/01-intro.png)

Avec un modèle chargé, la visionneuse 3D peut ressembler à ceci :

![](https://raw.githubusercontent.com/catenda/help-center/main/images/gkxpq62p/02-intro.png)

## 1. **Chargement de modèles**

Les modèles peuvent être chargés dans la visionneuse 3D de plusieurs façons. Sur la page des modèles, vous pouvez cliquer sur 3D à côté du modèle à ouvrir, ou sélectionner plusieurs modèles et cliquer sur 3D en haut de la liste. Dans la section documents, vous pouvez ouvrir des aperçus de fichiers IFC et de documents de nuage de points en cliquant sur 3D dans la vue de liste des documents ou dans le panneau d'information de droite. Vous pouvez également ouvrir la visionneuse 3D sans charger de modèles en cliquant sur 3D en haut à droite pour ouvrir le panneau de la visionneuse 3D.

### 1.1 **Sélecteur de révision**

Si vous n'avez pas encore chargé de modèles, vous pourrez charger et décharger des modèles en ouvrant la liste déroulante du sélecteur de révision. Cliquez [ici](https://support.catenda.com/en/articles/4670279-revision-selector) pour en savoir plus sur le sélecteur de révision

![](https://raw.githubusercontent.com/catenda/help-center/main/images/gkxpq62p/03-revision-selector.png)

### 1.2 **Liste déroulante de la grille**

S'il y a des modèles dans vos projets qui contiennent des grilles, vous pourrez les charger dans la visionneuse avec cette liste déroulante. Cliquez [ici](https://support.catenda.com/en/articles/4670327-showing-grids) pour en savoir plus sur les grilles dans la visionneuse 3D.

![](https://raw.githubusercontent.com/catenda/help-center/main/images/gkxpq62p/04-grid-dropdown.png)

## 2. **Visibilité**

La liste déroulante de visibilité vous permet d'afficher, de masquer et de rendre les objets translucides.

![](https://raw.githubusercontent.com/catenda/help-center/main/images/gkxpq62p/05-visibility.png)

Quand vous ouvrez la liste déroulante de visibilité, elle peut ressembler à ceci :

![](https://raw.githubusercontent.com/catenda/help-center/main/images/gkxpq62p/06-visibility.png)

Voici les actions que vous pourrez effectuer :

### 2.1 **Afficher les objets**

**Afficher tout** Afficher tous les objet(s). Cela peut être utile après la lecture d'une capture ou d'un signet qui a géométrie cachée que vous aimeriez voir.

**Afficher la sélection** Afficher les objet(s) sélectionné(s). Si vous avez masqué un objet sélectionné, il reste sélectionné mais masqué jusqu'à ce que vous sélectionniez autre chose et le retiriez de votre sélection. Dans le panneau d'information, vous pourrez sélectionner les objets masqués à partir de l'arborescence des objets et via les sélections. Dans le panneau de contenu, vous pourrez sélectionner les objets à partir des bibliothèques de propriétés et des objets liés à des problèmes ou des documents.

### 2.2 **Translucidité des objets**

Il n'est pas possible de sélectionner des objets translucides en cliquant sur eux dans la visionneuse 3D.

**Rendre translucide** Rendre les objet(s) sélectionné(s) translucides. Dans le panneau d'information, vous pourrez sélectionner les objets masqués à partir de l'arborescence des objets et via les sélections. Dans le panneau de contenu, vous pourrez sélectionner les objets à partir des bibliothèques de propriétés et des objets liés à des problèmes ou des documents. Les objets translucides sélectionnés peuvent être ramenés à la visibilité par défaut en affichant la sélection ou en affichant tous les objets.

**Rendre les autres translucides** Rendre tous les objets sauf les objet(s) sélectionné(s) translucides. Cliquez sur Afficher tout pour ramener les objets à la visibilité par défaut.

### 2.3 **Masquer les objets**

Les objet(s) masqué(s) resteront chargé(s) dans la mémoire de votre système.

**Masquer** Masque un objet de la visionneuse

**Masquer les autres** Masquer tous les objets sauf l'objet sélectionné. Ceci est similaire à l'isolement des objets. Les objets masqués resteront chargés en mémoire.

## 3. **Options de caméra**

La liste déroulante des options de caméra vous permet de choisir les points de vue, les types de caméra et les vues.

![](https://raw.githubusercontent.com/catenda/help-center/main/images/gkxpq62p/07-br-camera-options.png)

### 3.1 **Point de vue**

Voici ce à quoi le menu des points de vue peut ressembler :

![](https://raw.githubusercontent.com/catenda/help-center/main/images/gkxpq62p/08-viewpoint.png)

_Accueil_ - Réinitialiser la visionneuse à la position par défaut. Cela peut vous aider à avoir un aperçu de tous les modèles chargés dans la visionneuse.

Les vues suivantes vous permettent de voir le modèle sous différents angles :

**Arrière** Voir le modèle par l'arrière

**Gauche** Voir le modèle de gauche

**Avant** Voir le modèle par l'avant

**Droite** Voir le modèle de droite. Ces vues peuvent être excellentes pour créer des sections en créant des plans de coupe verticaux.

**Dessus** Voir le modèle par le dessus. Cette vue peut être excellente pour produire des plans d'étage en créant un plan de coupe horizontal.

### 3.2 **Type de caméra**

Voici ce à quoi le menu du type de caméra peut ressembler :

![](https://raw.githubusercontent.com/catenda/help-center/main/images/gkxpq62p/09-camera-type.png)

_Perspective_ - Par défaut. Vous permet de voir le modèle comme un humain le ferait. Ce type de caméra est idéal pour comprendre ce que cela pourrait être d'être dans le modèle. Comme la caméra se trouve au niveau des yeux humains, vous pouvez vous déplacer à travers les murs. Le champ de vision pour le type de caméra en perspective peut être ajusté dans les paramètres de la visionneuse.

**Orthographique** Vous permet de voir le modèle comme si vous étiez infiniment loin. Ce type de caméra est idéal pour la planification car les distances sont plus faciles à juger. Comme la caméra est infiniment loin, elle ne peut pas traverser les objets et vous devrez masquer ou rendre les objets translucides pour voir ce qui se trouve derrière eux.

### 3.3 **Vue**

Voici ce à quoi le menu de vue peut ressembler :

![](https://raw.githubusercontent.com/catenda/help-center/main/images/gkxpq62p/10-view.png)

> **Remarque :** Assurez-vous que le matériel et le logiciel sur lequel Catenda est ouvert sont configurés [comme recommandé](https://support.catenda.com/en/articles/6921941-hardware-recommendation), car cela peut affecter la façon dont la visionneuse s'affiche.

**Normal** Par défaut, la vue normale est sélectionnée. Lignes fines pour un affichage d'objets précis. Idéal pour les mesures.

Quand aucune couleur n'est définie pour un objet dans l'IFC, l'objet reçoit une couleur attribuée en fonction de l'entité configurée pour l'objet. L'entité se trouve dans le menu d'inspection du panneau d'information.

Comme il n'y a pas de soleil ou de lumière artificielle dans la visionneuse Catenda, les objets ne projettent pas d'ombre les uns sur les autres. Les surfaces sont rendues par leur couleur plate assombrie selon que la caméra regarde droit devant elle ou si la surface est inclinée par rapport à la caméra.

**Contour** L'ombrage par cellules transforme les lignes sur les coins avec des angles vous éloignant en des lignes plus épaisses, tandis que les lignes sur les coins vous faisant face deviennent des lignes plus fines. Cela crée une couche de profondeur supplémentaire, mais les lignes épaisses pourraient couvrir les mesures précises. Le contour est une excellente vue pour exporter des images.

> **Remarque :** Assurez-vous que le matériel et le logiciel sur lequel Catenda est ouvert sont configurés [comme recommandé](https://support.catenda.com/en/articles/6921941-hardware-recommendation), car cela peut affecter la netteté des lignes.

**Croquis** Idéal pour les réunions avec des parties externes qui ont besoin d'un affichage simple du modèle. Les objets sans couleur spécifiée dans l'IFC sont colorés en teintes de gris et ont une texture de sorte que les surfaces ne paraissent pas parfaitement lisses. Les fenêtres garderont leur couleur bleu transparent habituelle.

> **Remarque :** Assurez-vous que le matériel et le logiciel sur lequel Catenda est ouvert sont configurés [comme recommandé](https://support.catenda.com/en/articles/6921941-hardware-recommendation), car cela peut affecter la netteté des lignes.

## 5. **Joystick**

Cliquer sur ce bouton affiche le joystick à l'écran. Le joystick vous permet de naviguer facilement à travers le modèle en mode perspective et de tourner autour d'un point central en mode orthographique.

![](https://raw.githubusercontent.com/catenda/help-center/main/images/gkxpq62p/13-joystick.png)

Si vous activez le joystick, vous verrez que le bouton change et obtient une liste déroulante.

![](https://raw.githubusercontent.com/catenda/help-center/main/images/gkxpq62p/14-joystick.png)

Avec la liste déroulante à côté du joystick, vous pouvez configurer l'emplacement du joystick :

- Milieu gauche
- Bas gauche
- Bas center
- Bas droite
- Milieu droit

## 8. **Pied de page des informations sur l'objet**

Par défaut, un pied de page s'affiche en bas de la visionneuse quand vous sélectionnez un objet.

![](https://raw.githubusercontent.com/catenda/help-center/main/images/gkxpq62p/17-object-information-footer.png)

Ce pied de page peut contenir des informations sur les sujets et les éléments de bibliothèque liés à l'objet.

### 8.1 **Identification**

Nom de l'objet - Entité d'objet - Type d'objet. Si plusieurs objets sont sélectionnés, cela affichera indéfini.

### 8.2 **Informations sur le sujet**

**Ouvert** Nombre de problèmes ouverts (Rouge). Cliquez sur "Ouvert" pour voir tous les problèmes ouverts où cet objet est lié dans le panneau droit.

**Fermé** Nombre de problèmes fermés (Vert). Cliquez sur "Fermé" pour voir tous les problèmes fermés où cet objet est lié dans le panneau droit.

### 8.3 **Bibliothèques**

Si vous avez lié votre objet sélectionné à des bibliothèques sur Catenda Hub, vous pourrez voir la bibliothèque avec le nombre de liens dans le pied de page des informations.

**SharePoint** Si vous avez lié votre objet sélectionné à des documents SharePoint, vous verrez le nom de la bibliothèque SharePoint avec le nombre de documents liés ici

**Documents** Si vous avez lié votre objet sélectionné à des documents sur la page de documents, vous pourrez les voir sous documents et avec le nombre de documents liés ici.

**Classification** Si vous avez lié votre objet sélectionné à des classifications, vous pourrez voir le nom de la classification et le nombre de classifications liées ici.

**Liens** Si vous avez lié votre objet sélectionné à des liens de la bibliothèque de liens, vous pourrez voir "Liens" et le nombre de liens liés ici.

**Dropbox** Si vous avez lié votre objet sélectionné à des documents Dropbox, vous verrez le nom de la bibliothèque Dropbox avec le nombre de documents liés ici.

### 8.4 **Masquage du pied de page des informations sur l'objet**

Cliquez sur le x pour masquer le pied de page des informations sur l'objet de la visionneuse 3D.

### 8.5 **Affichage du pied de page des informations sur l'objet**

Si vous avez choisi de fermer le pied de page des informations, vous pouvez le rouvrir avec le bouton i en dessous de la visionneuse 2D ou du bouton 2D.

![](https://raw.githubusercontent.com/catenda/help-center/main/images/gkxpq62p/18-showing-the-object-information-footer.png)
