## 1. **Chargement des modèles**

Il est possible de charger des modèles dans la visionneuse 3D de plusieurs façons : sur les modèles, vous pouvez cliquer sur le bouton 3D à côté du modèle que vous souhaitez ouvrir ou sélectionner plusieurs modèles et cliquer sur le bouton 3D en haut de la liste.

Dans la section des documents, vous pouvez ouvrir l'aperçu des fichiers .IFC ou des nuages de points (.LAS, .e57) en cliquant simplement sur le titre du document. Vous pouvez également ouvrir la visionneuse 3D sans charger de modèles en cliquant sur le bouton 3D en haut à droite pour ouvrir le panneau de la visionneuse 3D.

### 1.1 **Sélecteur de révisions**

Si vous n'avez pas encore chargé de modèles, vous pourrez charger et décharger des modèles en ouvrant la liste déroulante du sélecteur de révision. Cliquez [ici](https://support.catenda.com/fr/articles/4670279-selecteur-de-revision) pour en savoir plus sur les grilles dans la visionneuse 3D.

### 1.2 **Menu déroulant des grilles (IfcGrid)**

Si vos projets contiennent des modèles avec des grilles, vous pourrez les charger dans la visionneuse à l'aide de cette liste déroulante.

Cliquez [ici](https://support.catenda.com/fr/articles/4670327-rendre-visibles-les-quadrillages-dans-bimsync) pour en savoir plus sur les grilles dans le visualiseur 3D.

## 2. **Options de visibilité**

La liste déroulante de visibilité permet d'afficher, de masquer et de rendre les objets transparents.

![](https://downloads.intercomcdn.com/i/o/areracg3/1299359831/4b790c2f03f1dba860466ed86a0e/image.png?expires=1781092800&signature=a4487f5d27243eab1966e1cfa7677a334cbe5c34a211590e2c29815aa494ac9d&req=dSIuH8p7lIlcWPMW3nq%2Bgez3m388LDCZr0l0HZfKaBsQ5Jr%2F8T164iD4Y7Bc%0Azws9xNKLV27vZh%2F381HV8Zebqec%3D%0A)

Lorsque vous déroulez les options de visibilité, cette liste apparaît :

![](https://downloads.intercomcdn.com/i/o/areracg3/1299561142/b51245caf68098100959f6053355/image.png?expires=1781092800&signature=4eeaf6e35d79661694759e26d2186d99482a668f026caa3128b4640a750489ab&req=dSIuH8x4nIBbW%2FMW3nq%2Bge%2B%2F%2B05Q4nC%2Fm%2FO5yFBqonrhK3MOqBqFwpF2hsyy%0AetSMQ3noLF1ocpnvjFQg5EjwfOI%3D%0A)

Vous pourrez effectuer les actions suivantes :

### 2.1 **Montrer des objets**

_Tout afficher_ - Montrer tous les objets

Cette fonction peut s'avérer utile après la lecture d'un snapshot ou d'un signet contenant une ou plusieurs géométrie(s) cachée(s) que vous aimeriez voir.

_Afficher la sélection_ - Montrer le ou les objet(s) sélectionné(s)

Si vous avez caché un objet sélectionné, cet objet reste sélectionné mais caché jusqu'à ce que vous sélectionniez quelque chose d'autre et/ou que vous le retiriez de votre sélection.

Dans le panneau d'information, vous pourrez sélectionner des objets masqués depuis l'arbre des objets.

Dans le panneau de contenu, vous pourrez sélectionner des objets depuis les bibliothèques de propriétés et aussi des objets liés à des sujets ou à des documents.

### 2.2 **Rendre des objets transparent**

> **Note:** **NB**: Il n'est pas possible de sélectionner des objets transparents en cliquant dessus dans la visionneuse 3D.

_Rendre transparent_ - Rendre les objets sélectionnés transparent

Dans le panneau d'information, vous pourrez sélectionner des objets masqués depuis l'arbre des objets.

Dans le panneau de contenu, vous pourrez sélectionner des objets depuis les bibliothèques de propriétés et aussi des objets liés à des sujets ou à des documents.

Les objets transparents sélectionnés peuvent être rendus de noveau visible en affichant les objets sélectionnés ou en affichant tous les objets.

_Rendre les autres objets transparents_  - Rendre tout transparent sauf les objets sélectionnés. Cliquez sur "Tout afficher" pour retrouver la visibilité par défaut de tous les objets.

### 2.3 **Masquer des objects**

Les objets masqués seront toujours chargés dans la mémoire de votre système.

_Masquer_ - Masque un objet ou plusieurs objets de la visionneuse

_Masquer les autres objets_ - Masque tous les objets sauf l'objet sélectionné. Cette action est similaire à l'isolation des objets. Les objets cachés seront toujours chargés en mémoire.

## 3. **Options de caméra**

Le menu déroulant des options de la caméra vous permet de choisir différents points de vue, des types de caméra et ausi des styles de vue.

![](https://downloads.intercomcdn.com/i/o/areracg3/1299359838/cbd1821dd929d9ec0e1dc0568f1a/image.png?expires=1781092800&signature=1917e7c66c9a2fa96a9a87bd7dd37a1bfe2b5999a9d042b77605ab3ecb603584&req=dSIuH8p7lIlcUfMW3nq%2BgfFNs1XC9BhmBiSqDJimFa8Uj80a587weeUoheDc%0As767XxTc0gRsSAea00eugQWwMBI%3D%0A)

Le menu déroulant des options de la caméra ressemble à ceci :

![](https://downloads.intercomcdn.com/i/o/areracg3/1299796969/306e5696de820434417d64f9a80a/image.png?expires=1781092800&signature=de2acaa5c72ac835af27c63bc70a5f044858abf3917f671c4f3b822f1aed9104&req=dSIuH853m4hZUPMW3nq%2BgUm2QEPl5JRHW4VUjxp%2FD4FAfWxgG9peyhgzkBn9%0Agz%2FazZPdUKHpwfSyzmRSz67eYG4%3D%0A)

### 3.1 **Points de vue**

_Accueil_ - Réinitialise la visionneuse à sa position par défaut.

Cela permet d'avoir une vue d'ensemble de tous les modèles chargés dans la visionneuse.

Les vues suivantes vous permettent de voir le modèle sous différents angles :

_Retour_ - Vue arrière du modèle

_Gauche_ - Visualiser le modèle depuis le côté gauche

_Avant_ - Vue avant du modèle

_Droite_ - Visualiser le modèle depuis le côté droit

Ces vues peuvent être utiles pour créer des sections en créant des plans de coupe verticaux.

_Dessus_ - Visualiser le modèle depuis le dessus

Cette vue peut être très utile pour produire des plans d'étage en créant un plan de coupe horizontal.

### 3.2 **Types de caméra**

_Perspective_ - Par défaut - Permet de voir le modèle comme une personne.

Ce type de caméra est idéal pour comprendre ce que l'on peut ressentir dans le modèle.

Comme la caméra se trouve dans l'œil de l'homme, il est possible de passer à travers les murs.

Le champ de vision pour le type de caméra perspective peut être ajusté dans les paramètres de la visionneuse.

_Orthographique_ - Vous permet de voir le modèle comme si vous étiez à une distance infinie.

Ce type de caméra est idéal pour la planification, car les distances sont plus faciles à évaluer.

Comme la caméra est infiniment éloignée, elle ne peut pas couper à travers les objets et vous devrez cacher les objets ou les rendre translucides pour voir ce qu'il y a derrière.

### 3.3 **Styles de vue**

_Normal_ - Affichage d'objets précis par de fines lignes. Idéal pour les mesures.

_Contour_ - Cette vue affiche des lignes d'arête plus épaisses, et est idéale pour l'exportation d'images.

_Esauisse_ - Cette vue est idéale pour les réunions avec des parties externes qui ont besoin d'un affichage simple du modèle.

Les objets dont la couleur n'est pas spécifiée dans l'IFC seront colorés avec des tons gris et une texture par défaut.

Les fenêtres conserveront leur couleur bleue transparente habituelle.

## 5. **Manette (joystick)**

En cliquant sur ce bouton, le joystick apparaît à l'écran.

Le joystick vous permet de parcourir le modèle en douceur en mode perspective et de tourner autour d'un point central en mode orthographique.

![](https://downloads.intercomcdn.com/i/o/areracg3/1299359835/906eb0bc392f2427e876cbad1a6a/image.png?expires=1781092800&signature=040ee5c7af2831143c2d35115debf27fddeab216b221dad2910950ab5e9b0a64&req=dSIuH8p7lIlcXPMW3nq%2BgdpGcMZNZHTD7kgednWqYejvnKCsyj4Ycpw2261J%0A%2BvG86n5mx0cKestwGihRKW8Z05I%3D%0A)

## 8. **Pied de page d'informations sur l'objet**

Par défaut, un pied de page s'affiche au bas du visualiseur lorsque vous sélectionnez un objet.

![](https://downloads.intercomcdn.com/i/o/areracg3/1299359841/63f24d295306c33845bc9b35bc42/image.png?expires=1781092800&signature=7f913368634518a62ca1a4c1b8d61242a81fe87bfef1d5d34fed72f634c0e53d&req=dSIuH8p7lIlbWPMW3nq%2BgSH3qEjHc%2BTwSu0Brl73IA%2FjprAaORpRw41y52%2F2%0ApQntqSl3AEmVuQRQe1vr3CLxWaM%3D%0A)

Ce pied de page peut contenir des informations sur les sujets et les éléments de bibliothèque liés à l'objet.

### 8.1 **Identification**

Nom de l'objet - Entité de l'objet - Type d'objet

Si plusieurs objets sont sélectionnés, la réponse sera indéfinie.

### 8.2 **Information des sujets attachés aux objets**

**Ouvert**

Nombre de sujets ouverts (en rouge)

Cliquez sur "Ouvert" pour voir tous les sujets ouverts auxquels cet objet est lié dans le panneau de gauche.

**Fermé**

Nombre de sujets fermés (en vert)

Cliquez sur "Fermé" pour voir tous les sujets fermés auxquels cet objet est lié dans le panneau de gauche.

### 8.3 **Bibliothèques**

Si vous avez lié l'objet sélectionné à des bibliothèques sur Catenda Hub, vous pourrez voir la bibliothèque avec le nombre de liens dans le pied de page d'information.

**SharePoint**

Si vous avez lié l'objet sélectionné à des documents SharePoint, vous verrez ici le nom de la bibliothèque SharePoint avec le nombre de documents liés.

**Documents**

Si vous avez lié l'objet sélectionné à des documents depuis la section "Documents", vous pourrez voir le nombre de documents liés ici.

**Classification**

Si vous avez lié l'objet sélectionné à des classifications, vous pourrez voir ici le nom de la classification et le nombre de classifications liées ici.

**Liens**

Si vous avez lié l'objet sélectionné à des liens URL depuis la bibliothèque de liens URL, vous pourrez voir ici la mention "Liens" avec le nombre de liens URL liés à l'objet.

**Dropbox**

Si vous avez lié l'objet sélectionné à des documents Dropbox, le nom de la bibliothèque Dropbox et le nombre de documents liés s'affichent ici.

### 8.4 **Masquer le pied de page d'information sur l'objet**

Cliquez sur "x" pour masquer le pied de page des informations sur l'objet dans la visionneuse 3D.

### 8.5 **Affichage du pied de page d'information sur l'objet**

Si vous avez choisi de fermer le pied de page d'information, vous pouvez l'ouvrir à nouveau à l'aide du bouton "i" situé sous la visionneuse 2D ou le bouton 2D.

![](https://downloads.intercomcdn.com/i/o/areracg3/1299359836/f052b8bfbe172ea1246b2187cfea/image.png?expires=1781092800&signature=86b6986dc6245ad83338b50803acee5b78116d25656ad4d627789ee5fcdf5af9&req=dSIuH8p7lIlcX%2FMW3nq%2BgapXkn2SyONrRLIuQJ%2FggejJNkIN6jD86bgHlcct%0ArtPGm960ttDgm%2BnkKqf%2FCpcU6oM%3D%0A)
