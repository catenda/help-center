Vous pouvez soit appuyer sur l'icône 3D pour ouvrir ce panneau, soit appuyer sur [Maj+2](https://support.catenda.com/en/articles/4670267-summary-of-most-important-shortcuts-and-basic-controls#:~:text=3D%20viewer-,Shift%20%2B%202,-Trees%20panel).

## 1. **Chargement de modèles**

Les modèles peuvent être chargés dans la visionneuse 3D de plusieurs façons.

Sur la page des modèles, le bouton 3D à côté du modèle à ouvrir peut être cliqué, ou plusieurs modèles peuvent être sélectionnés et le bouton 3D peut être cliqué en haut de la liste.

Dans la section des documents, vous pouvez ouvrir des aperçus de fichiers IFC et de documents nuages de points en cliquant sur 3D dans la vue de liste des documents ou dans le panneau d'informations de droite.

Vous pouvez également ouvrir la visionneuse 3D sans charger de modèles en cliquant sur 3D en haut à droite pour ouvrir le panneau de la visionneuse 3D.

### 1.1 **Sélecteur de révision**

Si vous n'avez pas encore chargé de modèles, vous pourrez charger et décharger des modèles en ouvrant la liste déroulante du sélecteur de révision.

Cliquez [ici](https://support.catenda.com/en/articles/4670279-revision-selector) pour en savoir plus sur le sélecteur de révision

### 1.2 **Liste déroulante de grille**

S'il y a des modèles dans vos projets qui contiennent des grilles, vous pourrez les charger dans la visionneuse avec cette liste déroulante.

Cliquez [ici](https://support.catenda.com/en/articles/4670327-showing-grids) pour en savoir plus sur les grilles dans la visionneuse 3D.

## 2. **Visibilité**

La liste déroulante de visibilité vous permet d'afficher, de masquer et de rendre les objets translucides.

![](https://downloads.intercomcdn.com/i/o/845934711/3df6af991ef5772735f8781c/image.png?expires=1779991200&signature=dca171ed06454fcd8d7e7ae056dfe9f6ff328c59c6edd069312f61b0bd7b607f&req=fCQiH8p6moBeFb4V1XW4gS5O5h2LhmYU%2BwpNf5%2BZRiY1HXXO%2BXzhIhz4bZKf%0AhtYo4yEqD5k2v%2BWTqTEa%2FGl8LA%3D%3D%0A)

Quand vous ouvrez la liste déroulante de visibilité, cela peut ressembler à ceci :

![](https://downloads.intercomcdn.com/i/o/845941119/e9b39bbe5d117a597aca78df/image.png?expires=1779991200&signature=d8e169d8dfae1ba577f9755f336083babd3c76936c4099e3cd6401d90004a8bd&req=fCQiH81%2FnIBWFb4V1XW4ga%2F7qvKmHR8Z%2FIHcK78bQ1jxtNshfpGDFHFdVN%2BK%0AM6FAYnFYNp95MdveC%2FB69GoGAA%3D%3D%0A)

Voici les actions que vous pouvez effectuer :

### 2.1 **Affichage des objets**

_Afficher tout_

Affiche tous les objet(s)

Ceci peut être utile après avoir lu un instantané ou un favori qui a de la géométrie cachée que vous aimeriez voir.

_Afficher la sélection_

Affiche le(s) objet(s) sélectionné(s)

Si vous masquez un objet sélectionné, celui-ci reste sélectionné mais caché jusqu'à ce que vous sélectionniez autre chose et le supprimiez de votre sélection.

Dans le panneau d'informations, vous pourrez sélectionner les objets masqués dans l'arborescence des objets et via les sélections.

Dans le panneau de contenu, vous pourrez sélectionner les objets des bibliothèques de propriétés et les objets liés à des problèmes ou à des documents.

### 2.2 **Translucidité des objets**

Il n'est pas possible de sélectionner des objets translucides en cliquant sur eux dans la visionneuse 3D.

_Rendre translucide_

Rend le(s) objet(s) sélectionné(s) translucide

Dans le panneau d'informations, vous pourrez sélectionner les objets masqués dans l'arborescence des objets et via les sélections.

Dans le panneau de contenu, vous pourrez sélectionner les objets des bibliothèques de propriétés et les objets liés à des problèmes ou à des documents.

Les objets translucides sélectionnés peuvent être retournés à la visibilité par défaut en affichant la sélection ou en affichant tout.

_Rendre les autres translucides_

Rend tout translucide sauf le(s) objet(s) sélectionné(s)

Afficher tout pour que les objets retournent à la visibilité par défaut.

### 2.3 **Masquage des objets**

Le(s) objet(s) masqué(s) seront toujours chargé(s) dans la mémoire de votre système.

_Masquer_

Masque un objet de la visionneuse

_Masquer les autres_

Masque tous les objets sauf l'objet sélectionné

Ceci est similaire à l'isolement des objets.

Les objets masqués seront toujours chargés en mémoire.

### 3.2 **Type d'appareil photo**

Voici à quoi peut ressembler le menu du type d'appareil photo :

![](https://downloads.intercomcdn.com/i/o/areracg3/1771082517/baca855e6cf24a2ece1f5cdb6355/image.png?expires=1779991200&signature=b50dd2ea23844eed61faa7667cb2e61e232fc5cc638b07724736cabf3cd5d5c9&req=dScgF8l2n4ReXvMW3nq%2Bgej5Tr4cvKVsBE5RMsJphVTXCMHJRhT9OHUadSBz%0AhrOa5Z0k77CsA6X%2BDGAR6CPdyCI%3D%0A)

_Perspective_ - Par défaut

Vous permet de voir le modèle comme un humain le ferait.

Ce type d'appareil photo est excellent pour comprendre ce que cela pourrait être d'être dans le modèle.

Comme l'appareil photo est au niveau de l'œil humain, vous pouvez passer à travers les murs.

Le champ de vision pour le type d'appareil photo en perspective peut être ajusté dans les paramètres de la visionneuse.

_Orthographique_

Vous permet de voir le modèle comme si vous étiez infiniment loin.

Ce type d'appareil photo est excellent pour la planification car il est plus facile de juger les distances.

Comme l'appareil photo est infiniment loin, il ne peut pas passer à travers les objets et vous devrez masquer ou rendre les objets translucides pour voir ce qui se trouve derrière eux.

Lignes fines d'affichage d'objets précis. Idéal pour les mesures. Lorsqu'aucune couleur n'est définie pour un objet dans l'IFC, l'objet reçoit une couleur basée sur l'entité configurée pour cet objet. Comme il n'y a pas de lumière solaire ou artificielle dans la visionneuse Catenda, les objets ne projettent pas d'ombre les uns sur les autres. Contour est une excellente vue pour exporter des images. **Remarque :** Assurez-vous que le matériel et les logiciels sur lesquels Catenda est ouvert sont configurés [comme recommandé](https://support.catenda.com/en/articles/6921941-hardware-recommendation) car cela peut affecter la netteté des lignes. _Esquisse_ Idéal pour les réunions avec des parties externes qui ont besoin d'un affichage simple du modèle. ## 8. **Pied de page d'informations sur les objets**

Par défaut, un pied de page s'affichera en bas de la visionneuse lorsque vous sélectionnez un objet.

![](https://downloads.intercomcdn.com/i/o/1019401439/4d388129fa2d03729dc31e76/image.png?expires=1779991200&signature=1e2d5d45ea86b0f1def0eb6a8971448b722b63f290a425a481abb2c141ee540d&req=dSAmH81%2BnIVcUPMW3nq%2BgexpaN7X8l172Nva71O74Sd38%2BeLDn9nwcCn9rt4%0AVVFDa%2BxynIdig3UVk3DaJMm28ts%3D%0A)

Ce pied de page peut contenir des informations sur les sujets et les éléments de bibliothèque liés à l'objet.

### 8.1 **Identification**

Nom de l'objet - Entité d'objet - Type d'objet

Si plusieurs objets sont sélectionnés, cela indiquera indéfini.

### 8.2 **Informations sur le sujet**

_Ouvert_

Nombre de problèmes ouverts (Rouge)

Cliquez sur "Ouvert" pour voir tous les problèmes ouverts où cet objet est lié dans le panneau de droite.

_Fermé_

Nombre de problèmes fermés (Vert)

Cliquez sur "Fermé" pour voir tous les problèmes fermés où cet objet est lié dans le panneau de droite.

### 8.3 **Bibliothèques**

Si vous avez lié votre objet sélectionné à des bibliothèques sur Catenda Hub, vous pourrez voir la bibliothèque avec le nombre de liens dans le pied de page d'informations.

_SharePoint_

Si vous avez lié votre objet sélectionné à des documents SharePoint, vous verrez le nom de la bibliothèque SharePoint avec le nombre de documents liés ici

_Documents_

Si vous avez lié votre objet sélectionné à des documents de la page des documents, vous pourrez les voir sous documents et avec le nombre de documents liés ici.

_Classification_

Si vous avez lié votre objet sélectionné à des classifications, vous pourrez voir le nom de la classification et le nombre de classifications liées ici.

_Liens_

Si vous avez lié votre objet sélectionné à des liens de la bibliothèque de liens, vous pourrez voir "Liens" et le nombre de liens liés ici.

_Dropbox_

Si vous avez lié votre objet sélectionné à des documents Dropbox, vous verrez le nom de la bibliothèque Dropbox avec le nombre de documents liés ici.

### 8.4 **Masquage du pied de page d'informations sur les objets**

Cliquez sur le x pour masquer le pied de page d'informations sur les objets de la visionneuse 3D.

### 8.5 **Affichage du pied de page d'informations sur les objets**

Si vous avez choisi de fermer le pied de page d'informations, vous pouvez le rouvrir avec le bouton i sous la visionneuse 2D ou le bouton 2D.
