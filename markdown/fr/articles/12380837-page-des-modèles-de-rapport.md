# Page des modèles de rapport

> **Remarque 1 :** La page des rapports est une fonctionnalité d'accès optionnel qui peut être demandée pour être activée pour les projets en cours. **Remarque 2 :** Les nouveaux projets créés à partir d'un projet modèle où cette fonctionnalité est activée n'ont pas cette fonctionnalité activée.

La page des modèles de rapport se trouve comme sous-page de la [page des rapports](https://support.catenda.com/en/articles/12303098-reports-page) dans le menu de navigation de gauche. Sur la page des modèles de rapport, les modèles de Documents et de topic disponibles pour le projet peuvent être gérés. _Accès requis :_ Administrateur

Voici à quoi peut ressembler la page des modèles de rapport :

![](https://raw.githubusercontent.com/catenda/help-center/main/images/5aw6t2k2/01-intro.png)

## 1. **Actions sur la page des modèles de rapport**

Les actions de la page des modèles de rapport peuvent être trouvées en cliquant sur le menu d'actions en haut à droite de la page des modèles de rapport. Voici à quoi peuvent ressembler les actions de la page des modèles de rapport :

![](https://raw.githubusercontent.com/catenda/help-center/main/images/5aw6t2k2/02-actions-on-the-report-templates-page.png)

### 1.1 **Nouveau modèle de topic**

Utilisez l'action nouveau modèle de topic pour créer un nouveau modèle de topic

### 1.2 **Nouveau modèle de Documents**

Utilisez l'action nouveau modèle de Documents pour créer un nouveau modèle de Documents

## 2. **Filtrer par nom**

Écrivez dans la barre de recherche pour filtrer le tableau des modèles de rapports par les noms des rapports.

![](https://raw.githubusercontent.com/catenda/help-center/main/images/5aw6t2k2/03-filter-by-name.png)

Recherche textuelle - `search=test`

## 3. **Tableau des modèles de rapport**

Dès qu'un modèle de rapport est créé, il s'affichera sous forme de ligne dans le tableau des modèles de rapport. Voici à quoi peut ressembler le tableau des modèles de rapport :

![](https://raw.githubusercontent.com/catenda/help-center/main/images/5aw6t2k2/04-report-templates-table.png)

Cliquez [ici](https://support.catenda.com/en/articles/11748020-tables-on-catenda) pour en savoir plus sur la façon de travailler avec des tableaux dans Catenda.

### 3.1 **Contenu de la ligne**

**Ligne de rapport** Bien qu'il soit possible de cliquer sur une ligne pour la sélectionner et la mettre en évidence en vert, il n'y a aucune action qui peut être effectuée en sélectionnant une ligne. La ligne sera sélectionnée lorsque le menu d'actions de la ligne dans la colonne de droite sera sélectionné.

### 3.2 **Colonnes**

En fonction de l'ordre des colonnes configuré, les premières colonnes sont affichées tandis que le tableau peut devoir être défilé latéralement pour afficher d'autres colonnes activées. L'ordre par défaut et le paramètre de visibilité des colonnes sur la page des modèles de rapport sont les suivants :

Nom - _Par défaut_ Le nom du modèle de rapport

_Créé par_ - _Par défaut_ Le créateur du modèle de rapport

Créé - _Par défaut_ L'heure et la date de création du modèle de topic

Type - Par défaut Le type de modèle de rapport, soit Documents, soit Topics.

Menu d'actions - Par défaut Cliquez sur le menu d'actions dans la colonne du menu d'actions pour ouvrir le menu d'actions de la ligne. Voici à quoi peut ressembler le menu d'actions de la ligne du modèle de rapport :

![](https://raw.githubusercontent.com/catenda/help-center/main/images/5aw6t2k2/05-columns.png)

Modifier Utilisez l'action modifier pour modifier le modèle de rapport sélectionné.

Télécharger Utilisez l'action télécharger pour télécharger le modèle de rapport sélectionné avec le format de fichier `.mrt` qui peut être réimporté dans tout autre projet.

Supprimer Utilisez l'action supprimer pour supprimer le modèle de rapport sélectionné.
