# Taille limite des fichiers IFC

> taille limitée IFC; quelle taille maximale par fichier;

La taille maximale d'un fichier IFC est de 700MB. Si vous souhaitez mettre en ligne un fichier supérieur à 700MB, vous pouvez essayer de l'archiver avant de le déposer sur la plateforme. Il est conseillé d'utiliser le format standard .zip afin d'éviter des sources potentielles d'erreur. Une fois ayant obtenu un fichier .zip, renommez l'extension de nom représentant le format : de **.zip** vers **.ifczip**. Certains outils BIM proposent directement ce type d'export.
