# Catenda Hub Navisworks plugin (description + lien téléchargement)

> Court descriptif du plugin Catenda Hub pour NavisWorks

> **Note:** notre fichier d'installation est disponible [ici](https://intercom.help/bimsync-arena/en/articles/8396532-catenda-plugins-and-integrations) (prochainement disponible également dans la boutique Autodesk).

L'extension Catenda Hub pour Autodesk® NavisWorks® est un outil dévéloppé pour les projets en collaboration dans Catenda Hub. Vos sujets sont synchronisés en temps réel via NavisWorks et Catenda Hub, ce qui vous permet de créer, visualiser, partager, échanger facilement d'une interface à l'autre. Le sujet est en format BCF ce qui signifie que les sujets peuvent être transférés et lus sur toute autre plateforme collaborative BIM disposant d'un lecteur BCF. L'échange fonctionne dans les 2 sens : NavisWorks vers Catenda Hub et viceversa.

Les fonctionnalités comprennent :

- Accéder à tous vos projets Catenda Hub
- Filtrer et gérer vos sujets dans vos tableaux de gestion
- Créer des sujets depuis NavisWorks
- Localiser les sujets dans NavisWorks
- Créer une vue 3D associé à chaque commentaire de sujet
- Créer des BCFs sur base de détection de clash NavisWorks
- Affecter les sujets avec un descriptif de tâche aux autres membres du projet
- Modifier le statut des sujets et autres paramètres de sujet

Catenda Hub est une plateforme SAaS qui vous accompagne durant tout le cycle de vie du bâtiment. La solution en facilite la gestion de la donnée en assurant sa traçabilité et traitement sur toutes les phases de projet. Notre outil de collaboration se plie à tous les standards buildingSMART (IFC, bSDD, BCF, COBie) et offre une série d'APIs vous permettant des développements complémentaires.
