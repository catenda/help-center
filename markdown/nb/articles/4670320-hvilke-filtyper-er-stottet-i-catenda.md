# Hvilke filtyper er støttet i Catenda?

For vår modell viser støtter vi bare IFC filer. Alle filtyper kan lastes opp i dokument delen. De følgende filtypene kan forhåndsvises i dokument delen:

> **Note:** **NB:** IFC og Point cloud forhåndsvisninger kan lastes inn i 3D viser.

- _PDF_ - pdf, fdf, xfdf
- _Bilder_ - bmp, wmf, emf, gif (static image only), hdp, jpg, jp2, png, tif, tiff
- _CAD_ - dwg, dxf, dgn, dwf
- _Office_ - doc, docx, xls, xlsx, xlsm, ppt, pptx, pub
- _Web_ - html, htm, mht, svg
- _Epost_ - msg
- _Open XML_ - txt, xps, oxps, xml, xod, xaml
- _Point Cloud -_ e57, las
- _IFC_

    _Akseptert_ (Offentlig og avviklede slipp av BuildingSMART)
    - IFC 4
        - IFC 4.3 - [IFC4.3 TC1 (zip)](https://github.com/buildingSMART/IFC4.3-html/releases/tag/sep-13-release)
        - IFC 4.0 - [IFC4 ADD2 TC1](https://standards.buildingsmart.org/IFC/RELEASE/IFC4/ADD2_TC1/HTML/)
    - IFC 2
        - IFC 2.3 - [IFC2x3](https://standards.buildingsmart.org/IFC/RELEASE/IFC2x3/FINAL/HTML/), [IFC2x3 TC1](https://standards.buildingsmart.org/IFC/RELEASE/IFC2x3/TC1/HTML/)
        - IFC 2.2 - [IFC2x2](https://standards.buildingsmart.org/IFC/RELEASE/IFC2x2/FINAL/HTML/), [IFC2x2 ADD1](https://standards.buildingsmart.org/IFC/RELEASE/IFC2x2/ADD1/HTML/)
        - IFC 2.1 - [IFC2x](https://standards.buildingsmart.org/IFC/RELEASE/IFC2x/FINAL/HTML/), [IFC2x ADD1](https://standards.buildingsmart.org/IFC/RELEASE/IFC2x/ADD1/HTML/)
        - IFC 2.0 - IFC2.0

    _Ikke akseptert_ (Dratt inn av BuildingSMART)
    - IFC 4
        - IFC 4.2 - [IFC4.2](https://standards.buildingsmart.org/IFC/DEV/IFC4_2/FINAL/HTML/)
        - IFC 4.1 - [IFC4.1](https://standards.buildingsmart.org/IFC/RELEASE/IFC4_1/FINAL/HTML/)

    Du kan finne hvilke versjoner av IFC er offentlig sluppet [her](https://technical.buildingsmart.org/standards/ifc/ifc-schema-specifications/)

    Offentlige BuildingSMART prøve IFC filer kan bli funnet [her](https://github.com/buildingSMART/Sample-Test-Files)
