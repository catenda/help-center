# Bestandstypen bekijken op Catenda Hub

Voor onze modelviewer ondersteunen we alleen IFC's. Alle bestandstypen kunnen naar onze sectie Documenten worden geüpload.

> **Opmerking:** IFC, Point cloud (e57, las) en GML-voorbeelden kunnen in de 3D-viewer worden geladen.

## 1. **Leverancier-neutrale formaten**

Bij Catenda willen we graag met open standaarden werken zodat iedereen samen kan werken, ongeacht welke software zij bezitten.

_Doorgaans kan iedereen deelnemen aan hun ontwikkeling vanwege hun inherent open aard ([Wikipedia](https://en.wikipedia.org/wiki/Open_standard#:~:text=Typically%2C%20anybody%20can%20participate%20in%20their%20development%20due%20to%20their%20inherently%20open%20nature.))_

Dit betekent dat niet slechts één organisatie controle heeft over wat wordt ontwikkeld.

### 2.1 **IFC-versies**

Afhankelijk van de status van de IFC-versie ondersteunen we de versie op verschillende manieren: _Official, Retired_ - Volledige ondersteuning. Alle officiële en vervallen IFC-versies worden ondersteund. _Withdrawn_ - Gedeeltelijke ondersteuning. Modellen worden verwerkt volgens de laatste officiële versie. _In ontwikkeling_ - Geen ondersteuning. Deze kunnen nog veranderen voordat ze worden gepubliceerd.

U kunt hier zien welke IFC-releases officieel zijn [hier](https://technical.buildingsmart.org/standards/ifc/ifc-schema-specifications/) Officiële BuildingSMART-voorbeeld-IFC-bestanden vindt u [hier](https://github.com/buildingSMART/Sample-Test-Files)
