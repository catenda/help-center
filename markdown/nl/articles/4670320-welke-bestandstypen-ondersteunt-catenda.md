# Welke bestandstypen ondersteunt Catenda?

Voor onze model viewer ondersteunen we alleen IFCs.

Voor ons documenten gedeelte ondersteunen we alle bestandstypen maar we ondersteunen geen documentenvoorbeelden van types ander dan:

- PDF
- Fotos
- DWG
- DXF
- Office Foormaten
- Point Clouds (Alleen .e57 en .las)
- IFC
