# 3D preview of documents

> Preview your model directly inside the Document section of Bimsync.

When you upload an IFC or [point cloud](https://support.catenda.com/en/articles/5606625-point-clouds-in-catenda-hub) to the documents section a preview will be generated that lets you see the 3D model in the [documents section](https://support.catenda.com/en/articles/8204673-documents-page). This preview will generate quickly as only the shapes of the objects are processed without any data connected to them. If there is a problem with your data you will therefore be able to see the preview even if there is an error with the properties of your objects. Because the objects have no ID you will not be able to link them to issues and documents. If you click on revisions you will also be able to load this preview in the [3D viewer](https://support.catenda.com/en/articles/8227211-3d-viewer). GUID's, properties and materials will therefore not be visible in the inspect section. It will also not be possible to use property libraries with these previews.

If you want to see the data connected to your objects please make a model in the models section. When you upload an ifc file to a model in the models section it processes fully which takes slightly longer.
