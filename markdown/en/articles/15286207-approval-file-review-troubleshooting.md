# Approval file review troubleshooting

In this article you will find information about the errors that can occur on the [file review in an approval](https://support.catenda.com/en/articles/12507002-file-review-in-an-approval-request).

## 1. **Withdrawn document**

Document revisions that are part of approval requests can can be withdrawn in the documents area. This is what it looks like when the page is loaded for a document revision that is withdrawn after it is added to an approval request.

![](https://raw.githubusercontent.com/catenda/help-center/main/images/pew9gsvy/01-withdrawn-document.png)

Document revisions that were added to the approval and were later withdrawn can be discarded from the approval request.

## 2. **Deleted document**

Documents that are part of approval requests can can be deleted in the documents area. This is what it looks like when the page is loaded for a document that is deleted after it is added to an approval request.

![](https://raw.githubusercontent.com/catenda/help-center/main/images/pew9gsvy/02-deleted-document.png)

After some seconds an error will be displayed which can look something like this:

![](https://raw.githubusercontent.com/catenda/help-center/main/images/pew9gsvy/03-deleted-document.png)

Documents that were added to an approval and later deleted can either be restored by an administrator or discarded from the approval request.
