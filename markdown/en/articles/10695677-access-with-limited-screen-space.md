# Access with limited screen space

With limited screen size, some functions can be accesssed in a user friendly way on Catenda Site. If having access to the entire functionality is more important to you than accessibility, this is always available through the browser. Browser pages can be added to the home screen/dashboard/dock to improve the experience of accessing Catenda with limited screen space.

## 1. **Catenda Site -** Accessible and offline

On devices with limited screen size, buttons need to be larger so you see what you are tapping with your finger. In this situation our mobile app, [Catenda Site](https://support.catenda.com/en/articles/6530821-download-your-mobile-app-catenda-site), is the best tool for you.

- Functionality is limited and kept simple so you can easily tap the buttons you need.
- Content can be downloaded and submitted without an internet connection

## 2. **Home screen / Dashboard / Dock -** Full functionality

The full functionality of Catenda Hub is always available through the browser. Catenda supports being [added to the home screen](https://support.catenda.com/en/articles/10741601-add-to-home-screen), dashboard or dock by installing it via a browser. Installing Catenda via a browser instead of creating a shortcut gives the following benefits:

- Catenda is more accessible as this is often the core way content is accessed on mobile devices.
- Screen space is saved as the browser bar towards the top will be hidden.
- Especially on tablets you would like to use the full functionality of Catenda while using the the screen space you have.
