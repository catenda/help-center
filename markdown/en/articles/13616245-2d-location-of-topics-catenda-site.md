# 2D location of topics - Catenda Site

Location can be applied to topics by placing a marker at an x, y, z coordinate in the [2D viewer](https://support.catenda.com/en/articles/4854537-2d-viewer). Colored dots mark the location of topics in the 2D viewer giving an immediate visual understanding of where topics are as well as their status and type. Using topic locations your project can leverage massively improved control over the placement of topics . This feature complements the functionality of our browser application, [Catenda Hub](https://support.catenda.com/en/articles/6530821-download-your-mobile-app-catenda-site). With makers, projects are empowered with seamless creation, monitoring and management of topics created in the field, back at the desk.

This is what markers can look like in the 2D viewer on Catenda Site:

![](https://raw.githubusercontent.com/catenda/help-center/main/images/6mjtp9hj/01-intro.png)

## 1. **Adding a marker to a topic**

### 1.1 **Add marker from topic**

Create a new topic or open a topic without a marker and tap on "Location".

![](https://raw.githubusercontent.com/catenda/help-center/main/images/6mjtp9hj/02-add-marker-from-topic.png)

The 2D/3D Menu will now open up and a new marker with an anchor is displayedin green:

![](https://raw.githubusercontent.com/catenda/help-center/main/images/6mjtp9hj/03-add-marker-from-topic.png)

Zoom in the 2D viewer by pinching with two fingers and move the marker to the right location and click on the checkmark to lock it in position.

### 1.2 **Create a new topic with a marker**

The green plus button in a topic board is not the only way to create a topic. Tap anywhere on the 2D viewer to create a topic at that location.

![](https://raw.githubusercontent.com/catenda/help-center/main/images/6mjtp9hj/04-create-a-new-topic-with-a-marker.png)

The topic that is created in this way will automatically have the marker applied at that location.

## 2. **Interacting with markers**

Tap on a marker on Catenda Site to display the topic number and title of the topic that belongs to that marker. This is what a tapped marker can look like in Catenda Site.

![](https://raw.githubusercontent.com/catenda/help-center/main/images/6mjtp9hj/05-interacting-with-markers.png)

### 2.1 **Marker configuration in Catenda Site**

Tap on the gear icon in the 2D viewer to open the Catends Site 2D viewer [settings](https://support.catenda.com/en/articles/7748370-2d-viewer-catenda-site#h_63e1728758)

![](https://raw.githubusercontent.com/catenda/help-center/main/images/6mjtp9hj/06-marker-configuration-in-catenda-site.png)

Here different 2D viewer options can be configured.

![](https://raw.githubusercontent.com/catenda/help-center/main/images/6mjtp9hj/07-marker-configuration-in-catenda-site.png)

**Marker visibility** Turn markers on or off

**Marker coloring mode** Change the type of color you wan the markers to have

**Space name format** Rooms can have short or long names in an IFC file

## 3. **Displaying markers**

To work with topics and locations open the 2D/3D viewer menu.

### 3.1 **Select a topic board**

Select a topic board to see the markers on the storey from that topic board.

![](https://raw.githubusercontent.com/catenda/help-center/main/images/6mjtp9hj/08-select-a-topic-board.png)

If a storey has been configured in the storey configurator, the first storey of the first building will have opened by default as soon as the 2D/3D viewer menu was opened.

### 3.2 **Filter the topic board**

Narrow down the amount of markers on the screen by filtering the topic board.

![](https://raw.githubusercontent.com/catenda/help-center/main/images/6mjtp9hj/09-filter-the-topic-board.png)

> **Note:** While any amount of markers can be viewed with the default filter there is a limit of 250 markers being visible when a filter is applied on Catenda Site. In the browser it is possible to view more markers with filters appplied.

### 3.3 **Expand the 2D view**

Drag on the divider between the 2D and 3D viewers to expand it.

![](https://raw.githubusercontent.com/catenda/help-center/main/images/6mjtp9hj/10-expand-the-2d-view.png)

Select a storey from an IFC model or from a configured building by tapping on the storey menu on the top left.

![](https://raw.githubusercontent.com/catenda/help-center/main/images/6mjtp9hj/11-expand-the-2d-view.png)

## 4. **Finding topics from markers**

### 4.1 **2D view to topic**

Tap on a marker in the 2D view to see the topic number and title of the topic it belongs to. Tap on the arrow of the selected marker to display the selected topic in its topic board on the topics page.

### 4.2 **Topic to 2D view**

Find a topic that says "Has location" in the Location field and tap on it.

![](https://raw.githubusercontent.com/catenda/help-center/main/images/6mjtp9hj/12-topic-to-2d-view.png)

In the menu that opens up tap "View location on 2D":

![](https://raw.githubusercontent.com/catenda/help-center/main/images/6mjtp9hj/13-topic-to-2d-view.png)

When a topic is opened in a topic board the color of that marker is highlighted in green with a checkmark in the 2D/3D menu.

![](https://raw.githubusercontent.com/catenda/help-center/main/images/6mjtp9hj/14-topic-to-2d-view.png)

## 5. **Moving markers**

Tap on the location field in a topic and tap on replace location in the menu that opens up. Drag the marker with the anchor to its new location and confirm.

## 6. **Removing markers**

Tap on the location field in a topic and tap on replace location in the menu that opens up. Tap on remove location.
