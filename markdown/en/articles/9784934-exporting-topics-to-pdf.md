# Exporting topics to PDF

Topics can be exported to PDF by clicking on the PDF option in the [Export topics dialogue](https://support.catenda.com/en/articles/4670289-exchange-topics#h_aa0b8d7789). The PDF export will let you export a report of your topics. This export option is useful to have an easily readable version of topics and can be used for archiving purposes or to report to authorities.

This page includes information about the following topics:

## 1. **Settings that affect all pages of the PDF**

These settings affect all pages of the PDF

### 1.1 **Logo**

The logo on the top left of every page of the PDF will be the Catenda logo by default. Clicking on the logo or on hub.catenda.com on the top right will take you to the projects page. It is possible to upload a custom logo on the [project settings page](https://support.catenda.com/en/articles/4670273-project-settings-page) to get your own logo to show up here.

### 1.2 **Language**

The PDF export will be in the language that you have configured in your [account preferences](https://support.catenda.com/en/articles/6880968-account-page#h_4aa257b72f).

### 1.3 **Topic order**

If you have sorted the topics in your topics board in any way, the order the topics will appear in the report will reflect the order in which you have sorted your topic board.

## 2. **Front page**

The downloaded PDF will contain a front page. Here you will see the profile picture of the project along with the following export details: The front page can look something like this:

![](https://raw.githubusercontent.com/catenda/help-center/main/images/nx64p9qh/01-front-page.png)

### 2.1 **Project name**

The name of the project

### 2.2 **Exported**

The amount of topics and from which topic board they were exported. Clicking the topic board name will take you to that topic board.

### 2.3 **Time of export**

The UTC timezone time when the pdf export was made.

### 2.4 **Topics overview**

Below the topics that are part of this export will be listed. Clicking the name of the topic will take you to that topic on Catenda Hub.

## 3. **Topic pages**

A page for every topic with all the detail of each topic will be added after the front page. A typical topic page can look something like the following:

![](https://raw.githubusercontent.com/catenda/help-center/main/images/nx64p9qh/02-topic-pages.png)

### 3.1 **Title**

The title of the topic followed by the number of the topic on Catenda.

### 3.2 **Included metadata**

The following headers are inluded:

**Type** The type of the topic

**Status** The status of the topic

**Assigned to** The assignee of the topic

**Created by** The creator of the topic

**Created** The time the topic was first created

**Closed at** If the topic is closed, the latest time of closing of the topic is displayed here.

**Due date** The due

**Custom fields from the topic board** Each custom field and its value if there is any is included

**Labels** Labels are listed with name and color.

### 3.3 **Excluded topic headers**

**Milestone** Milestones are not included in the PDF export.

**Marker** Click [here](https://support.catenda.com/en/articles/4854523-2d-location-of-topics) to read more about the exporting of 2D location to PDF.

### 3.4 **Topic relations**

Links to topic relations on Catenda are inluded in the PDF export. The related elements are not exported automatically so consider separately exporting this information.

**Topics** If topics are linked to the exported topic each topic is listed with a link to that topic on Catenda.

> **Note:** These topics are not necessarily included in the report as they might be from different topic board with different access settings.

**Documents** If there are any documents that are linked to the topic you will see them listed here with links to each document in Catenda.

### 3.5 **Description**

The description of the topic

### 3.6 **Comments**

If any comments have been placed on the topic you will see them here. First you will see the username of the comment author, and the date they posted the comment followed by any images that are attached and the text of the comment.

## 4. **Reports page**

With the reports page it is possible to make custom reports about not only topics but also documents. The reports page is an on-demand feature that can be requested to be enabled for ongoing projects. New projects that are created based on a template project where this feature is enabled do not have this feature enabled. These reports can then be exported not only to Excel and PDF, but also many more file formats. These reports can also export topics from multiple topic boards at a time. Click [here](https://support.catenda.com/en/articles/12303098-reports-page) to read more about the reports page
