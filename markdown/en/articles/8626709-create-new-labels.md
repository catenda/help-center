# Create new labels

Labels are useful for many areas in Catenda HUB. They can be used for both documents and tasks. Assigned labels help you to better organize and filter your tasks and documents.

In the following article you will find instructions on how to create new labels, how to create label groups and how to import a group of labels from an Excel spreadsheet.

## 1. Create new label

To do this, click on Labels in the Settings area. You now have the option of creating a new label

![](https://raw.githubusercontent.com/catenda/help-center/main/images/y6vse6fp/01-create-new-label.png)

A view window opens in which you can assign a label name, assign a color to the label or assign a group to the label.

![](https://raw.githubusercontent.com/catenda/help-center/main/images/y6vse6fp/02-create-new-label.png)

## 2. Create label group

It is a good idea to group / organize labels in groups.

If you would like to group different labels, please proceed as follows. To do this, click on "New group" in the "Groups" area

![](https://raw.githubusercontent.com/catenda/help-center/main/images/y6vse6fp/03-create-label-group.png)

Enter a name here too. The group is created.

![](https://raw.githubusercontent.com/catenda/help-center/main/images/y6vse6fp/04-create-label-group.png)

## 3. Import from Excel

If you want to create several labels at the same time from an Excel file, proceed as follows. We are in the labels area and create a new label:

![](https://raw.githubusercontent.com/catenda/help-center/main/images/y6vse6fp/05-import-from-excel.png)

The following window opens:

You now do not have to select a color or group, but can insert this file directly from an Excel table using "Copy" and "Paste".

![](https://raw.githubusercontent.com/catenda/help-center/main/images/y6vse6fp/06-import-from-excel.png)

Here is an example table. The color codes can be found on the relevant websites. Simply enter "color code" in Google.

![](https://raw.githubusercontent.com/catenda/help-center/main/images/y6vse6fp/07-import-from-excel.png)

Now we can insert this table into the Name column. To do this, simply select, copy and paste everything from Label1 to the last color code.

![](https://raw.githubusercontent.com/catenda/help-center/main/images/y6vse6fp/08-import-from-excel.png)

Now simply insert the selected cells in the name area. The names, groups and colors are assigned automatically. Make sure you use the correct spelling and a complete color code. Use "Submit" to create the new labels.

![](https://raw.githubusercontent.com/catenda/help-center/main/images/y6vse6fp/09-import-from-excel.png)

Your view should now look like this:

![](https://raw.githubusercontent.com/catenda/help-center/main/images/y6vse6fp/10-import-from-excel.png)

You have successfully created your labels.
