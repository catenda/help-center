# File review in an approval request

The file review for an approval request can be opened by clicking on review or view on the approval request page. This is what the review page of a file in an approval request can look like for projects that enabled shared revisions after 2 October 2025:

![](https://raw.githubusercontent.com/catenda/help-center/main/images/zr7fa0bv/01-intro.png)

Projects where shared revisions were enabled before 2 October 2025 will see the legacy file review page instead.

## 1. Opening the file review page

### 1.1 All documents

Click on the review button for a team on the overview tab of an approval page to open the file review for all documents in an approval on behalf of that team.

### 1.2 Selected documents

Click on the review button in the right menu of the documents tab of an approval page after having selected some documents in the documents tab to open the file review for the selected documents on behalf of that team.

### 1.3 View only

Click on the view button in the right menu of the documents tab of an approval after having selected some documents in the documents tab to view the file review for the selected documents.

## 2. Breadcumbs

### 2.1 Documents

Click on documents to go to the documents page

### 2.2 Approvals

Click on Approvals to go to the approvals page

### 2.3 Approval number and title

The number and the title of the approval Click on the name of the approval request to go the the approval requets page of the approval request that this revision is part of.

## 3. File review navigation

### 3.1 Back to button

Go back to the approval request page that files under review are part of.

### 3.2 Progress bar

Shows the revisions that can be navigated to

> **Note:** It is only possible to navigate to as many revisions as were selected on the approval requets page. Clicking review on the overview tab of the approval request will let you navigate to all revisions while selecting documents in the documents tab of the approval request and clicking review in the right menu will only let you navigate to the selected documents.

This total number is therefore not always indicative of the total files to review in the approval request.

_Previous and next buttons_ - Go to the previous or next revision in the approval request review.

## 4. Right menu

The right menu can be opened by clicking on the info icon towards the right of the approval request review page. The Info icon can look something like this:

![](https://raw.githubusercontent.com/catenda/help-center/main/images/zr7fa0bv/02-right-menu.png)

Click here to read more about the right menu on the approval request review page.

## 5. Fullscreen

The review fullscreen button can be found towards the top right of the approval request review page:

![](https://raw.githubusercontent.com/catenda/help-center/main/images/zr7fa0bv/03-fullscreen.png)

## 6. Document preview

When reviewing the file review on behalf of a team all [document annotation tools](https://support.catenda.com/en/articles/11384618-document-preview-tools) will be available. The document preview on the approval request review page works the same as the [document preview](https://support.catenda.com/en/articles/5001165-document-preview-and-annotations) when opening a document from the documents table.

### 6.1 Read only

When viewing the file review in read only mode only the [view tool](https://support.catenda.com/en/articles/11384618-document-preview-tools#h_27c7badabb) will be available in the toolbar of the document preview.
