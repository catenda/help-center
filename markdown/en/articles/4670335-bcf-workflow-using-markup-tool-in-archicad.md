# BCF workflow using markup tool in Archicad

**BCF from Archicad to Bimsync** In Archicad select the markup tool to create a BCF. Here you can highlight the related element and attach different views to the BCF etc. Export the selected markup entries as a bcfzip file.

![](https://raw.githubusercontent.com/catenda/help-center/main/images/5fgdptf6/01-intro.png)

Go to issues and exchange issues to upload the BCF. Import BCF and select the saved BCF from Archicad.

![](https://raw.githubusercontent.com/catenda/help-center/main/images/5fgdptf6/02-intro.png)

![](https://raw.githubusercontent.com/catenda/help-center/main/images/5fgdptf6/03-intro.png)

The BCF is now loaded into the issues section.

![](https://raw.githubusercontent.com/catenda/help-center/main/images/5fgdptf6/04-intro.png)

You can then go and change the different settings inside the issue by clicking on it.

![](https://raw.githubusercontent.com/catenda/help-center/main/images/5fgdptf6/05-intro.png)

**BCF from Bimsync to Archicad** Filter out the issue you want to get.

![](https://raw.githubusercontent.com/catenda/help-center/main/images/5fgdptf6/06-intro.png)

Export the filtered issue.

![](https://raw.githubusercontent.com/catenda/help-center/main/images/5fgdptf6/07-intro.png)

Download the selected BFC. The file will come out as a bcfzip file.

![](https://raw.githubusercontent.com/catenda/help-center/main/images/5fgdptf6/08-intro.png)

Go to the markup tool in Archicad and import the BCF via the import markup entries. The selected BCF comes up in markup tool bar.

![](https://raw.githubusercontent.com/catenda/help-center/main/images/5fgdptf6/09-intro.png)
