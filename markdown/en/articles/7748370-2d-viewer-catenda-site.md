# 2D viewer - Catenda Site

You will be able to find the 2D viewer of Catenda Site on the [2D / 3D page](https://support.catenda.com/en/articles/7734138-2d-3d-catenda-site)

![](https://raw.githubusercontent.com/catenda/help-center/main/images/t22xrf09/01-intro.png)

## 1. **2D viewer**

In the 2D viewer of Catenda you will be able to see the storeys that are available in your project.

### 1.1 **Floor selector**

![](https://raw.githubusercontent.com/catenda/help-center/main/images/t22xrf09/02-floor-selector.png)

Select the floor that is to be displayed in the 2D viewer.

![](https://raw.githubusercontent.com/catenda/help-center/main/images/t22xrf09/03-floor-selector.png)

Both storeys from the different IFC models in the project and storeys from buildings that are configured in the Catenda Hub storey configurator are displayed in the list. Storeys that are not part of a building are displayed below the bottom floor of the last building.

### 1.2 **2D viewer appearance**

With [markers enabled](https://support.catenda.com/en/articles/4854537-2d-viewer#h_381a9d4098) any topics that have a specified location on the story of a an IFC or a configured building are displayed. Tap a marker to select it. This is what a selected marker can look like on Catenda Site.

![](https://raw.githubusercontent.com/catenda/help-center/main/images/t22xrf09/04-2d-viewer-appearance.png)

When selected the topic number and title are displayed. Tap on the arrow to open the topic in the topics menu. Click [here](https://support.catenda.com/en/articles/4854523-2d-location-of-issues) to read more about the 2D location of topics.

### 1.3 **Toolbar**

![](https://raw.githubusercontent.com/catenda/help-center/main/images/t22xrf09/05-toolbar.png)

a. Zoom to extents of the storey - Get an overview b. [3D query](#h_8a2c752641) - Load a section of the 2D view in 3D c. 2D viewer [settings](#h_63e1728758)

**3D query** If you get an error when loading 3D models it is often because your mobile device cannot handle all the data from one or more models.

![](https://raw.githubusercontent.com/catenda/help-center/main/images/t22xrf09/06-toolbar.png)

With this option parts of the model can be loaded so they can be viewed even with less resources available on the mobile device.

**2D viewer settings** Here different 2D viewer options can be configured

![](https://raw.githubusercontent.com/catenda/help-center/main/images/t22xrf09/07-toolbar.png)

- _Marker visibility -_ Turn markers on or off
- _Marker coloring mode -_ Change the type of color you wan the markers to have
- _Space name format -_ Rooms can have short or long names in an IFC file
