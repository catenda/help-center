# Interaction Catenda HUB & Solibri

If you want to use Catenda Hub in combination with Solibri Modelchecker we have a suitable workflow for you.

Workflow CAD - Catenda HUB - Solibri Modelchecker

![](https://raw.githubusercontent.com/catenda/help-center/main/images/13sb7ht8/01-intro.png)

Once the current IFC files have been uploaded to Catenda HUB, please proceed as follows:

1. Open Solibri
1. Open the menu item "Connections". For this you should have installed the current Solibri version.

![](https://raw.githubusercontent.com/catenda/help-center/main/images/13sb7ht8/02-intro.png)

3\. Connect Solibri to the CDE server (Catenda).

![](https://raw.githubusercontent.com/catenda/help-center/main/images/13sb7ht8/03-intro.png)

![](https://raw.githubusercontent.com/catenda/help-center/main/images/13sb7ht8/04-intro.png)

4\. Once Catenda is selected, the connection to the server is created in the background. Now click on "Open".

![](https://raw.githubusercontent.com/catenda/help-center/main/images/13sb7ht8/05-intro.png)

5\. Catenda HUB opens automatically, log in if necessary. Now select the corresponding project.

![](https://raw.githubusercontent.com/catenda/help-center/main/images/13sb7ht8/06-intro.png)

6\. Now select either the models (1.) or the documents (2.) you want to migrate to Solibri.

![](https://raw.githubusercontent.com/catenda/help-center/main/images/13sb7ht8/07-intro.png)

7\. Then select "Download model

![](https://raw.githubusercontent.com/catenda/help-center/main/images/13sb7ht8/08-intro.png)

8\. The downloaded models are automatically opened in Solibri. Only the discipline should be checked briefly. .

![](https://raw.githubusercontent.com/catenda/help-center/main/images/13sb7ht8/09-intro.png)

9\. now start your workflow in Solibri.

10\. at the end of your workflow, you should save the Solibri model .smc on Catenda HUB so that all project participants can work with the current file. To do this, select "Upload model" in the Connections menu.

![](https://raw.githubusercontent.com/catenda/help-center/main/images/13sb7ht8/10-intro.png)

11\. Now select the appropriate location on Catenda HUB and click "Upload".

![](https://raw.githubusercontent.com/catenda/help-center/main/images/13sb7ht8/11-intro.png)

If you now want to connect the issue management via the BCF Live Connector, please have a look at this article how to do it.
