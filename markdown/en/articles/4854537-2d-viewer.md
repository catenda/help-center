# 2D Viewer

> Describes the different tools in the 2D toolbar

To open the 2D viewer, click on the 2D button on the right in the [3D panel](https://support.catenda.com/en/articles/8227211-3d-viewer).

![](images/01-intro.png)

## 1. **Model storeys**

In the 2D viewer a slice through 3D objects from a model are displayed. When a new ifc file is uploaded as a revision either to a model or to a document that is linked to a model, 2D slices through the objects in the ifc file at each of the storeys are generated. While it is possible to configure these slices to be shown at a different height with the storey configurator in the case of for example an in-between storey it is not possible to alter the content of each slice after it is generated. This is what a the 2D viewer can look like when a storey is displayed:

![](images/02-model-storeys.png)

### 1.1 **Default storey**

If no 2D view has previously been selected and there are models on the models page of the project the default model storey is displayed. The default storey is the storey from the latest revision of one of the models that has the most objects of interest. Objects of interest include objects like walls, doors and windows.

**Configuring the default storey** If a building has been configured in [the storey configurator](https://support.catenda.com/en/articles/6921756-storey-configurator-page), the first storey of the first building that is created is displayed instead of the default storey that is picked based on objects of interests. Double-check project configurations if the pre-configured storey does not display as expected, and revisit settings in the storey configurator if necessary.

### 1.2 **Available storeys**

Load 2D views of different storeys in models into the 2D viewer in different ways.

**2D button** Click on the 2D button for a model. 2D buttons for models can be found in multiple places:

- [Revision selector](https://support.catenda.com/en/articles/4670279-revision-selector) towards the top right of the 3D viewer
- [Models table](https://support.catenda.com/en/articles/4670286-models-page#h_bbc42c56bb) on the [models page](https://support.catenda.com/en/articles/4670286-models-page)
- Viewer column in the [documents table](https://support.catenda.com/en/articles/8204673-documents-page#h_54e8dfcac2) on the [documents page](https://support.catenda.com/en/articles/8204673-documents-page)

**Playing a snapshot** Play [2D or 3D snapshot](https://support.catenda.com/en/articles/9184555-2d-and-3d-viewpoints) from a topic or play a [snapshot](https://support.catenda.com/en/articles/10345863-snapshots) from [a bookmark](https://support.catenda.com/en/articles/8471481-bookmark) to load in the 2D models that were enabled at the time the snapshot was created.

**View marker on 2D** A marker, of a topic with a marker that is placed on a storey, can be clicked to select play on 2D. Click this button to make the storeys of models, that were enabled when the marker was made created, available in the 2D viewer. _Access required:_ Read access to the document connected to each of the models.

**Configured storeys** If building are configured in the story configurator, the storeys that are configured in each of these buildings will always be available. _Access required:_ Read access to the document connected to each of the models.

### 1.3 **Combining storeys**

When 2D views of multiple models have been enabled at the same time they are displayed together in the 2D viewer. Select a story from one of the models to view the storeys from all enabled models that are generated for this height together in the 2D viewer. For example, when 2D views of both the architectural and structural model are enabled and the first floor of the architectural model is selected, if the structural model has a storey that is at the same height of the first floor of the architectural model then both the walls from the architectural model and the beams from the structural model are displayed together in the 2D viewer.

**Load in 2D views of multiple models at once** To load in 2D views of multiple models at once, select the models on the models page and click on the 2D button towards the top of the models page.

**Configured storeys** Select a storey from a building that is configured in [the storey configurator](https://support.catenda.com/en/articles/6921756-storey-configurator-page) to automatically load the 2D views of all models that are configured for that storey into the 2D viewer.

## 2. **2D viewer appearance**

2D viewer elements such as:

- Objects
- Underlay layers such as drawings or map
- Markers
- Room labels

Can be configured to be displayed to users needs. Click [here](https://support.catenda.com/en/articles/13615984-2d-viewer-appearance#h_73338e8c67) to read more about the appearance of 2D viewer elements.

## 3. **2D viewer interactions**

The following buttons and areas can be clicked on to be able to perform different actions.

- Top bar
- Canvas
- Navigation tools
- Selection tools and Settings
- Bottom bar

Click [here](https://support.catenda.com/en/articles/12699525-2d-viewer-interactions) to read more about these different actions.

## 4. **Exporting an image**

A good way to export an image from the 2D viewer is by making a [2D snapshot](https://support.catenda.com/en/articles/8227211-3d-viewer#h_a1594324c8) in a topic. Vector line images in the form of `.svg` files can be extracted from the webpage element by inspecting the page with the browsers developer tools in the elements tab.
