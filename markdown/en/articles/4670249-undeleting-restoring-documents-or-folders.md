# Undeleting / Restoring documents or folders

NB: This feature is only available to Administrators of the project

This assumes you are in the "Documents" view. You get there by clicking the "Documents" button on the main menu bar (on the left)

1\. Click on the filter button

2\. Select the "Deleted" option in the Filter list (this might be a different word depending on your language settings.)

3\. Select using the checkboxes which files/folder you want to undelete

4\. Click the "Restore" button
