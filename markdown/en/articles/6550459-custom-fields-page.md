# Custom fields page

> Customize topic boards and naming conventions

You will be able to find the custom fields page as a sub page to the [project settings page](https://support.catenda.com/en/articles/4670273-project-settings-page). Custom fields let you customize your topics and naming conventions by adding text fields, numbers and lists of values.

![](https://raw.githubusercontent.com/catenda/help-center/main/images/7c3te8t3/01-intro.png)

## 1. **New item actions**

The new item actions can be found on the top right of the page.

![](https://raw.githubusercontent.com/catenda/help-center/main/images/7c3te8t3/02-new-item-actions.png)

### 1.1 **New custom field**

Administrators will be able to see a green 'New custom fields' button here. See [here](https://support.catenda.com/en/articles/8445575-creating-a-custom-field) how to create a new custom field

### 1.2 **Archived custom fields**

All users will be able to see the custom fields that have been archived and are not in use anymore. Here they will be able to restore archived custom fields.

## 2. **Search or filter**

This is what the search or filter menu on the custom fields page can look like:

![](https://raw.githubusercontent.com/catenda/help-center/main/images/7c3te8t3/03-search-or-filter.png)

### 2.1 **Types**

Text - `type=text` Integer - `type=integer` Decimal number - `type=double` Dropdown - `type=enumeration` Date - `type=date`

### 2.2 **Text search**

Search - `search=<Text search>`

**Content that can be searched on** Custom field title

**Capitalization** The text search is not sensitive to upper or lowercase characters.

**Character amounts** One or more characters Content that matches the search phrase in any part of the content is included int he results.

## 3. **Custom field table**

As soon as a custom field is created it shows up as a row in the custom fields table. This is what the custom fields table can look like:

![](https://raw.githubusercontent.com/catenda/help-center/main/images/7c3te8t3/04-custom-field-table.png)

Click [here](https://support.catenda.com/en/articles/11748020-tables-on-catenda) to read more about how to work with tables in Catenda.

### 3.1 **Row content**

Once a custom field is made it is not possible to change its type. Click on any of the fields to be taken to the [custom field page](https://support.catenda.com/en/articles/8445588-custom-field-page) of that custom field.

### 3.2 **Columns**

Name - The name of the custom field Type - The type of the custom field Description - The description of the custom field

## 4. **Enabling a custom field**

After making a custom field the field can be added to [topics in a topic board](https://support.catenda.com/en/articles/6563368-how-to-use-custom-fields-at-my-issues) or as values in a [naming convention](https://support.catenda.com/en/articles/7869240-how-to-use-custom-fields-in-naming-convention).
