# Adding users without an email invitation

Organization owners will be able to add organization members to projects without inviting them.

## 1. **Add member(s) to organization**

To do this first add the user to your organization as a member. If your [domain has been registered](https://support.catenda.com/en/articles/8224886-organization-options#h_8369f64208) with us you will be able to add users with existing Catenda accounts from your domain as members to your organization. If your domain has [not been registered](https://support.catenda.com/en/articles/8224886-organization-options#h_8369f64208) and you do not see the green "add from domain" button please contact support through the black chat button next to your profile picture on the top right or via [support@catenda.com](mailto:support@catenda.com) where we will be happy to help you.

![](https://raw.githubusercontent.com/catenda/help-center/main/images/8g939d5c/01-add-member-s-to-organization.png)

## 2. **Add organization member(s) to a project**

If the user you want to add is part of your organization you will be able to add the member to a project on the organizations project page by selecting the project and clicking the 'Add new members' action in the action menu above the projects list. From here you can also invite user by email address without being part of the project.

![](https://raw.githubusercontent.com/catenda/help-center/main/images/8g939d5c/02-add-organization-member-s-to-a-project.png)

## 3. **Add organization members to project(s)**

If you want to add an organization member to multiple projects you can also do this by selecting the user on the users page and clicking the add to projects action in the the action menu above the users list.

![](https://raw.githubusercontent.com/catenda/help-center/main/images/8g939d5c/03-add-organization-members-to-project-s.png)
