# Using topic templates with approvals

> Apply an approval topic template to an approval workflow template

Approval topic templates can both be configured on the [topic template settings page](https://support.catenda.com/en/articles/14076010-topic-template-settings-page) and by clicking on the topic template dropdown in the final approval menu of the new approval workflow page and selecting create a new topic template. _Access required:_ Project administrator Topic template settings page

Create a new topic template on the new workflow page.

> **Important:** ℹ️ For a topic to get generated as the result of the closing of an approval, the approval **has to contain a markup**. After closing the configured approval, for each markup in the approval a separate topic is generated.

1. When a new approval workflow is created. 
You [create a new approval workflow](https://support.catenda.com/en/articles/12313251-creating-a-new-workflow) template, navigate at the bottom of the page until you reach the "Final approval" section. 
Here, you can link an approval topic template for each review state (and associate document final status).
​

1. Ensure you have filled all the previous necessary fields from the approval workflow template and click the "Submit" button.
1. Go through an approval workflow. 
After all reviews have been made and once the approval workflow is closed, the associated approval topics will be automatically generated and auto-populated with all the fields that were set in the associated topic template settings, accessible from the [topic template settings page](https://support.catenda.com/en/articles/14076010-topic-template-settings-page).
For a topic to get generated as the result of the closing of an approval, the approval has to contain a markup. 
If an approval topic template is configured for the workflow that the approval follows topics get generated when the approval is closed.
For each markup in the approval a separate topic is generated.
