# Receiving notifications that fit your use

Getting notifications about actions in Catenda Hub is important to be able to follow up on activities in the projects that you are part of. Notifications can be found in several places around Catenda Hub.

## 1. **Account notifications**

All notifications from all projects are visible om the [account notifications page](https://support.catenda.com/en/articles/7439223-account-notifications-page). Here you will be able to [filter through all the notifications](https://support.catenda.com/en/articles/8304417-filtering-and-functionality-on-notifications-pages) you have received.

### 1.1 **Account notification page**

On the [account notifications page](https://support.catenda.com/en/articles/7439223-account-notifications-page) you will see all the notifications that you have received from all your projects.

### 1.2 **Account notification settings page**

You can configure which notifications you would like to receive for all your projects together on the [account notification settings page](https://support.catenda.com/en/articles/8272435-account-notification-settings).

## 2. **Project notifications**

Within a project there are several places you can see notifications.

### 2.1 **Dashboard**

Here you see the [5 most recent notifications](https://support.catenda.com/en/articles/8212646-dashboard-page#h_77effb806b) that have been generated by users in the project

### 2.2 **Project notifications page**

Here you see all the [notifications generated by users](https://support.catenda.com/en/articles/4670295-project-notifications-page) in the project. On this page you will be able to f[ilter though all the notifications ](https://support.catenda.com/en/articles/8304417-filtering-and-functionality-on-notifications-pages)that have been generated by users in the project.

### 2.3 **Project specific notification settings**

Here you can [configure which notifications](https://support.catenda.com/en/articles/4670262-project-specific-notification-settings) you would like to receive for the project you are in. This setting will overwrite the account notification setting.

### 2.4 **Settings - Members - Member**

If you go to a members member page you will see the [topics generated by that user](https://support.catenda.com/en/articles/8228836-member-page#h_6b76063480) in the project.

### 2.5 **Settings - Members - Team**

If you go to a team page on the members page you will see the [topics that have to do with that team](https://support.catenda.com/en/articles/7891755-team-page#h_f91b3824f4) in the project.
