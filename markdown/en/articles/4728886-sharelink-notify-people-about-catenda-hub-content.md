# ShareLink - Notify people about Catenda Hub content

> Notify manually users when it's needed and important.

Find the sharelink button towards the [top right of the page](https://support.catenda.com/en/articles/4670260-landing-page#h_7e60f1b700) next to your profile picture.

![](https://raw.githubusercontent.com/catenda/help-center/main/images/e1zqszw1/01-intro.png)

If you are on the main page before entering a project the sharelink button will be grayed out. It is only possible to share pages from within a project.

## 1. **Notifying users**

Ever wanted to notify a project member about a document, some topics or other content in Catenda Hub, without leaving Catenda Hub? Or maybe users have switched off their Catenda Hub notifications? This feature solves these problems.

ShareLink sends an email from Catenda Hub to the project members you choose with a link to the content you want to share.

> **Note:** Users will always get a notification about a link that is shared with them this way. Even if they have their notifications turned off.

### 1.1 **To use ShareLink:**

1. First select up to 100 items of content you want to share.

    For example some documents or topics.

    A sharelink recipient will then only see these items.
If the page is shared with a sharelink recipient they will see the whole page, including filters, based on their access in the project.

1. Click the the ShareLink button.
​

![](https://raw.githubusercontent.com/catenda/help-center/main/images/e1zqszw1/02-to-use-sharelink.png)

## 2. **ShareLink window**

![](https://raw.githubusercontent.com/catenda/help-center/main/images/e1zqszw1/03-sharelink-window-br.png)

### 2.1 **Recipients**

Select the project members you want to send the message to. You can select teams by typing @ You can also easily send a message to all users in a project this way.

> **Note:** If you click and drag from outside the recipients box you can select all recipients. This is very useful if you want to copy the emails of all users in the project. The recipient field can be copied and pasted in the recipient To: field of any email application.

### 2.2 **Subject**

Change the title of the message to suit the reason for sharing the content

### 2.3 **Message**

Write you own message in the message body.

### 2.4 **Shared content**

The URL you will be sending is available here to check.

### 2.5 **Screenshot**

If you want to make sure the user knows what to expect when clicking on the link you can add a screenshot of the link you are sharing. In this case the following image will be attached to the sharelink email notification:

![Screenshot of shared content](https://raw.githubusercontent.com/catenda/help-center/main/images/e1zqszw1/04-screenshot.png)

## 3. **Send the message**

The message is sent from [noreply@catenda.com](mailto:noreply@catenda.com) to the recipients you chose in step 1. You get a CC message to your own email.

The mail message can look like this if you added a screenshot:

![](https://raw.githubusercontent.com/catenda/help-center/main/images/e1zqszw1/05-send-the-message.png)

## 4. **Accessing the content**

1. Click on this link, Catenda Hub will open a page with the shared items.

    or

1. Copy this URL to your browser.

When the link is followed only the selected items are shown:

![](https://raw.githubusercontent.com/catenda/help-center/main/images/e1zqszw1/06-accessing-the-content.png)

Selected items work for topics as well as documents.
