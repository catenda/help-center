# Catenda Revit plugin changelog - 1.0.15

New features in 1.0.15

- This plugin now has its .MSI own installer that is separate from the Autodesk plugins page.
- When you type @ in a comment or description you now get a list of members you can choose from so you can select which one you want to mention.
- Images for 3D snapshots and other images in topics comments load a lot faster now.
- Support for Revit 2026

> **Note:** Uninstalling the previous version and installing the new version is required to use this plugin.
