# Solibri Topics Integration

> **Important:** **Solibri version 9.13:** **Important:** The BCF Connector plugin is since version 9.13 a separate tab called BCF Live connector in Solibri. Much of the same information still applies in this tab. **Solibri version 9.9:** Since update 9.9 of Solibri it is now possible to synchronize your topics with Catenda.

This document explains how to setup the BCF connector with your Catenda project. This allows topics to be communicated across the two applications seamlessly, avoiding the manual export and import BCF steps.

1\) Purchase and download the Solibri Model Checker from the Solibri Solution Center. If you already have the tool remember to upgrade it.

2\) You can get/upgrade the BCF Connector from the Solibri Solution Center - [https://solution.Solibri.com/application/#!ProductsView](https://solution.solibri.com/application/#!ProductsView)

Go to Solibri Extensions and select BCF connector

![](https://raw.githubusercontent.com/catenda/help-center/main/images/qv33595h/01-intro.png)

> **Note:** To enable the BCF Connector your Solibri Admin needs to enable it in the user management screen in Solibri Solution Center.

3\) Open SMC, and go to the settings screen. From there select the BCF connector

![](https://raw.githubusercontent.com/catenda/help-center/main/images/qv33595h/02-intro.png)

4\) Add server

1. First press the Add Server button and you will be sent to the second page.
1. In this step you can choose which server you want to connect to, in this case the Catenda one.
1. Here you can paste the URL from your Catenda project. Where to find this URL is explained in the next picture.
1. OK to finish the connection.

![](https://raw.githubusercontent.com/catenda/help-center/main/images/qv33595h/03-intro.png)

6\) On the topics page in your Catenda project, you can find the URL for your project. It is located in the Exchange topics.

![](https://raw.githubusercontent.com/catenda/help-center/main/images/qv33595h/04-intro.png)

7\) Then go to the communications screen in Solibri, and select presentations. From there you can create a new presentation. Right click to the context menu to be able to synchronize with Catenda.

![](https://raw.githubusercontent.com/catenda/help-center/main/images/qv33595h/05-intro.png)

8\) From here you can also create a new presentation and import from Catenda to Solibri.

![](https://raw.githubusercontent.com/catenda/help-center/main/images/qv33595h/06-intro.png)

To optimize your workflow with the BCF server [See here](https://support.bimsync.com/hc/en-us/articles/360011987619)
