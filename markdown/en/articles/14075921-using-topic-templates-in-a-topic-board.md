# Using topic templates in a topic board

> Apply a general topic template upon topic creation

General topic templates can be configured on the [topic template settings page](https://support.catenda.com/en/articles/14076010-topic-template-settings-page). _Access required:_ Project administrator

![](https://raw.githubusercontent.com/catenda/help-center/main/images/be4v7emn/01-intro.png)

1. First, create a new topic from the regular topic section.
​_Access required:_ Write access to a topic board.
1. Then, select the right topic board within which you want to create your topic.
​_Access required:_ Write access to the topic board.
1. Depending on the topic board you have selected, if general topic templates are configured by an administrator they can be available and suggested in the Template dropdown.
​

    <div class="intercom-container intercom-align-center"><img height="549" src="https://raw.githubusercontent.com/catenda/help-center/main/images/be4v7emn/inline-b8fa9d6ab8dc.png" style="height: auto;" width="500"/></div>

1. Select the right template you want to use. 
As soon as you select the right template, the new topic will auto-populate with all the fields that were set in the associated topic template settings, accessible from the [topic template settings page](https://support.catenda.com/en/articles/14076010-topic-template-settings-page).
​

    <div class="intercom-container intercom-align-center"><img height="731" src="https://raw.githubusercontent.com/catenda/help-center/main/images/be4v7emn/inline-23a75554e46a.png" style="height: auto;" width="500"/></div>

1. You can finally save your topic by clicking the "Submit" button.
​
