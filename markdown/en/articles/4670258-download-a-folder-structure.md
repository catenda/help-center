# Download a folder structure

You can download folders and its content on the documents page.

Go to documents and use the check boxes on the left hand side to choose which folders to download.

When a folder is checked, a new window on the right hand side of the page will appear, press download.

A zipped file of the folder structure and files will be downloaded.
