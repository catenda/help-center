# Revit changelog - 1.0.8

New features in 1.0.8

- Revit 2024 availability
- Teams filtering
- Custom export settings are once again available in the plugin
- Issue side panel
    - Inside an issue it is now possible to open the side panel by clicking on related items.
    - See documents that are linked to the issue
    - Select objects that are linked to the issue
    - See which issues are related to this issue
