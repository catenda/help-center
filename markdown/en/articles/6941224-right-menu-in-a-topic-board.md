# Right menu in a topic board

You will be able to find the info icon towards the right of the search bar in a topic board. Clicking this icon will open an information menu that contains information about the topic(s) you have selected. The Info icon can look something like this:

![](https://raw.githubusercontent.com/catenda/help-center/main/images/azwgqsc0/01-intro.png)

When the side menu is expanded it looks like this when:

## 1. **No topics selected:**

![](https://raw.githubusercontent.com/catenda/help-center/main/images/azwgqsc0/02-no-topics-selected.png)

### 1.1 **Access control for topic board:**

If all selected topics are from the same topic board you can see who has what access without having to go into the topic board settings! Administrators will be able to configure access here as well.

### 1.2 **One topic selected:**

![](https://raw.githubusercontent.com/catenda/help-center/main/images/azwgqsc0/03-one-topic-selected.png)

1. Members with full access to both topic boards will be able to move the selected topic to another board here
1. The topic board of the selected topic
1. The title of the selected topic
1. The fields of the selected topic
1. The description of the selected topic
1. The comments in the selected topic
1. Make a new comment in the selected topic
1. Submit a comment to the selected topic

### 1.3 **Multiple topics** **selected:**

![](https://raw.githubusercontent.com/catenda/help-center/main/images/azwgqsc0/04-multiple-topics-selected.png)

1. Members with full access to both boards multiple selected topics from one topic board to another board here
1. The name of the topic board of the topics. Multiple boards selected if the topics are from multiple boards.
1. Set the status from multiple topics at once.
1. Unset the value of any of the following fields in all selected topics by clicking on the x next to the field.
1. Set the type of multiple topics at once.
1. Assign a user to multiple topics at once.
1. Set requester of multiple topics at once.
1. Set a due date/milestone of multiple topics at once.
1. Add a label to multiple topics at once.
1. Remove a label from multiple topics at once.
1. Click Save to commit your changes.

> **Note:** If topics from multiple boards are selected it will not be possible to update Status and type. The selectable users in assigned to and requester will also limited to the users that have access to the topic  boards of all the selected topics.
