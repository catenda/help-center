# Inviting new members to the project

[YouTube video](https://www.youtube.com/embed/yd6R0FUEj-4?rel=0)

If you are an administrator in a project you can add new members. As a member of a project, by default you are not able to add new members, but an administrator can give access to invite members.

A new member can be added by going to:

“Members” (left hand menu)

1. Invite member
1. type in their email address(es)
1. select their role in the project, and which team (optional)
1. +Add
1. Optional message=>Send invite(s)
