# Topic templates page - Legacy

The topic templates page can be found by clicking the topic templates action in the action menu to the top right right of the [topic settings page](https://support.catenda.com/en/articles/12367293-approval-settings-page#h_320e990d50). _Access required_: Administrator

This is what the topics templates page can look like:

![](https://raw.githubusercontent.com/catenda/help-center/main/images/n5z47ntk/01-intro.png)

## 1. **Actions on the topics templates page**

The topics templates page actions can be found by clicking on the action menu to right of the green plus button on the top right of topics templates page. This is what the topics templates page actions can look like when looking at topic templates:

![](https://raw.githubusercontent.com/catenda/help-center/main/images/n5z47ntk/02-actions-on-the-topics-templates-page.png)

This is what the topics templates page actions can look like when looking at archived topic templates:

![](https://raw.githubusercontent.com/catenda/help-center/main/images/n5z47ntk/03-actions-on-the-topics-templates-page.png)

### 1.1 **New topic template**

Click on the new topic template action to create a new topic template. Click [here](https://support.catenda.com/en/articles/12372007-creating-a-new-topic-template) to read more about creating new topic templates.

### 1.2 **Topic templates / Archived topic tempates**

Click on the topic templates and archived topic templates actions switch bteween viewing topic templates and viewing archived topic templates.

## 2. **Topic template card**

After a topic template is created a card for the topic template appears on this page. Click on a topic template card to open the [topic template page](https://support.catenda.com/en/articles/12371549-topic-template-page) of that topic template. This is what a topic template card can look like:

![](https://raw.githubusercontent.com/catenda/help-center/main/images/n5z47ntk/04-topic-template-card.png)

### 2.1 **Topic template title**

The title of the topic template

### 2.2 **Topic template created by**

The creator of the topic template

### 2.3 **Title**

The title the topic will have By default it is set to receive the name of the approval request and the name of the document whose validation outcome resulted in the creation of this topic.

### 2.4 **Topic board**

**Board** The topic board the topic is created in.

**Status** The status the topic will have

**Type** The type the topic will have

### 2.5 **Due date**

The due date that is applied to the topic.

### 2.6 **Milestone**

The milestone that is applied to the topic.

### 2.7 **Assigned to**

The member that is assigned to the topic. By default this is set to be the publisher of the approval.

### 2.8 **Requested by**

The member that is requesting the topic. By default this is set to be the uploader of the revision whose validatoin outcome resulted in the creation of this topic.

### 2.9 **Labels**

The labels that are applied to the topic upon creation.

### 2.10 **Description**

The description that the generated topic will have. By default the generated topic will include a link to the approval request as well as a link to the document revision in the approval request.

## 3. **New validation workflow**

The topic templates page is not available for projects starting out with the new validation workflow.
