# System software recommendation

Operating systems are what are between the application where catenda is accessed and the hardware that makes everything run.

## 1. **Dedicated Graphics Card vs Integrated GPU**

If a device both has a dedicated GPU and an integrated GPU that is part of the CPU make sure the dedicated GPU of the system is in use and not the integrated GPU that is part of the CPU which is often less powerful. Mobile devices often switch to the integrated GPU that is part of the CPU when no power adapter is plugged in. Follow the steps described in [this article](https://superuser.com/questions/1439471/how-can-i-force-my-dedicated-gpu-to-handle-all-applications-or-disable-my-integr#:~:text=use%20these%20steps%3A-,Open%20Settings.,-Click%20on%20System) to make sure the dedicated GPU is used on Windows 10 and Windows 11 even when the power adapter is not plugged in. Note that this method can drain the battery of your device faster.

## 2. **Monitor configuration**

It is recommended to configure the default resolution and refresh rate settings for each of the monitors that are available to the operating system.
