# 3D measurements update!

> Improvements to the measurement tools in Catenda Hub

There are some improvements that has been made to the measurement tools in Catenda Hub.

## 1. **Measurement units**

Recently we added support for imperial measurement, in addition to the existing metric ones. The default can be defined in project settings.

![](https://raw.githubusercontent.com/catenda/help-center/main/images/8kcjrrhv/01-measurement-units.png)

## 2. **Snap to objects**

Both with the measurement tape and the laser measure tool can you now snap to objects.

- Curves and lines created by a clipping plane can be snapped to.
- Centres of circular objects like pipes can also be snapped to.
- Points in point clouds can be snapped to.

Notice the “bullseye” in the measurement pointer.

- A red bullseye indicates snapping to a line or a curve.
- A green bullseye indicates snapping to a corner.
- A blue bullseye indications that it is still waiting to snap to something.

![](https://raw.githubusercontent.com/catenda/help-center/main/images/8kcjrrhv/02-snap-to-objects.png)

## 3. **Point measure**

A new tool in the measure toolbox is the point measure. With this tool you can get the x, y and z position of the part you are pointing at in 3D. It is supported both for IFC and point clouds. This tool also supports snapping.

![](https://raw.githubusercontent.com/catenda/help-center/main/images/8kcjrrhv/03-point-measure.png)

## 4. **Measurement toolbox**

All of these tools can be accessed by clicking on the arrow icon in the top right corner of the 3D viewer.

![](https://raw.githubusercontent.com/catenda/help-center/main/images/8kcjrrhv/04-measurement-toolbox.png)
