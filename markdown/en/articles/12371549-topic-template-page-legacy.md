# Topic template page - Legacy

The topic template page can be found by clicking on a topic template card on the [topic templates page](https://support.catenda.com/en/articles/12367730-topic-templates-page). _Access required_: Administrator

This is what the topics template page can look like:

![](https://raw.githubusercontent.com/catenda/help-center/main/images/3hd924wy/01-intro.png)

## 1. **Actions on the topic template page**

The topics templates page actions can be found by clicking on the action menu to right of the green plus button on the top right of topics templates page. This is what the topics templates page actions can look like:

![](https://raw.githubusercontent.com/catenda/help-center/main/images/3hd924wy/02-actions-on-the-topic-template-page.png)

### 1.1 **New topic template**

Click [here](https://support.catenda.com/en/articles/12372007-creating-a-new-topic-template) to read more about creating new topic templates.

## 2. **Topic template**

This is what a topic template can look like:

![](https://raw.githubusercontent.com/catenda/help-center/main/images/3hd924wy/03-topic-template.png)

### 2.1 **Edit**

Click on edit to edit the topic template. Click here to read more about how to edit topic templates.

### 2.2 **Archive**

Click on Archive to archive the topic template.

### 2.3 **Restore**

Click on restore to restore an archived topic template.

## 3. **New validation workflow**

The topic template page is not available for projects starting out with the new validation workflow.
