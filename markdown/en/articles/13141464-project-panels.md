# Project panels

On the project level the main panel buttons are always displayed towards the top right, no matter the project page.

![](https://raw.githubusercontent.com/catenda/help-center/main/images/39gcn7x2/01-intro.png)

This is what the panel icons towards the top right can look like:

![](https://raw.githubusercontent.com/catenda/help-center/main/images/39gcn7x2/02-intro.png)

On devices with limited screen space a button called view is displayed instead. Click on view to show switch between the different panels.

![](https://raw.githubusercontent.com/catenda/help-center/main/images/39gcn7x2/03-intro.png)

## 1. **Opening panels**

Panels can be opened in different ways.

### 1.1 **Automatically opened**

By default, when a project is opened from the projects page the content page is opened and the dashboard page is displayed. There are several situations where the model viewer/3D panel automaticaly gets opened.

### 1.2 **Manually opening and closing panels**

**Devices with larger screens** When a panel is open it is highlighted in green. Select a panel that is not active yet to open it side-by-side with the other active panels. Selct an active panel to disable it. There has to at least be one panel active at once. As an exampe, to switch from the content panel to the 3D panel, enable the 3D panel along with the content panel first and then disable the content panel to only have the 3D panel active.

To easily switch between the different panels keybinds can be used. Shift+1 for the main content panel Shift+2 for the model viewer panel Shift+3 for the model tree panel Shift+4 for the model information panel

**Devices with limited screen space** The open panel has a gray background. Select another panel to switch to that panel. With limited screen space it is only possible to have one panel open at a time.

## 2. **Main content panel**

The main content panel is the first panel and has the most information. Some users strictly work in the content panel. This panel is also referred to as the content panel.

![](https://raw.githubusercontent.com/catenda/help-center/main/images/39gcn7x2/04-main-content-panel.png)

The content panel can contain the following main pages:

Dashboard Models Topics Documents Libraries Reports Notifications Settings

Click [here](https://support.catenda.com/en/articles/8212646-dashboard-page) to read more about the content panel.

## 3. **Model viewer panel**

The Model viewer panel is the second panel. This panel is also referred to as the 3D viewer panel.

![](https://raw.githubusercontent.com/catenda/help-center/main/images/39gcn7x2/05-model-viewer-panel.png)

The model viewer panel can contain the following main pages:

3D viewer tools 2D viewer

Click [here](https://support.catenda.com/en/articles/8227211-3d-viewer) to read more about the 3D panel.

## 4. **Model tree panel**

The model tree panel is the third panel. This panel is also referd to as the trees panel.

![](https://raw.githubusercontent.com/catenda/help-center/main/images/39gcn7x2/06-model-tree-panel.png)

The trees panel can contain the following main pages:

Containment tree Component tree Type tree Layer tree System tree

Click [here](https://support.catenda.com/en/articles/4670290-trees-panel) to read more about the trees panel.

The model info panel is the fourth and last panel. This panel is also referred to as the inspect panel.

![](https://raw.githubusercontent.com/catenda/help-center/main/images/39gcn7x2/07-model-info-panel.png)

The In panel can contain the following main pages:

Inspect QTO (Quantity Take-Off) 4D Selection Measure
