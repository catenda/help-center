# Libraries page

The Libraries page can be found as the fifth page in in the left navigation menu in a project. This page is part of the content panel. Here you will be able to find libraries that contain sets of items.

![](https://raw.githubusercontent.com/catenda/help-center/main/images/7wg8652x/01-intro.png)

## 1. **New item actions**

By clicking on the the new item button you will be able to add a new library.

![](https://raw.githubusercontent.com/catenda/help-center/main/images/7wg8652x/02-new-item-actions.png)

### 1.1 **Document libraries**

[Documents](https://support.catenda.com/en/articles/8204673-documents-page) - The document library in the documents section [Links library](https://support.catenda.com/en/articles/4670254-links-library) - Connect links to objects and issues _Dropbox_ - Link files from Dropbox to objects in the BIM

![](https://raw.githubusercontent.com/catenda/help-center/main/images/7wg8652x/03-document-libraries.png)

After giving your library a name, Catenda will redirect you to Dropbox to get permission to access your Dropbox account. You will then be able to browse your Dropbox and select which folder to share with the project. If you wish, you can create several Dropbox libraries, even using the same Dropbox account.

> **Note:** Catenda will never get access to your Dropbox password as OAuth 2.0 is used for authentication.

_[Sharepoint](https://support.catenda.com/en/articles/5744548-sharepoint-library)_ - add a Sharepoint library to the project. This will allow members to view the files and connect them to others.

> **Note:** This is different from our [SharePoint Application](https://support.catenda.com/en/articles/8396496-catenda-sharepoint-application) which lets members post documents from SharePoint to Catenda

_coBuilder Pro_ - Documents - Tag objects with doucments from coBuilder Pro This library is only available in legacy projects.

![](https://raw.githubusercontent.com/catenda/help-center/main/images/7wg8652x/04-document-libraries.png)

Enter your coBuilderPRO username and password to connect.

### 1.2 **Generic libraries**

_[Classification](https://support.catenda.com/en/articles/4670255-classification-library)_ - Tag objects with custom classifications from a CSV file _coBuilder Pro_ - tag objects with product info from coBuilder Pro _Lookup_ - Include an external website as a library _[Property values](https://support.catenda.com/en/articles/4670252-property-value-library)_ - Create entries based on property values

### 1.3 **Suggested libraries**

If any of the models in your project contain classification, an embedded library for that classification will be suggested here. Click on the suggested library to activate it. A suggested library will look up the codes for that classification online and try to match it with codes that have been specified in your file. These public classification libraries are not editable and will only find objects tagged with a classification code if this code is correct according to the classification.

## 2. **Libraries list**

Here you will see a list of all you configured libraries. Libraries will either appear as document library or as generic library

### 2.1 **Starring**

Add a star to one of your libraries to have it show up as a sub page to the libraries page in the right information menu.

### 2.2 **Document libraries**

Here you will always see the documents and links library towards the top. The documents library will take you to the documents page. The links library will take you to the links library. Any Dropbox, Sharepoint libraries that have been configured will also sow up here.

### 2.3 **Generic libraries**

Here the buildingSMART Data Dictionary library can be seen towards the top. Below that you will see any classification, embedded classification, coBuilder Pro, lookup and property value libraries in the order they have been added.
