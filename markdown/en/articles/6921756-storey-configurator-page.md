# Storey Configurator Page

> How to have a PDF as an Overlay in your 2D View

By configuring storeys in a project prepared views in the [2D viewer](https://support.catenda.com/en/articles/4854537-2d-viewer) can be created for project members. The Storey configurator page can be found as a sub page to the [models page](https://support.catenda.com/en/articles/4670286-models-page). The names Storey and Building used on this come from the IFC standard. While the term Storey and Building are often more used social infrastructure this tool is as useful, if not more useful, for physical or digital infrastructure projects. _Access required:_ Project administrator

![](https://raw.githubusercontent.com/catenda/help-center/main/images/hn8gfwtp/01-intro.png)

In a project members can compose storeys by turning on, and off, 2D views with generated geometry from different models that are available in the project. Configure a storey to make a configured storey with generated 2D views from a set of models available to members in the project. Underlay a PDF document in a storey to show linework that may otherwise not be visible in the generated geomtry from the model.

This is what the storey configurator page can look like in a new project:

## 1. **Actions**

The action menu on the storey configurator page can be found by openen the three dots to the right of the green plus button on the top right.

![Add new building](https://raw.githubusercontent.com/catenda/help-center/main/images/hn8gfwtp/04-actions.png)

### 1.1 **Add new building**

Click [here](https://support.catenda.com/en/articles/12291366-storey-configurator-actions) to read more about adding a new building.

### 1.2 **PDF**

In order to complete the steps in this article you will need a PDF Plan for every floor where you want a pdf as underlay.

> **Note:** Every floor has to be in an separate PDF. One floor can have multiple PDFs

## 2. **Setup**

Set your View so that you can see your 2D Model and all of your Models.

![](https://raw.githubusercontent.com/catenda/help-center/main/images/hn8gfwtp/05-setup.png)

## 3. **Storey Configuration**

This is where you align your PDF Plan with your 2D View, this has to be done once for every floor.

By default you will have one Building in your Overview. This also contains the Levels of the imported IFC.

![](https://raw.githubusercontent.com/catenda/help-center/main/images/hn8gfwtp/06-storey-configuration.png)

Setting a Name for the Building is important as it will be shown later, so choose some thing that is self-explanatory.

![](https://raw.githubusercontent.com/catenda/help-center/main/images/hn8gfwtp/07-storey-configuration.png)

Every IFC Level is represented in a Storey. Here you can set the Name and Elevation.

> **Note:** Changing the Elevation will change the representation of your 2D View.

Within the Storey you can set the Overlay, click on "Show extra Options"

![](https://raw.githubusercontent.com/catenda/help-center/main/images/hn8gfwtp/08-storey-configuration.png)

Here you see the "Drawings attached to the storey" function. Click on Add.

![](https://raw.githubusercontent.com/catenda/help-center/main/images/hn8gfwtp/09-storey-configuration.png)

A dialogue will open up and ask you for the Uploaded PDF.

![](https://raw.githubusercontent.com/catenda/help-center/main/images/hn8gfwtp/10-storey-configuration.png)

When you have selected the wanted PDF click on Add this document. This will bring you to the alignment dialogue.

## 4. **Alignment Dialogue**

This is where you will define the overlay and position it.

![](https://raw.githubusercontent.com/catenda/help-center/main/images/hn8gfwtp/11-alignment-dialogue.png)

Your building should be selected now All that is left is selecting the storey you want to align your drawing with.

![](https://raw.githubusercontent.com/catenda/help-center/main/images/hn8gfwtp/12-alignment-dialogue.png)

Now you should see your selected PDF and the selected Storey on your screen.

![](https://raw.githubusercontent.com/catenda/help-center/main/images/hn8gfwtp/13-alignment-dialogue.png)

With the Marker A and B you define the position of the overlay. The Points you select have to match. Best to choose interception of two grids for example.

Use the marker via drag and drop to place them. You can also zoom by scrolling in the drawing.

After placing them click on  "Next: Confirm result"

![](https://raw.githubusercontent.com/catenda/help-center/main/images/hn8gfwtp/14-alignment-dialogue.png)

### 4.1 **Approve Dialogue**

After confirming the position of the markers the approve dialogue opens. Here you can have a look at the overlay to see if that works for you.

![](https://raw.githubusercontent.com/catenda/help-center/main/images/hn8gfwtp/15-approve-dialogue.png)

Here a change can always be made by clicking on Previous: Place model and drawing points. Once you are happy with your alignment you confirm by clicking on Approve.

This will bring you back to the Storey Configurator and now you can set up the next Storey. Placement of drawings can be modified in this dialogue as well.

![](https://raw.githubusercontent.com/catenda/help-center/main/images/hn8gfwtp/16-approve-dialogue.png)

**Important:** Here you have to click on save configuration.

![](https://raw.githubusercontent.com/catenda/help-center/main/images/hn8gfwtp/17-approve-dialogue.png)

## 5. **How to use it**

Now that everything is set up you can use it in your daily work. After your building configuration is done processing you can refresh the page. If you are in your Model view and open the 2D View there is an Option to select the Building.

![](https://raw.githubusercontent.com/catenda/help-center/main/images/hn8gfwtp/18-how-to-use-it.png)

![](https://raw.githubusercontent.com/catenda/help-center/main/images/hn8gfwtp/19-how-to-use-it.png)

Select the just renamed Building

![](https://raw.githubusercontent.com/catenda/help-center/main/images/hn8gfwtp/20-how-to-use-it.png)

And your PDF overlay will be visible.

![](https://raw.githubusercontent.com/catenda/help-center/main/images/hn8gfwtp/21-how-to-use-it.png)

The storey configurator will always show the latest revision of the aligned drawing(s)
