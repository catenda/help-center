# 2D location of topics

> Describing of the 2D location markers on topics

Location can be applied to topics by placing a marker at an x, y, z coordinate in the [2D viewer](https://support.catenda.com/en/articles/4854537-2d-viewer). Colored dots mark the location of topics in the 2D viewer giving an immediate visual understanding of where topics are as well as their status and type. Using topic locations your project can leverage massively improved control over the placement of topics . This feature complements the functionality of our mobile application, [Catenda Site](https://support.catenda.com/en/articles/6530821-download-your-mobile-app-catenda-site). With makers, projects are empowered with seamless creation, monitoring and management of topics created in the field, back at the desk.

## 1. **Adding a marker to a topic**

### 1.1 **Add marker from topic**

Create a new topic or open a topic without a marker and click on "Add marker". If the 3D panel was not yet open it will now open and a information message is displayed towards the top of the screen.

![](https://raw.githubusercontent.com/catenda/help-center/main/images/qm9hb6bs/02-add-marker-from-topic.png)

Notice that the cursor turns into a crosshairs as soon as you hover over the 2D viewer indicating that a marker will be placed at that location when it is clicked.

### 1.2 **Add marker from 2D viewer**

**Add marker** Navigate to a topic or select a single topic in the topic board. Right-click in the 2D viewer and choose "Add marker" in the context dialogue that opens up to add a marker to the selected or open topic at that location.

![](https://raw.githubusercontent.com/catenda/help-center/main/images/qm9hb6bs/03-add-marker-from-2d-viewer.png)

**Create a new topic with marker** First navigate to the topic board you would like to create a topic with a marker in. Next, right-click in the 2D viewer and choose "Create a new topic with marker" in the context dialogue that opens up to initiate the [new topic creation process](https://support.catenda.com/en/articles/4670268-creating-a-new-topic) with the marker already applied.

![](https://raw.githubusercontent.com/catenda/help-center/main/images/qm9hb6bs/04-add-marker-from-2d-viewer.png)

## 2. **Interacting with markers in the browser**

Hover over a [marker in the 2D viewer](https://support.catenda.com/en/articles/4854537-2d-viewer#h_d904cebf2e) in the browser to display the topic number and title of the topic that belongs to that marker. This is what a hovered marker can look like in the browser.

![](https://raw.githubusercontent.com/catenda/help-center/main/images/qm9hb6bs/05-interacting-with-markers-in-the-browser.png)

Click on the marker to open the topic in the corresponding topic board in the content panel.

## 3. **Displaying markers**

To work with topics and locations, open the content and 3D panels side by side:

![](https://raw.githubusercontent.com/catenda/help-center/main/images/qm9hb6bs/07-displaying-markers.png)

### 3.1 **Choose the model for you 2D background**

![](https://raw.githubusercontent.com/catenda/help-center/main/images/qm9hb6bs/08-choose-the-model-for-you-2d-background.png)

If a storey has been configured in the storey configurator, the first storey of the first building will have opened by default as soon as the 3D panel was opened.

### 3.2 **Expand the 2D view**

![](https://raw.githubusercontent.com/catenda/help-center/main/images/qm9hb6bs/09-expand-the-2d-view-br.png)

Drag on the top left corner of the 2D view or doubleclick on the title of the 2D view to expand it. Select the building and storey on which to add or view topic locations on the bottom left.

### 3.3 **Finding topics from markers**

**2D view to topic** Hover over a marker in the 2D view to see the topic number and title of the topic it belongs to. When a marker in the 2D view is clicked the content panel changes to display the selected topic in its topic board on the [topics page](https://support.catenda.com/en/articles/4670271-topics-page).

**Topic to 2D view** To find the location of a topic in, open the topic in its topic board and click on the "Has marker" text in the topic header.

![](https://raw.githubusercontent.com/catenda/help-center/main/images/qm9hb6bs/10-finding-topics-from-markers.png)

In the menu that opens up select "View marker on 2D":

![](https://raw.githubusercontent.com/catenda/help-center/main/images/qm9hb6bs/11-finding-topics-from-markers.png)

When a single topic is selected in the the topic board or when a topic is opened, the color of that marker is highlighted in green:

![](https://raw.githubusercontent.com/catenda/help-center/main/images/qm9hb6bs/12-finding-topics-from-markers.png)

### 3.4 **Moving markers**

In the 2D viewer, navigate to the building and storey the maker is to be placed, In a topic with a marker, select the Choose new location option from the Has marker menu. Click on the new location of the marker in the 2D viewer to reposition the marker to its new location.

### 3.5 **Removing markers**

Select the Delete marker option in the topic.

## 4. **Exporting location to PDF**

Marker information is not automatically included in the PDF export. To ensure marker information is included in the PDF export, click on "Has marker" in the topic and click on view on 2D. The storey of the building where the marker exists will be displayed in the 2D viewer. With this storey open, go back to the topic and click on the plus button next to the comment field. In the add snapshot menu a snapshot of the 2D viewer can be added to your topic. An image of the current view will be added as a comment to the topic this way. Being in the topic the marker belonging to the current topic is highlighted in green.
