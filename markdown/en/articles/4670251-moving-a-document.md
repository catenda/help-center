# Moving a document

## 1. How to move a document

1. Select the document using the checkboxes to the left of the document
1. On the top right, a hamburger menu appears (Three dots). Press that
1. Click on move
1. Select the folder you want to move it to.
1. Press "Move here"

### 1.1 Access for moving documents

To move a document you need full access to the folder, not just write access.
