# Boards page

You will be able to find the boards page as a sub page to the [topics page](https://support.catenda.com/en/articles/4670271-topics-page). The boards page will give you an overview of your available topic boards.

![](https://raw.githubusercontent.com/catenda/help-center/main/images/zq777cpq/01-intro.png)

## 1. **Boards actions**

### 1.1 **New topic board**

With the new topic board action you will be able to open the window to create a new topic board.

![](https://raw.githubusercontent.com/catenda/help-center/main/images/zq777cpq/02-new-topic-board.png)

The new topic board page can look something like this:

![](https://raw.githubusercontent.com/catenda/help-center/main/images/zq777cpq/03-new-topic-board.png)

Fill in the name and description and press add. Click [here](https://support.catenda.com/en/articles/6379614-topic-board-from-an-existing-topic-board) to read more about creating new topic boards. _Access required:_ Write access to [creating new topics boards](https://support.catenda.com/en/articles/4670299-topic-board-creation-acl) in [project settings](https://support.catenda.com/en/articles/4670273-project-settings-page#h_f82a1850f3).

## 2. **Topic board list**

From here you have a complete overview of all the topic boards. The topic board list can look something like this:

![](https://raw.githubusercontent.com/catenda/help-center/main/images/zq777cpq/04-topic-board-list.png)

### 2.1 **Topic board name**

By clicking on the topic board name you can jump straight into the topic board. _Access required:_ Read access to the topic board.

### 2.2 **Settings**

Click the settings button to go to the [settings of that board](https://support.catenda.com/en/articles/4670277-topic-board-settings). You can also enter the board and select the settings menu item in the left navigation menu. _Access required:_ Full access to the topic board

### 2.3 **Statistics**

Click on statistics to open the statistics page for that board. _Access required:_ Read access to the topic board.

### 2.4 **Archived**

If you click on show archived towards the bottom of the list you will be able to open archived topic boards. _Access required:_ Project administrator
