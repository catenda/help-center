# Using topic templates when saving markup topics

> Apply a document topic template when creating a markup from a document

Document topic templates can be configured on the [topic template settings page](https://support.catenda.com/en/articles/14076010-topic-template-settings-page). _Access required:_ Project administrator

![](https://raw.githubusercontent.com/catenda/help-center/main/images/qwdn8hpc/01-intro.png)

1. First, create an annotation on a document in the document viewer.
1. Save your Markup as a Topic.
​_Access required:_ Write access to a topic board.
1. A pop-up window will appear to create a Topic from your Markup.
​

    <div class="intercom-container intercom-align-center"><img height="541" src="https://raw.githubusercontent.com/catenda/help-center/main/images/qwdn8hpc/inline-9980053aae3a.png" style="height: auto;" width="500"/></div>

1. Then, select the right topic board within which you want to create your topic.
​_Access required:_ Write access to the topic board.
1. Depending on the topic board you have selected, if document topic templates are configured by an administrator they can be available and suggested in the Template dropdown.
​

    <div class="intercom-container intercom-align-center"><img height="540" src="https://raw.githubusercontent.com/catenda/help-center/main/images/qwdn8hpc/inline-b8fd0218f860.png" style="height: auto;" width="500"/></div>

1. Select the right template you want to use. 
As soon as you select the right template, the new topic will auto-populate with all the fields that were set in the associated topic template settings, accessible from the [topic template settings page](https://support.catenda.com/en/articles/14076010-topic-template-settings-page).

    <div class="intercom-container intercom-align-center"><img height="814" src="https://raw.githubusercontent.com/catenda/help-center/main/images/qwdn8hpc/inline-50d2237a44b3.png" style="height: auto;" width="500"/></div>

1. You can finally save your topic by clicking the "Save as topics" button.
​
