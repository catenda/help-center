# Collision testing in Navisworks

## 1. Collision testing and creating BCF viewpoint using the plugin

1. Set up clash detective and run a clash test.
1. Go through clashes in NW.

    a. Give names if needed

    b. Adjust and save viewpoints using **Focus on clash** and **Save viewpoint** commands

    c. Group together clashes of similar type

1. Create a report as viewpoints. Switch on **Preserve result highlighting**
1. These viewpoints appear in clashes tab in plugin
1. Create issue by selecting one or more clash

    a. Give issue a title

    b. Click add issue.

1. To get meaningful thumbnail:

    a. Select clash in Clash Detective

    b. Select same clash in **Clashes** tab.

    c. Click add issue.
