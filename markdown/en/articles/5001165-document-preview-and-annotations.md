# Document preview and annotations

> Using annotations tools, redlining, comment documents in Issues

New in Catenda Hub is our comprehensive annotation feature. The feature brings the power of our issue management system to Documents, so that Catenda Hub users can enjoy precise commenting, showing where documents and drawings need change and additional information, just like you do using models and viewpoints today.

![](https://raw.githubusercontent.com/catenda/help-center/main/images/ezpb53st/01-intro.png)

## 1. **Annotation toolbar**

Annotation tools have been added to our document viewer:

![](https://raw.githubusercontent.com/catenda/help-center/main/images/ezpb53st/02-annotation-toolbar.png)

You can draw freehand, add shapes, text and call-outs, highlight text and measurements. Click [here](https://support.catenda.com/en/articles/11384618-document-preview-tools) to read more about how the different annotation tools work.

Projects can set up RFI (Request for Information) and similar workflows using topic types and topic boards.

The annotations are attached to the document revision they were drawn on. When you open a document the original document will be displayed. Click [here](https://support.catenda.com/en/articles/10205725-right-menu-of-a-revision-markups#h_e79077cdfd) to read about how to display annotations that have been created for each of the revisions in the document.

## 2. **Saving annotations**

Your annotations can be saved to a topic using our topic management. Either save to an existing issue or create a new topic. Click [here](https://support.catenda.com/en/articles/10209740-saving-a-markup) to read more about how to save annotations to a topic.

In order to save this commented text you can export the topic to BCF, PDF or Excel. The report will contain the comment text as well as a link to the document and a screenshot of where the annotation is in the document.

> **Note:** With the Excel export you only get the topic comment content and not the screenshot.

### 2.1 **Uploading documents with annotations**

If you use the Save As feature you can download the document with its annotations. Annotations can also be added to documents with external tools. If you upload a document with annotations already on it, these will be displayed each time you preview the document on Catenda.

## 3. **File types that can be annotated**

You can now annotate all document formats that our viewer handles (pdf, doc, docx, xlsx, pptx, dwg and dgn).

## 4. **Full screen annotations**

Click the fullscreen button in the [document banner](https://support.catenda.com/en/articles/8461918-document-banner-navigation) to open the document in fullscreen. This can help give you more space on the screen for placing annotations.
