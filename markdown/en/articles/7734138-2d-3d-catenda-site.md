# 2D / 3D - Catenda Site

The 2D / 3D page can be found in the [menu](https://support.catenda.com/en/articles/7338516-navigation-menu-catenda-site) or by tapping the issues button on the bottom navigation bar.

![2D / 3D](https://raw.githubusercontent.com/catenda/help-center/main/images/wy1t8c1t/01-intro.png)

Here is what the 2D / 3D page can look like:

![](https://raw.githubusercontent.com/catenda/help-center/main/images/wy1t8c1t/02-intro.png)

Towards the top of the 2D / 3D page you see the issue board selector.

![](https://raw.githubusercontent.com/catenda/help-center/main/images/wy1t8c1t/03-intro.png)

The issues that result from the filter on the chosen issue board are the issues who have their markers displayed in the 2D view.

### Upper half

On the upper half of the 2D / 3D page you will see the [3D viewer](https://support.catenda.com/en/articles/7744403-3d-viewer-catenda-site).

### Lower half

On the lower half of the 2D / 3D page you will see the [2D viewer](https://support.catenda.com/en/articles/7748370-2d-viewer-catenda-site).

### Dividing bar

Between the two viewers you will see a dividing slider

![](https://raw.githubusercontent.com/catenda/help-center/main/images/wy1t8c1t/04-dividing-bar.png)

Drag on this slider to configure how big a part of the screen you want each of the viewers to take up
