# BCF Workflow using NTI Tools for Revit

**BCF from Revit with NTITools to Bimsync.**

Go to NTItools toolbar and open BCF collaboration. Here you can synchronize with the Bimsync server in your project by clicking on open then open Bimsync issue board and logging in to your Bimsync account.

![](https://raw.githubusercontent.com/catenda/help-center/main/images/2bk19fa9/01-intro.png)

There you can see all your issues.

![](https://raw.githubusercontent.com/catenda/help-center/main/images/2bk19fa9/02-intro.png)

Add issue and fill in the related information. If you have selected the related elements before this step it automatically gets inside the issue

![](https://raw.githubusercontent.com/catenda/help-center/main/images/2bk19fa9/03-intro.png)

Click add comment. If you have not selected elements related to the issue you click pick and select the related elements.

![](https://raw.githubusercontent.com/catenda/help-center/main/images/2bk19fa9/04-intro.png)

Now there is added an issue to the list. You can see the related elements and you click synchronize to get the newly made issue to Bimsync.

![](https://raw.githubusercontent.com/catenda/help-center/main/images/2bk19fa9/05-intro.png)

The BCF made in Revit added to Bimsync. This workflolw works both ways.

![](https://raw.githubusercontent.com/catenda/help-center/main/images/2bk19fa9/06-intro.png)
