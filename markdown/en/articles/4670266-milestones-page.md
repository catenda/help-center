# Milestones page

The milestones page can be found as a sub page to the [topics page](https://support.catenda.com/en/articles/4670271-topics-page). On the milestones page you will be able to see which milestones that are available in the project.

![](https://raw.githubusercontent.com/catenda/help-center/main/images/5tw3m693/01-intro.png)

This is what the milestones page can look like:

![](https://raw.githubusercontent.com/catenda/help-center/main/images/5tw3m693/02-intro.png)

This page contains information about the following topics: [Planning](#h_595c5ebf4c) - [Actions](#h_ef3bbf0441)​ - [Milestones list](#h_5403eb541f)

## 1. Planning

Milestones can be used in many different ways:

- For setting a due date every quarter
- Setting a due date by the end of the month
- Before construction start

Milestones can work as a tool to set more than one due date. When you set a due date for a topic, you can choose a milestone the topic will be apart of.

## 2. Actions

Towards the top right of the page you will be able to find the following actions:

![](https://raw.githubusercontent.com/catenda/help-center/main/images/5tw3m693/03-actions.png)

### 2.1 New milestone

Click on the green + or on the New milestone action in the action menu to create a new milestone.

### 2.2 Archived milestones

Click this action to see which milestones have been archived in the project.

## 3. Milestones list

In the milestones list you will see an overview of each of the milestones that are part of the project. The milestones list can look something like this:

![](https://raw.githubusercontent.com/catenda/help-center/main/images/5tw3m693/04-milestones-list.png)

Click anywhere on the milestone to open the page for that milestone. For each milestone you will see the following information

### 3.1 Title

The title of the milestone

### 3.2 Due date

Here you will see a flag and the due date of the milestone. The flag will be red if the due date is in the next two weeks, gray if the it is more than two weeks ahead in time and pink when the milestone has passed.

### 3.3 Description

The description of the milestone

### 3.4 Topic statistic

In the topic statistic of the milestone you will be able to see what the progress of this milestone is for all topic boards together. See the milestone page for topic board specific data.

**Percentage** Here you will see the percentage of topics that have this milestone assigned and are closed in all topic boards. Candidate topics do not count towards this percentage.

**Open/closed/candidate** Here you will see the amount topics with open, closed or candidate meta-statuses that this milestone has been assigned to. Click on the word open, closed or canidate to see the related topics in the all topics board.

**Member profile picture** Here you will first see the creator of the milestone and the unassigned picture. For each profile picture you will see an amount of topics. For the milestone creator and unassigned this amount can be 0 and will then be green. Other users will only be visible if they have any open topics to which this milestone has been assigned with an amount of open topics to which they have been assigned in red.

### 3.5 Created by

Here you will see the name of the user that created the milestone and the date the milestone was created.
