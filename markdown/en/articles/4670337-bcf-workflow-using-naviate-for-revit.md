# BCF Workflow using Naviate for Revit

**BCF from Revit with Naviate to Bimsync.**

Create an issue in Revit with Naviate BCF manager.

Select the elements you want to make an issue about and go to the Naviate BCF manager. Make sure you select either version 1.0 or 2.0. Click ok.

![](https://raw.githubusercontent.com/catenda/help-center/main/images/a8twaqnf/01-intro.png)

Click save BCF file and save it as a bcfzip file.

![](https://raw.githubusercontent.com/catenda/help-center/main/images/a8twaqnf/02-intro.png)

Go to issues in bimsync and import the BCF you created in Revit

![](https://raw.githubusercontent.com/catenda/help-center/main/images/a8twaqnf/03-intro.png)

The selected BCF.

![](https://raw.githubusercontent.com/catenda/help-center/main/images/a8twaqnf/04-intro.png)

The BCF is now loaded under issues in Bimsync.

You can then go and change the different settings inside the issue by clicking on it.

![](https://raw.githubusercontent.com/catenda/help-center/main/images/a8twaqnf/05-intro.png)

**BCF from Bimsync to Revit with Naviate.**

Filter out the issue you want to get.

![](https://raw.githubusercontent.com/catenda/help-center/main/images/a8twaqnf/06-intro.png)

Export the filtered issues.

![](https://raw.githubusercontent.com/catenda/help-center/main/images/a8twaqnf/07-intro.png)

You will get notified when the BCF is exported. Download the BCF.

![](https://raw.githubusercontent.com/catenda/help-center/main/images/a8twaqnf/08-intro.png)

Go back to Revit to get the downloaded BCF.

Open it up in BCF manager.

![](https://raw.githubusercontent.com/catenda/help-center/main/images/a8twaqnf/09-intro.png)

The BCF is loaded in Revit with the related components. Revit makes a individual view for the BCF.

![](https://raw.githubusercontent.com/catenda/help-center/main/images/a8twaqnf/10-intro.png)

You can also section out the related components.

![](https://raw.githubusercontent.com/catenda/help-center/main/images/a8twaqnf/11-intro.png)
