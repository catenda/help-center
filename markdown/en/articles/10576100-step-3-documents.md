# Step 3 - Documents

> Up- and download, view and communicate about project documents

> **Tip:** Find [step 2](https://support.catenda.com/en/articles/10576075-step-2-interface) in [getting started](https://support.catenda.com/en/collections/2657208-getting-started) on the help center or in the help menu.

## 1. **Uploading**

This is the delivery area of Catenda Hub where you can upload all file types by simply selecting the desired destination and dragging and dropping to upload (just like Windows Explorer). You can also upload via the upload button in the top right corner.

![](https://raw.githubusercontent.com/catenda/help-center/main/images/mdxrvj2r/01-uploading.png)

![](https://raw.githubusercontent.com/catenda/help-center/main/images/mdxrvj2r/02-uploading.png)

When a document match is found in the folder, Catenda creates a new revision and keeps the records of all superseded revisions.

## 2. **Downloading**

Simply search for required documents using the search bar or filter sidebar and select them. Filters can be saved for future usage.

![](https://raw.githubusercontent.com/catenda/help-center/main/images/mdxrvj2r/03-downloading.png)

After selecting your documents click on the download button.

![](https://raw.githubusercontent.com/catenda/help-center/main/images/mdxrvj2r/04-downloading.png)

The documents will be downloaded onto your PC.

## 3. **Viewing**

The document viewer in Catenda supports a lot of file types ranging from PDFs to various Microsoft files (doc, ppt, xls, etc.) as well as point cloud files. The Viewer also supports video files such as mp4. Simply search for the required file using the search bar or filter sidebar and click on it to open in the viewer.

![](https://raw.githubusercontent.com/catenda/help-center/main/images/mdxrvj2r/05-viewing.png)

## 4. **Markups**

Use the markup tools to identify areas you would like to communicate with other project members. The markups can be added to existing topics or you can create a new one which can be placed on a topic board of choice.

![](https://raw.githubusercontent.com/catenda/help-center/main/images/mdxrvj2r/06-markups.png)
