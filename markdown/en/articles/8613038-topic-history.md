# Topic history

You will be able to find the topic history button towards the top of the [topic body](https://support.catenda.com/en/articles/8053352-issue-body). By clicking on this button you will expand the history of any changes made to the topic during its time in your Catenda project.

In the topic history you will be able to find information about the following topics:

## 1. **Title**

_Title set to: \<Title> -_ This will always be set when a new topic is made

## 2. **Status and type**

### 2.1 **Status**

_Status set to \<Status> -_ The status that was applied to the topic _Status changed to \<Status> -_ If the status has changed you will see it here.

### 2.2 **Type**

_Type set to \<Type> -_ The type that was applied to the topic _Status changed to \<Type> -_ If the type has changed you will see it here.

## 3. **Scheduling**

### 3.1 **Milestones**

_Milestone changed to \<Milestone>_ - See what milestone was added

### 3.2 **Due date**

_Due date changed to \<Due date>_ - See what due date was added _Due date changed to \<Due date>_ \<Due time> See what due date and due time were added

## 4. **Posts**

### 4.1 **Description**

_Description changed -_ Click on this box to see what the previous description looked like.

**Description set** Click on this box to see what the description of the topic was when it was initially made

### 4.2 **Comments**

_Comment deleted_ - See if a comment was deleted _Comment changed_ - This has no box but you will see the time of the initial action and the time of the latest edit.

## 5. **Label**

_Label added \<Label name>_ - The label that was added Any labels that were added to the topic. If the label has been deleted it will be struck through.

_Label added \<Label name>_ - The label that was removed

## 6. **Custom field**

**\<Custom field> set to \<Custom field value>** If a custom field was assigned you will see it here

**\<Custom field> changed to \<Custom field value>** See what the custom field value was changed to

## 7. **Image**

_Image added_ - If an image was added you will see it here _Image removed_ - If an image attached to a comment was removed from the topic.

## 8. **Responsibility**

### 8.1 **Assigned user**

Assigned user set to \<User> Assigned user set to \<User> \<Team> Assigned user changed to \<New user> Assigned user changed to \<New user> \<New Team> Assigned team set to \<Team> Assigned team changed \<New team> Assigned user removed

### 8.2 **Requester user**

Requester user set to \<User> Requester user set to \<User> \<Team> Requester user changed to \<New user> Requester user changed to \<New user> \<New Team> Requester team set to \<Team> Requester team changed \<New team> Requested by user removed

## 9. **topic movement**

Topic _was moved from \<_Topic _board>_ - See which topic board this topic was originally in

## 10. **Document links**

_Document linked \<Document name>_ - See what document was linked to this topic _Document unlinked \<Document name>_ - See what document was unlinked from this topic
