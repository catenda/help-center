# ステップ 1 - アカウント設定

> アカウントを作成してプロジェクトへの招待を受け取る

登録の次のステップへのリンク付きで、Catenda からメール経由で招待が送信されます。このリンクを使用して、個人認証情報を使ってアカウントを作成できます。

![](https://raw.githubusercontent.com/catenda/help-center/main/images/m6qttvze/01-intro.png)

登録が完了すると、任意の Web ブラウザから Catenda Hub でプロジェクトにアクセスできます。[https://hub.catenda.com/signin](https://hub.catenda.com/signin)

参考資料: [サインインページ](https://support.catenda.com/en/articles/7891486-sign-in-page), [アカウント作成](https://support.catenda.com/en/articles/8412895-creating-an-account), [ランディングページ](https://support.catenda.com/en/articles/4670260-landing-page)

> **注:** ヘルプセンターまたはヘルプメニューの[スタートガイド](https://support.catenda.com/en/collections/2657208-getting-started)で[ステップ 2](https://support.catenda.com/en/articles/10576075-step-2-interface) を確認してください。
