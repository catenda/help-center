# Revit IFCエクスポートマニュアル

このIFCエクスポートマニュアルの目的は、適切な目的でIFCをエクスポートする方法についてのガイドをユーザーに提供することです。 IFCファイルは、モデルから多くの情報を取得する必要がある場合、すぐに重く、大きくなる可能性があります。 このため、IFCをエクスポートする場合は、不要な情報をチェック解除する必要があります。 Catendaにモデルをアップロードする場合、モデルに多くの情報と高い詳細度が常に必要とは限りません。 このマニュアルの後半では、モデルをより小さく、操作しやすくするために推奨される設定について説明します。 ここでは、RevitからCatendaへのIFCエクスポートの最も適切な方法をステップバイステップで説明します。

この記事では、以下の情報をご確認いただけます：

_[IFCエクスポート](#h_38cc715c02) - [セットアップの変更](#h_85d4335aad) - [IFCオプション](#h_487f3aa176) - [色と素材](#h_f458b56011)_

## **プロジェクト設定**

エクスポートする前に、Revitプロジェクトのグローバル一意識別子（GUID）が正しいことを確認することが重要です。

`管理 -> 設定 -> プロジェクト情報 -> IFCパラメータ`

![プロジェクト情報 IFCパラメータ IfcSite GUID IfcBuilding GUID IfcProject GUID](https://downloads.intercomcdn.com/i/o/975614819/ee92de6f2477949e208dac45/image.png?expires=1779991200&signature=488a733e6dce35b3f9f765548a38ebcbb902309625bfa1827447b3e10ff5d1ca&req=fSciEMh6lYBWFb4V1XW4gW4DZHJ8Y%2FMyB4KsohYxe9JI9upp%2BSOrcthV9MYf%0AU5M5bUP8q8Ck5XUUcK86OtHqKw%3D%3D%0A)

GUIDが前のエクスポートと異なる場合、新しいエクスポートのオブジェクトがBCFトピックのGUIDに正しくリンクされません。

新しいプロジェクトを作成すると、一意のIDが割り当てられます。

_パーツを建築要素としてエクスポート_

標準IFC要素のようにパーツをエクスポート

_混合「ソリッドモデル」表現の使用を許可_

ユニットに対してBRepと抽出幾何学の混在を許可するにはこのオプションを選択します。

_幾何学を作成する際にアクティブビューを使用_

幾何学を生成するアクティブビューを使用するにはこのオプションを選択します。 これを3Dビュー以外で使用すると予期しない結果が生じる可能性があることに注意してください。

_参照用のファミリとタイプ名を使用_

参照のためにファミリとタイプ名を使用するにはこのオプションを選択します。

_ルームボリュームに2Dルーム境界を使用_

IFC 2x2にエクスポートする場合も標準である、簡略計算法（2Dスペース境界の抽出に基づく）を使用してルームボリュームを計算するにはこのオプションを選択します。

_IfcSiteの高度をサイトのローカルプレースメント原点に含める_

Z-オフセットの高さをIfcSiteのローカル位置に含めるにはこのオプションを選択します。 オプションを削除して除外します。

_エクスポート後にエレメントパラメータにIFC GUIDを保存_

エクスポート後にプロジェクトファイルに生成されたIFC GUIDを保存するにはこのオプションを選択します。 これにより、プロジェクト、ウェブサイト、ビルディングガイドのアイテムとそのタイプ、プロジェクト情報に「IFC GUID」パラメータが追加されます。

_バウンディングボックスをエクスポート_

「バウンディングボックス」表現をエクスポートするにはこのオプションを選択します。 このオプションはGSAエクスポートでは自動的に選択されたままです。

_テッセレーション幾何学を三角形分割として保持_

複雑な曲線要素または殻がIFCエクスポート後に正しく表示されない場合は、このオプションを選択できます。 非常に重いIFCファイルが生成される可能性があることに注意してください。

_IFCTypeネームのタイプネームのみを使用_

オブジェクトのBAT-IDまたはIDをエンティティの名前として表示する場合は、このオプションを選択します。

_表示されているRevit名をIFCエンティティ名として使用_

Revitオブジェクト名をエンティティの名前にする場合は、このオプションを選択します

_常に面取りされた床と屋根を単一のIFCエンティティとしてエクスポート_

複数の面を持つ床と屋根の面を単一のエンティティに統合するにはこのオプションを選択します。

_プロジェクト情報で「最後に変更」ユーザーを作成者に設定_

このエクスポートの変更の作成者である場合は、このオプションを選択します

### **地理参照**

Revit座標がプロジェクト内の他のモデルと同期され、同じ場所に配置されるようにすることが重要です。

したがって、Catenda Hubでポイント測定を使用して座標を測定し、Catenda Hubで測定されたポイントと同じ場所にあるRevitのポイントに座標ベースを指定します。

![ポイントで座標を指定](https://downloads.intercomcdn.com/i/o/969764101/0c12135b4cfe78982de2d2c9/image.png?expires=1779991200&signature=060be8f8a603fc0b1a1df40a9afcb314add9da447d344221a97c1a482e645427&req=fSYuEc96nIFeFb4V1XW4gXGcH65AQ%2BvBsmsiUzSJvpOZcWtpsV6NEtp3COB%2B%0AaAl33RB7R1SlFNStV1N44yOhtw%3D%3D%0A)

[管理]タブ -> [座標] -> [座標ベースを指定]でこのオプションを見つけることができます。

_ポイントで座標を指定_

北/南、東/西、および高度の座標を指定することにより、モデルを再配置し、モデルを北に向けて回転させます。

Revitでは90度の角度でモデル化する方が簡単なことが多く、モデル全体を回転させたくない場合があります。

この場合、代わりに北を回転させることができます。

[管理]タブの[座標]の下のドロップダウンオプションで、このオプションを見つけることができます。

## **IFCオプション**

Revitプロジェクトの[IFCオプション]は以下の場所にあります：

`ファイル -> エクスポート -> オプション -> IFCオプション`

![エクスポート -> オプション -> IFCオプション](https://catenda-as.intercom-attachments-1.com/i/o/271057200/45a1257c55b1828c71e969a4/image-11.png?expires=1779991200&signature=19ba7f1f06c737fd1570f0a06070ad9db96d0694ad820be88fb17bb60ed41856&req=dicmFsx5n4FfFb4V1XW4gakKrGXQvzEGkCAUJOtaQtRh3HZCglVgstn6e2Ch%0AaxVYb3DMOLXIpE8hXgvg%2F%2FKfNg%3D%3D%0A)

> **注意：** Revit 2026以降、このオプションは以下で利用可能になりました： `エクスポート -> IFC -> 一般 -> カテゴリマッピング -> ドロップダウンの右側にあるアクションメニュー` ここで、_IFCオプション_内にモデルをIFCファイルにエクスポートするための設定を行います。 ここでは、モデルをIFCにエクスポートするための セットアップ プロパティをカスタマイズできます。 このマニュアルの開始時に述べたように、モデルから不要な情報をすべて取り出す必要はありません。 エクスポート前に不要な情報をチェック解除してください。 ![IFCエクスポートクラス](https://catenda-as.intercom-attachments-1.com/i/o/271057211/d475dd84fb2efef9d7bf4c1d/image-12.png?expires=1779991200&signature=ceb04d005a9687945fd4c5efc680f142fa8f71ab5daddaf6ef11c8828c6103ea&req=dicmFsx5n4BeFb4V1XW4gdpd3cvRJhUNG2sGxDwDCuCNDMBg6YsRp7MEvaGe%0Ao0oBh68YB8CtIWlb%2Bwt%2Fl%2BbXug%3D%3D%0A) Catenda Hubでグリッドを表示することが可能であり、Revitモデルにこれらがある場合、IFCオプションでグリッドをIFCでエクスポートするように設定できます。 デフォルトでは、これらはRevitからエクスポートされません。

## **色と素材**

Catendaに表示される色は、インポートされたIFCファイルから読み込まれます。

ファミリのマテリアルプロパティがIFCパラメータに追加されると、マテリアルプロパティのマテリアルの色がIFCに追加され、Catendaに表示されます。

Revitのマテリアルはマテリアルブラウザにあります：

`[管理]タブ -> [設定]セクション -> [マテリアル]`

マテリアルブラウザでは、色の設定はマテリアルのグラフィックスタブにあります：

![管理 -> マテリアル -> マテリアルブラウザ -> 新しいマテリアルを作成](https://downloads.intercomcdn.com/i/o/areracg3/1798363404/1016d10e61dcd4bbf3969dd97a39/image.png?expires=1779991200&signature=6538f237377a045f7b7f567244f4fdefd99bc74142dc54780c79610f54b88159&req=dScuHsp4noVfXfMW3nq%2BgRHARe%2BQbSvVJU1tDswOuiSVbxNsRYJibJHNSy4a%0ADvZyAbDR4cFQbN%2FOUhLbnJ9ZKrw%3D%0A)

シェーディングをレンダリング設定にロックすることも可能です。
