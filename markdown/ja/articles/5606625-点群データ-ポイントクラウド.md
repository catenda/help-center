# Catenda Hubのポイントクラウド

> Catenda HubでレーザースキャンおよびLIDAR技術の利点を活用する方法をご確認ください。

> **注:** サンプルファイルは[こちら](https://drive.google.com/file/d/1G8U916oihDl5qHrTfQaVHxDeLsbq1ulG/view?usp=sharing)からダウンロードできます。

ポイントクラウドデータセット(PC)はCatenda Hub内で可視化できます。個別のPCはドキュメント領域内でプレビューできます。複数のPCは[3Dビューア](https://support.catenda.com/en/articles/8227211-3d-viewer)に読み込むことができます。3Dビューアでは、PCはIFCモデルやGMLファイルなどの他の3Dドキュメント形式と一緒に表示できます。

データを表示するためのさまざまな設定については、以下を参照してください。

## 1. **ドキュメントセクションプレビュー**

以下の2つのファイル形式のPCデータはCatendaでプレビューできます。

- `*.e57`
- `*.las`

PCはドキュメントセクション内の他のドキュメントと同様にアップロードできます。これら2つのファイル形式では、最大25 GBのファイルをドキュメントセクションにアップロードできます。複数のファイルをドキュメントセクションに読み込み、3Dビューアで一緒に表示できます。

### 1.1 **PCデータのアップロード**

[Catenda Desktop Connector](https://support.catenda.com/en/articles/8358861-catenda-desktop-connector)を使用するか、PCファイルを圧縮フォルダにしてから[zip import](https://support.catenda.com/en/articles/7945410-upload-a-zip-file-structure)を使用してPCをアップロードすることをお勧めします。これらの方法はアップロードファイルサイズを小さくし、ネットワークエラーのリスクを最小化してファイルを迅速にアップロードできるため、時間を節約するのに役立ちます。

PCをドキュメントセクションにアップロードすると、ドキュメントプレビューの処理が開始されます。プレビューが処理されている間、ドキュメントプレビューの上部に灰色のバーが表示されます。

![](https://raw.githubusercontent.com/catenda/help-center/main/images/yve4vvqf/01-uploading-pc-data.png)

プレビュー処理時間はPCのサイズに依存します。処理にはGB当たり1時間かかりますが、ポイントクラウドによってはそれ以上またはそれ以下になる可能性があります。

プレビューの処理が完了したら、ドキュメントをクリックしてドキュメントプレビューでPCを表示します。

![](https://raw.githubusercontent.com/catenda/help-center/main/images/yve4vvqf/02-uploading-pc-data.png)

### 1.2 **共有PCリビジョン**

プロジェクトに対して[リビジョン公開](https://support.catenda.com/en/articles/9874698-revision-publishing-iso-19650)が有効になっている場合、ポイントクラウドは共有リビジョンとしてアップロードされます。共有PCリビジョンはドキュメントセクションでのみプレビューできます。PCを3Dビューアに読み込むには、リビジョンを公開する必要があります。個別の共有リビジョンは、ドキュメントプレビューの右メニューの公開アクションで公開できます。複数の共有リビジョンは、ドキュメント構造の選択項目アクションで公開できます。リビジョンを公開すると、公開されたリビジョンのプレビューの処理が開始されます。

### 1.3 **公開PCリビジョン**

公開されたPCリビジョンのプレビューが処理されると、ドキュメント表の表示ビューア列に3Dボタンが表示されます。

![](https://raw.githubusercontent.com/catenda/help-center/main/images/yve4vvqf/03-br-published-pc-revisions.png)

3Dボタンをクリックすると、ドキュメントの最新公開リビジョンが3Dビューアに読み込まれます。

PC、IFC、またはGMLドキュメントなどの3Dドキュメントを含む1つ以上のドキュメントを選択している場合、ドキュメント表の選択項目アクションメニューに3Dドキュメントアクションも表示されます。この方法により、複数の3Dドキュメントの最新リビジョンを3Dビューアに一度に読み込むことができます。

![](https://raw.githubusercontent.com/catenda/help-center/main/images/yve4vvqf/04-br-published-pc-revisions.png)

公開されたPCリビジョンが1つ以上あり、処理が完了したドキュメントでは、右上のアクションメニューに3Dアクションが表示されます。

> **注:** 3Dアクションでは、ドキュメントの最新公開リビジョンが3Dビューアに読み込まれます。 以前のリビジョンを表示している場合でも同じです。

![](https://raw.githubusercontent.com/catenda/help-center/main/images/yve4vvqf/05-br-published-pc-revisions.png)

## 2. **3Dビューアプレビュー**

3Dボタンをクリックすると、PCのポイントが3Dビューアへの読み込みを開始します。PCの3Dビューは次のようなものです。

![](https://raw.githubusercontent.com/catenda/help-center/main/images/yve4vvqf/06-3d-viewer-preview.png)

3Dビューアの上部に緑色のローディングバーが表示されます。このローディングバーは、現在のカメラ位置と角度に対して3Dビューアに読み込まれたポイント数を示しています。周りを回転させるとポイントがビューから消えて読み込み解除されたり、より多くのポイントが範囲内に入ってきて読み込まれたりするため、ローディングバーは変わる可能性があります。

ズームとポジショニングをいくつか調整すると、PCプレビューは次のようなものになる場合があります。

![](https://raw.githubusercontent.com/catenda/help-center/main/images/yve4vvqf/07-3d-viewer-preview.png)

## 3. **3Dビューア設定**

このセクションでは、Catenda Hub内の設定を確認して、PCで最高のエクスペリエンスを実現できます。

> **注:** 3Dビューアを設定する前に、Catendaアプリケーションが開かれているアプリケーションが[推奨通り](https://support.catenda.com/en/articles/6921941-hardware-recommendation)設定されていることを確認してください。

ポイントクラウドを設定するために使用できる設定は、2つの場所にあります。

### 3.1 **1. ポイント予算:**

ポイント予算は、3Dビューアの右上の歯車アイコンに表示される[3Dビューア設定](https://support.catenda.com/en/articles/5784718-3d-viewer-settings)で設定できます。

![](https://raw.githubusercontent.com/catenda/help-center/main/images/yve4vvqf/08-1-point-budget.png)

3Dビューア設定メニューでは、ポイント予算を100,000から10,000,000に調整できます。より多くのポイントを読み込むと、より多くの時間がかかり、システムからより多くのリソースが必要になる場合があります。より多くのポイントを読み込むことにより、ポイントクラウドはより高い忠実度で表示できます。同じビューを以下で確認します:100,000ポイント

![](https://raw.githubusercontent.com/catenda/help-center/main/images/yve4vvqf/09-1-point-budget.png)

1,000,000ポイント

![](https://raw.githubusercontent.com/catenda/help-center/main/images/yve4vvqf/10-1-point-budget.png)

10,000,000ポイント

![](https://raw.githubusercontent.com/catenda/help-center/main/images/yve4vvqf/11-1-point-budget.png)

### 3.2 **2. リビジョンセレクター、および個別PC設定:**

3Dビューアの左上に[リビジョンセレクター](https://support.catenda.com/en/articles/4670246-accepting-an-invitation-to-join-a-project)が表示されます。

![](https://raw.githubusercontent.com/catenda/help-center/main/images/yve4vvqf/12-2-revision-selector-and-individual-pc-settings.png)

[モデルメニュー](https://support.catenda.com/en/articles/4670279-revision-selector#h_180a64490d)では、3Dビューアに読み込んだ個別の[3Dドキュメント](https://support.catenda.com/en/articles/4670279-revision-selector#h_fce60074ef)を見つけることができます。

![](https://raw.githubusercontent.com/catenda/help-center/main/images/yve4vvqf/13-2-revision-selector-and-individual-pc-settings.png)

ポイントクラウドは、リビジョンセレクターで歯車アイコンをクリックして設定できます。

![](https://raw.githubusercontent.com/catenda/help-center/main/images/yve4vvqf/14-2-revision-selector-and-individual-pc-settings.png)

設定メニューは次のようになる場合があります。

![](https://raw.githubusercontent.com/catenda/help-center/main/images/yve4vvqf/15-2-revision-selector-and-individual-pc-settings.png)

_属性_ _RGBA_ - デフォルト PCポイントを色で表示

_属性インテンシティ_ ポイントに色が含まれていない場合は、インテンシティオプションを使用できます。たとえば、ポイントクラウドがトンネルまたは暗い部屋でキャプチャされた場合です。

**不透明度**モデルがポイントクラウドと一緒に表示される場合、視点をより良く理解するためにポイントを暗くする必要がある場合があります。

_ポイントサイズ_ _適応型_ - デフォルト ポイントがカメラに近いほど、ポイントが大きくなります。表示中のポイントはメモリに読み込まれます。ビューの外に移動するポイントはメモリから読み込み解除されます。

> **注:** Catendaが開かれているハードウェアおよびソフトウェアが[推奨通り](https://support.catenda.com/en/articles/6921941-hardware-recommendation)設定されていることを確認してください。これはデバイスが処理できるポイントサイズ読み込みのタイプに影響を与える可能性があります。

**ポイントサイズ 固定サイズ**読み込まれたポイントはこのオプションでメモリ内に留まります。多くのポイントを使用すると、システムが処理できるポイント数に近づくにつれて、回転が遅延し、動きが遅くなることに気付くでしょう。また、デバイスの制限に近づくにつれて、新しいポイントの読み込みにかかる時間がますます長くなることに気付くかもしれません。ポイントはカメラに最も近いものから最初に読み込まれます。ドキュメントセクションからポイントクラウドを読み込む前に、ポイントを読み込みたい場所にカメラを配置してください。このカメラの位置をこのポイントクラウドでより頻繁に使用する予定がある場合は、このトピック内でブックマークまたはスナップショットを作成することを検討してください。このポジションに戻るために再生できます。

**ポイント密度スライダー**密度スライダーは最終設定を行うのに役立ちます。正しい値に対するルールはなく、アップロードするデータ(データセットのサイズ、ポイント密度、スキャンのタイプなど)に依存します。アドバイス:ポイントクラウドを読み込んで、しばらくの間、十分なポイントが表示されたら、必要に応じて設定を調整します。
